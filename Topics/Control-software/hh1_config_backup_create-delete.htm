Creating and Deleting Configuration Backups

See Also for links to information about backing up and restoring system configurations.

The control system automatically creates backups (located under the AUTOMATED tab) just before you:

● Restore a previously saved backup
● Make changes to a node (for example, adding, removing, or renumbering a node)

You can also create a manual backup at any time.

Manual and automatic backups are saved on both client computers. If you need to replace a client computer, backups are transferred from the client computer that is not being replaced.

Creating a Configuration Backup (Manual Configurations Only)

1. On any CONFIGURATION screen, tap the BACKUP AND RESTORE tab.
2. Tap MANUAL.
3. Tap NEW BACKUP.
4. When the SAVE CONFIGURATION SETTINGS BACKUP panel displays, tap the BACKUP NAME and BACKUP DESCRIPTION fields to enter information.
5. Tap CONTINUE.

Deleting a Configuration Backup

1. On any CONFIGURATION screen, tap the BACKUP AND RESTORE tab.
2. Select a backup from the list on the left side of the screen. (Choose between manually or automatically created backups with the AUTOMATED or MANUAL button.)
3. Tap DELETE BACKUP to remove the selected backup.
