Restoring a Configuration Backup

See Also for links to information about backing up and restoring system configurations.

You can restore a saved backup of configuration information to reapply the settings that were in force when the backup was made.

Note:   Restoring a backup overwrites all current system configuration information with configuration information from the backup.

Caution!  Restore only those backups that were created by the same ROV system you are applying them to. You should not apply backups from one ROV system to another.

Restoring a Backup from a Client Computer

Note:  You must have supervisor permissions to restore a configuration backup from a client computer.

1. On any CONFIGURATION screen, tap the BACKUP AND RESTORE tab.
2. Select the desired backup from the list on the left side of the screen. (Choose between manually or automatically created backups with the AUTOMATED or MANUAL button.)
3. Tap RESTORE SELECTED BACKUP.
4. When the RESTORE SELECTED CONFIGURATION SETTINGS panel displays, tap CONTINUE.

The system then automatically restarts, with the restored configuration settings in place.

Restoring a Configuration Backup from An External Device

You must restart the control system to restore a configuration backup from an external memory device.

Note:  The procedure below describes how to install a configuration backup if the external memory device containing the backup file is inserted into the right client computer. You could alternatively insert it into the left client computer, altering the instructions as appropriate. The process of restoring a configuration backup on one client computer automatically puts the configuration on the other client computer.

1. Insert the external memory device (flash drive, DVD, or CD) containing the configuration backup file into the right client computer. The file must be in the root directory of the memory device.
2. Restart both client computers.
3. On the right client computer, when the CONFIGURATION button changes to LOAD CONFIGURATION BACKUP, tap the button. Both the right and left screens then display controls for selecting configuration backups.

4. On the right client computer, tap the EXTERNAL tab.
5. Tap BROWSE, and the screen shows all items on the external memory device that have the file name extension .srBackup.
6. Tap the configuration backup that you wish to restore, and then tap OPEN. The screen then displays details about the backup.
7. To restore that backup, tap RESTORE SELECTED BACKUP. The configuration backup is installed on both client computers, and the control system starts up.
8. Remove the external memory device from the right client computer.
