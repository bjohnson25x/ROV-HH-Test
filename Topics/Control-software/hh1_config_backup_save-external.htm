Saving a Configuration Backup to an External Device

See Also for links to information about backing up and restoring system configurations.

Note:  You should save periodic configuration backups to an external memory device like a DVD or flash drive. If you ever need to replace both the left and right client computers, you can use the saved configuration file to restore configuration settings.

1. On any CONFIGURATION screen, tap the BACKUP AND RESTORE tab.
2. Insert an external memory device (DVD, CD, or flash drive) into the desired client computer.
3. Tap SAVE TO EXTERNAL DEVICE.
4. When the SELECT FILE SAVE LOCATION panel displays, select the external memory device.
5. If you wish to change the default backup name, tap the name field and enter a new name.
6. Tap SAVE.
7. Remove the external memory device from the client computer.

Backup files have the file name extension .srBackUp.
