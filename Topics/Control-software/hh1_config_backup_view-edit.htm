Viewing and Editing Configuration Backups

See Also for links to information about backing up and restoring system configurations.

To display available information for a backup:

1. On any CONFIGURATION screen, tap the BACKUP AND RESTORE tab.
2. Select a backup from the list on the left side of the screen to display available information for that backup. (Choose between manually or automatically created backups with the AUTOMATED or MANUAL button.)
3. If you want to edit the backup label, tap the BACKUP NAME field.
4. If you want to edit the backup description, tap the BACKUP DESCRIPTION field.
