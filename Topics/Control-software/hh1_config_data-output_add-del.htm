Adding and Removing Data Inputs and Outputs

See Also for links to information about configuring data inputs and outputs.

Adding a Data Output

1. On any CONFIGURATION screen, tap the DATA I/O tab.
2. Tap DATA OUTPUT.
3. Select a data output string from the list on the left side of the screen.
4. Tap NEW DATA OUTPUT.
5. Enter a label for the data output.
6. Select a data output from the drop-down list of available data outputs.
7. Enter an update rate (in Hz).
8. Tap SAVE.

Adding a Data Input

1. On any CONFIGURATION screen, tap the DATA I/O tab.
2. Tap DATA INPUT.
3. Select a data input string from the list on the left side of the screen.
4. Tap NEW DATA INPUT.
5. Enter a label for the data input.
6. Select a data input from the drop-down list of available data inputs.
7. Tap SAVE.

Removing a Data Output

1. On any CONFIGURATION screen, tap the DATA I/O tab.
2. Tap DATA OUTPUT.
3. Select a data output string from the list on the left side of the screen.
4. Tap DELETE DATA OUTPUT.

Removing a Data Input

1. On any CONFIGURATION screen, tap the DATA I/O tab.
2. Tap DATA INPUT.
3. Select a data input string from the list on the left side of the screen.
4. Tap DELETE DATA INPUT.
