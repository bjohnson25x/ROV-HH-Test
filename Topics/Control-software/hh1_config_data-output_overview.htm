About Configuring Data Inputs and Outputs

See Also for links to information about configuring data inputs and outputs.

See Also for links to information about survey data and data import/export.

Purpose of the data input/output configuration screens: You can:

● View and edit information about each data input string and output string.
● Add or remove data input and output strings.
Access: On any CONFIGURATION screen, tap the DATA I/O tab.

The data input function lets you use a data stream from outside the ROV control system to aid in performing certain tasks. (For example, you can input information from a customer-supplied survey navigation system to guide the AutoTrack function.)

The data output function is used to stream ROV system data to an external device (like a survey computer). You can specify which port is used to output the data. You can also select the desired data format and the frequency with which the transmitted data is updated.

To display configuration screens for data input and output:

1. On any CONFIGURATION screen, tap the DATA I/O tab.
2. Tap DATA INPUT or DATA OUTPUT.
3. Select the desired data input or output from the list on the left side of the screen.
4. If you want to change the data string label, tap the label box.
5. If you want to view or change the input or output configuration, tap EDIT.
