Configuring Data Strings and Ports

See Also for links to information about configuring data inputs and outputs.

Configuring a Data String

For both data inputs and outputs, tap the DATA STRING tab to adjust:

● Data string label
● Data string format (options depend on system configuration)
● Frequency of data string update (measured in Hz)

Configuring the Data Port

Use the PORT DETAILS tab to edit the configuration of the port for data input or output. Port configuration varies depending on how the data input or output is being used.

In the EDIT DATA OUTPUT panel, tap the PORT DETAILS tab to adjust:

● Port type (serial or network)
● Port characteristics
