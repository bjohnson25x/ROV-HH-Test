Installing a Custom Device

See Also for links to information about installing and removing devices.

For screen illustrations of some of the steps below, see Installing a Preconfigured Device.

You can install custom devices that use either a serial interface or a network (Ethernet) interface (see Device Signal Types).

1. Ensure that the user profile for your control station includes supervisor permissions.
2. Display any CONFIGURATION screen.
3. Tap the DEVICE tab at the top of the screen.
4. Tap INSTALL DEVICE.
5. On the list on the left side of the screen, tap the CUSTOM tab.
6. When the INSTALL DEVICE panel displays, tap the button (on the panel’s right side) for the type of device that you are adding. (Note that INSTALL AS A USER TUNNELED DEVICE contains a check mark that you cannot remove.) Then tap NEXT STEP.

7. When the list of nodes displays, tap the tab (on the screen’s left side) for the subsea node that is attached to the device. A list of the node’s ports then displays.

8. Tap the button for the node port that is attached to the device. (You cannot select a port that is currently attached to another device.) The port interface module must match the device type; for example, a serial device must be attached to a SIM and an Ethernet device must be attached to a NIM. Then tap NEXT STEP.
9. The control system must then establish a connection for the device to a rack node:
— If you are installing a device with a serial interface, the SELECT SERIAL TUNNEL panel displays. Select a rack node SIM port from the SERIAL TUNNEL drop-down list. Then tap NEXT STEP.
— If you are installing a device with a network interface, the SELECT ETHERNET TUNNEL panel displays. Select a rack node NIM port from the ETHERNET TUNNEL drop-down list. Then tap NEXT STEP.
10. Tap INSTALL.

To communicate with the installed custom device, you must attach a computer or other appropriate control device to the rack node port for the device.
