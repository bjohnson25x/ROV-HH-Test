Installing a Preconfigured Device

See Also for links to information about installing and removing devices.

You must have supervisor permissions to install or remove a device from the control system.

To install a preconfigured device:

1. Ensure that the user profile for your control station includes supervisor permissions.
2. Display any CONFIGURATION screen.
3. Tap the DEVICE tab at the top of the screen.
4. Tap INSTALL DEVICE.
5. When the INSTALL DEVICE panel displays, tap the tab for the desired device category.

6. In the list of preconfigured devices for that category, tap the button for the device that you are adding. (If you do not see the device that you want to add, you must add the item as a custom device as described in Installing a Custom Device.)
7. If you want signals for the device to bypass the ROV control system and instead be sent to an external processor (as described in Device Signal Destinations), tap INSTALL AS USER TUNNELED DEVICE to add a check mark. Because some devices cannot have a user tunnel, the check box may not appear for a particular device.
8. Tap NEXT STEP.
9. Select the subsea node and port for the device. You must select a port that is currently not attached to another device, and the port interface module must match the device type. (For example, if you are adding a camera, it must be attached to a video interface module.) Screen appearance during the process varies; for example, some devices (like a pan and tilt) require more than one port connection. The figure below shows a typical screen for selecting a single port.

10. If you checked the box INSTALL AS A USER TUNNELED DEVICE, the SELECT SERIAL TUNNEL panel displays. From the drop-down list, select the rack node port to use to communicate with the device. (For information about user tunnels, see Device User Interfaces and Mirroring Compared.)

11. Tap NEXT STEP.
12. Tap INSTALL.
