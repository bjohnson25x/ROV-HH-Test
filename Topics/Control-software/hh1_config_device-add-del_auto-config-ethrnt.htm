Installing, Moving, and Deleting an Auto-Configuration Device

See Also for links to information about installing and removing devices.

Note:  You must have supervisor permissions to install or remove a device from the control system.

About Auto-Configuration

Auto-configuration is an option available on some ROV control systems. Auto-configuration allows “plug and play” functionality when you:

● Install certain Ethernet devices in node ports (see list below).
● Move such a device from one port to another.

Note:  To determine if the ROV control system is enabled for auto-configuration, you must use the Node Changer application (see Subsea Node: Configure ID Number, RSTP and VLAN Status.

Typically, when you plug in a device to a node port, you must configure the control system to install the device. Similarly, when you unplug a device from one port and plug it into another, you must configure the control system to identify the change.

However, some Ethernet devices use auto-configuration—when you plug the device into a port and turn on device power, the control system automatically adds that specific device at that port. Similarly, when you unplug a device and plug it into another port, the control system automatically makes that change. (You may still need to edit the device’s configuration; for example, if you have added a video decoder to the system, you should configure the decoder to indicate which monitor it is attached to.)

Unplugging a device from a port (without plugging it into another port) does not automatically remove that device from the control system. Automatic removal is not allowed because the control system cannot tell the difference between a device that has been unplugged from the node port and a device that is still plugged in but has stopped communicating.

Auto-configuration devices include:

● RTC
● Schilling HD cameras
● Video decoders
● Video computer
● Topside SD video encoder
● Schilling valve packs (8-ST, 9-ST, and 12-ST)

All auto-configuration devices are preconfigured devices.

Auto-Configuration Functions

Caution!  Always turn off node port power before you plug in or unplug a device from a node port; failing to do this may damage port hardware and/or the device.

Installing a New Auto-Configuration Device

1. Turn off power to the node port.
2. If a different device is already attached to the port, unplug that device and manually remove the device from the control system (see Removing a Device).
3. Plug the new, auto-configuration device into the node port.
4. Turn on power to the new device, and the device is automatically installed in the control system.
5. Examine the device’s properties in its configuration screen, and edit the configuration if necessary.

Replacing an Auto-Configuration Device with a Spare

To replace a device with a spare of the same device on the same node port (for example, to replace a Schilling HD camera with the same model of HD camera on the same port):

1. Turn off power to the node port.
2. Unplug the old device from the node port.
3. Plug in the new device to the same node port.
4. Turn on power to the new device. The control system recognizes that a new device of the same model has been plugged into the same port.
5. Examine the device’s properties in its configuration screen, and edit the configuration if necessary.

Replacing an Auto-Configuration Device with a Different Device Type

To remove a device from a port and replace it with a device of some other type (for example, to replace a Schilling HD camera with a video decoder):

1. Manually configure the control system to remove the old device (see Removing a Device).
2. Turn off power to the node port.
3. Unplug the old device from the node port.
4. Plug in the new device. Then:
— If the new device is an auto-configuration device, turn on power to the device. The device is automatically installed in the control system. Examine the device’s properties in its configuration screen, and edit the configuration if necessary.
— If the new device is not an auto-configuration device, you must manually install the new device in the control system (see Installing a Preconfigured Device or Installing a Custom Device).

Removing an Auto-Configuration Device Without Replacing It

Manually configure the control system to remove the device (see Removing a Device).
