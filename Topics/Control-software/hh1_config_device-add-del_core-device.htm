Core Preconfigured Devices

See Also for links to information about installing and removing devices.

Core preconfigured devices (devices that are critical to system functions) cannot be added or removed from the control system. These devices include:

● HPU reservoirs
● Instrument power equipment (IPDS or IPA)
● Position sensor array (PSA)
● Thruster control units (TCUs)
● HPU controllers
● TMS motors
● TMS latches

Note:  You cannot add or remove core preconfigured devices. You can move the connection for a core device to an open port on a node, but only to a node on the same platform. You cannot, for example, move the connection for the ROV IPDS or IPA to a node on the TMS.
