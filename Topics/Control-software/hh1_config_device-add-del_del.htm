Removing a Device

See Also for links to information about installing and removing devices.

Note:  You cannot remove a core preconfigured device (see Core Preconfigured Devices) from the ROV system.

1. Ensure that the user profile for your control station includes supervisor permissions.
2. Display any CONFIGURATION screen.
3. Tap the DEVICE tab at the top of the screen.
4. Tap the tab for the correct device category.
5. From the drop-down list, tap the button for the device.
6. Tap REMOVE DEVICE at the bottom of the screen.

If the REMOVE DEVICE button does not appear, the device is a core preconfigured device and cannot be removed from the system.
