Maximum Device Limit (for Installing Devices)

See Also for links to information about installing and removing devices.

The control system limits the maximum allowable number of a particular type of device. (For example, the ROV can contain a maximum of two depth sensors.) Some of the limits apply to the entire ROV system, while other limits apply only to a platform (like the ROV or TMS). A message will warn you if you attempt to add a device that would exceed the maximum allowable number.
