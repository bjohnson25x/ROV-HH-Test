Adding or Removing a Node or Changing a Node ID

See Also for links to information about installing and removing devices.

Adding or removing a node or changing a node’s ID number is a two-step process:

● You must use the Node Changer application (supplied in Support Files\4 - Test Software\) to configure the node’s electronics. This operation is done outside of the ROV control system, and is described in Subsea Node: Configure ID Number, RSTP and VLAN Status.
● You must then use the ROV control system’s Node Editor (described below) to tell the control system about the changes made in Node Changer.

Caution!  If the changes that you make in the Node Editor do not exactly match the changes made in Node Changer, equipment will malfunction.

Note:  You cannot operate any part of the ROV system while you are using the Node Editor. After you have made changes and exited from the Node Editor, the control system automatically restarts.

The NODE EDITOR panel provides information to the control system about changes that you made to nodes or node ID numbers in the Node Changer application. For information about displaying the NODE EDITOR panel, see individual instructions below for adding or removing a node or for changing a node ID number.

The illustration below shows an example of a NODE EDITOR panel for removing a node or changing its ID number.

For both an added node or an existing node, the NODE PARAMETERS panel lets you select a node ID number, label the node, and select the platform (ROV, TMS, topside, etc.) on which the node is located. For information about displaying the NODE PARAMETERS panel, see individual instructions below for adding a node or for changing the node ID number.

Adding a Node

1. Display any CONFIGURATION screen.
2. Tap the DEVICE tab at the top of the screen.
3. Tap EDIT NODE on the lower left side.
4. When the NODE EDITOR panel displays, tap ADD NODE.
5. When the NODE PARAMETERS panel displays, select the node number, label the node, and select the platform. Then tap OK.
6. In the NODE EDITOR panel, tap COMMIT CHANGES. The control system automatically restarts.

Removing a Node

Note:  You cannot remove a node that contains a core preconfigured device.

1. Display any CONFIGURATION screen.
2. Tap the DEVICE tab at the top of the screen.
3. Tap EDIT NODE on the screen's lower left side.
4. When the NODE EDITOR panel displays, tap the button for the node that you are removing.
5. Tap REMOVE NODE.
6. Tap COMMIT CHANGES, and the control system automatically restarts.

Changing a Node ID Number

Note:  Remember that you must use the Node Changer application (supplied in Support Files\4 - Test Software\) to configure the node’s electronics. This operation is done outside of the ROV control system, and is described in Subsea Node: Configure ID Number, RSTP and VLAN Status. You can than change the node’s ID number in the ROV control system as described below.

1. Display any CONFIGURATION screen.
2. Tap the DEVICE tab at the top of the screen.
3. Tap EDIT NODE on the lower left side.
4. When the NODE EDITOR panel displays, tap RENUMBER NODE.
5. When the NODE PARAMETERS panel displays, select a node number from the drop-down list and enter a node label. Then tap OK.
6. In the NODE EDITOR panel, tap COMMIT CHANGES. The control system automatically restarts.
