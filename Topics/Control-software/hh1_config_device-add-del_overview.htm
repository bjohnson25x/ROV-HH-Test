About Installing and Removing Devices

See Also for links to information about installing and removing devices.

Installing and/or removing ROV system devices involves two separate actions:

● You must plug in and/or unplug the devices from the node ports.
● You must configure the control system so that it contains accurate information about the hardware currently plugged into node ports.

Some devices (called auto-configuration devices, described in Installing, Moving, and Deleting an Auto-Configuration Device) are able to independently communicate with the control system (about the device type and port location) when you plug the devices into ports and turn them on. But most devices attached to node ports cannot communicate independently, and you must manually change the control system when you make device/location changes.

Caution!  If you do not accurately match device hardware on a node port with the control system information about the device, equipment may malfunction.

Caution!  Always turn off node port power before you plug in or unplug a device from a node port; failing to do this may damage port hardware and/or the device.

Note:  Before you configure the control system to install, move, remove, or configure nodes and/or devices, you may want to make a manual backup of the control system configuration setup. If your changes are not successful, you can "undo" them by reloading the previous configuration setup.

You must have supervisor permissions to install or remove a device in the control system.

Before you can install a device, you must know whether:

● The device is a preconfigured device or a custom device (as defined in Device Interface Categories).
● The device is an auto-configuration device (see Installing, Moving, and Deleting an Auto-Configuration Device).
● The device transmits serial, video, or Ethernet signals.

You can install a device in the control system before you have plugged the device into a node port. However, by plugging in the device first, you can test the device installation while you are installing the device in the control system.

After you have installed a new device in the control system, check to ensure that the device is using the proper node interface module.

Caution!  The control system cannot check to ensure that you have connected a device to a compatible interface module on a node port. (For example, the control system does not detect the connection of a device that sends serial signals to a port with a video interface module.) If you do not accurately match port hardware with the configuration information about that hardware, equipment may malfunction.
