Interface Module Locations and Quantities for Devices

See Also for links to information about signal processing and device interfaces.

A device can require either one or two interface modules to communicate. If a device requires two interface modules, each module may be classified as a “server” or “client” (see Device Signal Destinations) according to its location in the network.

Device Signal Types contains general information about the number (one or two) of interface modules required by different device interface types.
