Accessing Device Configuration Information

See Also for links to general information about configuring devices.

Some devices can be configured only by someone with supervisor permissions. If you want to configure such a device, ensure that the user profile for your control station has supervisor permissions.

To view a device's configuration information, display the device's configuration screen:

1. Display any CONFIGURATION screen. (If necessary, tap CONFIGURATION on the Main Menu.)
2. Tap the DEVICE tab at the top of the screen.
3. Either:
a. Tap INSTALLED DEVICES on the left side of the screen.
b. Tap the tab for the appropriate device category.
c. In the list of devices for that category, tap the button for the device.

or

a. Tap NODES/PORTS on the left side of the screen.
b. Tap the tab for the node that is connected to the device.
c. Tap the button for the port interface module that is connected to the device.

The configuration screen for each device displays:

● Node ID number and label.
● Node port number and label, and brief device description.
● Longer device description.
● Power settings (auto-on and fuse settings).
● For a device attached to a SIM or VIM, properties of the serial communication interface (like baud rate).
● Additional configuration (if any) specific to that device.
