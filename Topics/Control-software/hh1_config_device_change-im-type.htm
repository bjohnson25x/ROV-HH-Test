Changing Interface Module Type

See Also for links to general information about configuring devices.

Note:  If you change a port's interface module hardware to a different type (for example, you remove a SIM and install a NIM), you must inform the control system about the change.

Caution!  If the control system does not have correct information about the interface module type on each node port, equipment may malfunction.

Note:  You cannot change an interface module type while a device is installed on that interface module. Uninstall the device (or move it to a different port) and then change the interface module type.

To change a port interface module type:

1. Display any CONFIGURATION screen.
2. Tap the DEVICE tab at the top of the screen.
3. Tap NODES/PORTS.
4. Tap the button for the desired node.
5. When the list of ports displays, tap the desired port
6. From the drop-down list of port interface module types, select the correct type.
