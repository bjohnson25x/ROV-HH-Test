8-Liter Compensator/Reservoir Configuration Properties

Note:  This section applies only to configuring 8-liter compensators and reservoirs. To configure 2.2-liter compensators, see Compensator Configuration Properties.

The control system connects to all ROV-installed 8-liter compensators and 8-liter reservoirs through a position sensor array (PSA). Each PSA can have up to eight connections.

Configuration for an 8-liter compensator or reservoir consists of:

● Adding and removing a compensator or reservoir by indicating its PSA connection. See information below for instructions.
● Specifying volume. See information below for instructions.
● Setting high-volume and low-volume alarm levels using the FLIGHT screen COMPS panel (see About Compensation/Reservoir Monitoring).

You cannot remove a core device (like the HPU reservoir) that is attached to the PSA, but you can move the device from one slot on the PSA to an empty slot.

To configure the PSA for compensators and reservoirs, first display the list of installed devices.

See how
1. Display any CONFIGURATION screen. (If necessary, tap CONFIGURATION on the Main Menu.)
2. Tap the DEVICES tab near the screen top.
3. Tap INSTALLED DEVICES.

Then, configure the PSA:

1. Ensure that the user profile for your control station includes supervisor permissions.
2. Tap SENSORS.
3. When the list of sensors displays, select the PSA that is installed on the ROV (not the TMS). The screen then displays information for the PSA, including the DEVICE-SPECIFIC DETAILS section listing the devices that are attached to PSA slots.

4. To configure the PSA properties, tap EDIT CONFIGURATION.
5. Tap the PSA PROPERTIES tab.
6. As needed, you can:
● Add, remove, or move devices on the PSA. You can move a device only to a currently empty slot; to do this, tap the drop-down list for the empty slot and select the device that you want to move.
● Indicate device capacity (volume).
● Mark a device so that its volume is displayed on the dashboard.

7. Tap SAVE CHANGES.
