Viewing Device and Port Information

See Also for links to general information about configuring devices.

On any CONFIGURATION screen, tap the DEVICE tab to view information about installed devices and node port assignments.

To view devices by category, tap the INSTALLED DEVICES key.

Port color indicates interface module type:

● Gray = serial (SIM)
● Red = video (VIM)
● Blue = network (NIM)
● Green = standard-definition video (SDVIM) with flow-though penetrator (shown in touch screed port color designations as purple), used only with HDEVS
● Purple = standard-definition video (SDVIM) with feed-through penetrator, used only with HDEVS
● Gold = latch (LIM)

To view node port assignments, tap the NODES/PORTS key.
