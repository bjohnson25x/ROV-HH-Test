Editing Device Configurations

See Also for links to general information about configuring devices.

Note:  Before you add, remove, or configure nodes and/or devices, you may want to make a backup of the control system setup. If your changes are not successful, you can reload the previous setup.

1. Some devices can be configured only by someone with supervisor permissions. If you want to configure such a device, ensure that the user profile for your control station has supervisor permissions.
2. Display the device’s configuration screen (see Accessing Device Configuration Information).
3. Tap EDIT CONFIGURATION at the bottom of the screen.
4. When the device’s configuration panel appears, tap the desired tab.
5. Make the desired changes.
6. Tap SAVE CHANGES. Changes take place immediately and are retained when you restart the control system.
