MIRROR PORT Configuration Tab

See Also for links to general information about configuring devices.

For mirrored devices (where a device's data stream is sent to an additional rack node port), the MIRROR PORT tab lets you:

● Select the rack node port for the mirrored signals.
● Direct a device to take control signals from the mirrored port rather than sending data to the mirrored port. This option is typically used for a manipulator master controller. Do not enter a check mark for survey devices because they are controlled through a user tunnel.

To display the MIRROR PORT PROPERTIES tab for a device, first display the list of installed devices.

See how
1. Display any CONFIGURATION screen. (If necessary, tap CONFIGURATION on the Main Menu.)
2. Tap the DEVICES tab near the screen top.
3. Tap INSTALLED DEVICES.
4. Tap the device category on the left side of the screen.
5. When the list of devices in that category displays, select the desired device.
6. Tap EDIT CONFIGURATION.
7. Tap the MIRROR PORT PROPERTIES tab.

Note:  Do not check the MIRROR PORT CONTROLS DEVICE box for items used with survey functions.
