About Configuring Devices

See Also for links to general information about configuring devices.

Purpose: View and/or change device characteristics for all devices, including nodes.
Access: On any CONFIGURATION screen, tap the DEVICES tab.

Caution!  If you enter incorrect configuration information for a device, the device may malfunction.

Note:  Before you add, remove, or configure nodes and/or devices, you may want to make a backup of the control system setup. If your changes are not successful, you can reload the previous setup.
