POWER Configuration Tab: Setting Auto-On and Software Fuses

See Also for links to general information about configuring devices.

The POWER tab lets you configure auto-on and fuse-disable functions for some devices.

To display the POWER tab for a device, first display the list of installed devices.

See how
1. Display any CONFIGURATION screen. (If necessary, tap CONFIGURATION on the Main Menu.)
2. Tap the DEVICES tab near the screen top.
3. Tap INSTALLED DEVICES.
4. Tap the device category on the left side of the screen.
5. When the list of devices in that category displays, select the desired device.
6. Tap EDIT CONFIGURATION.
7. Tap the POWER tab.

Auto-On Function

The “auto-on” function designates certain node ports to automatically turn on power at system startup.

Caution!  Lights and other devices subject to overheating on deck should typically not be set to Auto-On.

You can individually apply auto-on only to subsea node ports; the topside rack node ports all turn on automatically at system startup.

See how
1. Display the device’s configuration screen.
2. Tap EDIT CONFIGURATION.
3. Tap the POWER TAB.
4. Install a check mark in the AUTO ON box.
5. Tap SAVE CHANGES.

To remove auto-on, remove the check mark.

If you change the device’s setting to auto-on, the change takes effect the next time that you:

● Apply system power after system startup.
● Command a platform’s subsea nodes to turn on all ports that use auto-on (see Changing Device Auto-On Status from the DEVICE POWER Panel).

If you configure a device so that it no longer uses auto-on, the change takes effect only when the control software is restarted and system power is applied.

Auto-on can also be enabled/disabled from the device power panel on the FLIGHT screen. For information on this option, see Changing Device Auto-On Status from the DEVICE POWER Panel.

Enabling and Disabling Device Software Fuses

The “fuse disable” function lets you disable software fuse protection for designated node ports. The default setting for all node ports is to have software fuses enabled.

If a device is not eligible for fuse disabling, the option does not appear in the device’s POWER tab.

All factory-supplied ROV devices run with node port software fuses enabled. However, if you have added a critical custom device with a starting (inrush) current that exceeds the port fuse rating (15 A), you can briefly disable the port fuse to allow the device to start. After the device has started, you should enable the fuse to protect node circuitry.

Caution!  During the time that the port fuse is disabled, operating a device with a very high starting current or with a run current above 15 A may damage other node components. Use extreme caution when you disable software fuses.

See how
1. Display the device’s configuration screen.
2. Tap EDIT CONFIGURATION.
3. Tap the POWER TAB.
4. Install a check mark in the FUSE DISABLE box.
5. Tap the SAVE CHANGES button.
6. After the device has started up, be sure to remove the check mark from the FUSE DISABLE box.

The fuse setting takes effect immediately and is retained when you restart the control system.
