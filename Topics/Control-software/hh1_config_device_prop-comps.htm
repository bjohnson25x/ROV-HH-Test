2.2-Liter Compensator Configuration Properties

This section applies only to 2.2-liter compensators. If your system uses 8-liter compensators (which are attached to the control system through a position sensor array or PSA), they are configured using the screen for the ROV's PSA.

To configure a 2.2-liter compensator, first, display the list of installed devices.

See how
1. Display any CONFIGURATION screen. (If necessary, tap CONFIGURATION on the Main Menu.)
2. Tap the DEVICES tab near the screen top.
3. Tap INSTALLED DEVICES.

Then, configure the compensator:

1. Tap COMPENSATORS.
2. When the list of compensators displays, select the desired compensator.
3. Tap EDIT CONFIGURATION.
4. Tap the COMP PROPERTIES tab.

5. Enter the capacity (maximum volume) of the compensator.
6. Install a check mark if you want the compensator to appear on the dashboard display of compensator volume over time.
7. Tap SAVE CHANGES.

8-Liter Compensator Configuration Properties

Because each 8-liter compensator on the ROV is attached to a position sensor array (PSA), these compensators are configured in the SENSORS configuration section (see 8-Liter Compensator/Reservoir Configuration Properties).
