ISOL-8 Pump Device Configuration Properties

See Also for links to information about configuring hydraulic device properties.

The ISOL-8 pump is used in the UHD-III hydraulic configuration to provide hydraulic circuits for auxiliary (tooling) use.

ISOL-8 pump configuration varies depending on the pump model. The two models are:

● Eight-cylinder model, which pumps fluid for two auxiliary circuits:
— Aux circuit 1 - uses six cylinders and has its own fluid reservoir.
— Aux circuit 2 - uses two cylinders and has its own fluid reservoir.
● Four-cylinder model, with a single hydraulic circuit (aux circuit 1) and its own fluid reservoir.

You must have a software license to use the eight-cylinder ISOL-8 pump.

To configure the ISOL-8 pump, first, display the list of installed devices.

See how
1. Display any CONFIGURATION screen. (If necessary, tap CONFIGURATION on the Main Menu.)
2. Tap the DEVICES tab near the screen top.
3. Tap INSTALLED DEVICES.

Then, configure the ISOL-8 pump:

1. Tap HYDRAULICS.
2. When the list of hydraulic devices displays, select the ISOL-8 pump.
3. Tap EDIT CONFIGURATION.
4. Tap the PUMP PROPERTIES tab.

Device Configuration Options, 8-Cylinder Pump

If you have mechanically configured an 8-cylinder ISOL-8 pump to mix the fluids in aux circuit 1 and aux circuit 2, you should configure aux circuit 1 to use eight cylinders. When you do this, aux circuit 2 controls become unavailable.

Cylinder Type and Availability Configuration

Scroll down in the configuration panel to access controls for cylinder type (part number) and availability (whether any cylinders are disabled).

Disable a cylinder if:

● The cylinder has been removed and a blanking plate installed in its place.
● The cylinder is not functioning properly. Control system software can then compensate for the malfunctioning cylinder.
