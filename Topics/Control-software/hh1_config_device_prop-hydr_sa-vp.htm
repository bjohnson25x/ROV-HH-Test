Sub-Atlantic Valve Pack Device Configuration Properties

See Also for links to information about configuring hydraulic device properties.

In the Sub-Atlantic 8-station valve pack and Sub-Atlantic 12-station valve pack, only the first seven valves can be configured for solenoid or proportional valves. Be sure that the entered information matches the hardware on the valve pack.

To configure a valve pack, first, display the list of installed devices.

See how
1. Display any CONFIGURATION screen. (If necessary, tap CONFIGURATION on the Main Menu.)
2. Tap the DEVICES tab near the screen top.
3. Tap INSTALLED DEVICES.

Then, configure the valve pack:

1. Tap HYDRAULICS.
2. When the list of hydraulic devices displays, select the desired valve pack.
3. Tap EDIT CONFIGURATION.
4. Tap the VALVE PACK PROPERTIES tab.

Select the desired hydraulic circuit from the drop-down list. For each valve, select the valve type.
