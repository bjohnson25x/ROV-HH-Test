Schilling Valve Pack Device Configuration Properties

See Also for links to information about configuring hydraulic device properties.

To configure a valve pack, first, display the list of installed devices.

See how
1. Display any CONFIGURATION screen. (If necessary, tap CONFIGURATION on the Main Menu.)
2. Tap the DEVICES tab near the screen top.
3. Tap INSTALLED DEVICES.

Then, configure the valve pack:

1. Tap HYDRAULICS.
2. When the list of hydraulic devices displays, select the desired valve pack.
3. Tap EDIT CONFIGURATION.
4. Tap the VALVE PACK PROPERTIES tab.

Schilling 8-ST, 9-ST, and 12-ST Valve Packs

When the Schilling 8-ST, 9-ST, and 12-ST valve packs are connected to the ROV control system, they automatically provide information about valve type for all installed valves. This means that when you configure one of these valve packs, you need only select a hydraulic circuit from a drop-down list.

Schilling 8-ST Valve Pack

The Schilling 8-ST valve pack is typically plumbed into the main hydraulic circuit to operate a rate-controlled manipulator.

Schilling 9-ST Valve Pack

The Schilling 9-ST valve pack is typically plumbed into an auxiliary (tooling) hydraulic circuit to operate hydraulic tools.

Schilling 12-ST Valve Pack

The Schilling 12-ST valve pack is typically plumbed into ISOL-8 pump hydraulic circuits to operate tooling.

Each valve on the Schilling 12-ST valve pack is mechanically plumbed into one of two hydraulic circuits on the ISOL-8 pump. Use the screen configuration options to select the two hydraulic circuits.

For information on how to mechanically plumb a valve into a hydraulic circuit, see the appropriate service instruction.

Schilling 3-ST Valve Pack

To configure the Schilling 3-ST valve pack, select the desired hydraulic circuit from the drop-down list.
