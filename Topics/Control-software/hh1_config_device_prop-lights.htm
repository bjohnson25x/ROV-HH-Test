Lighting Configuration Properties (Including LCUs)

See Also for links to information about configuring lights, including light groups and presets.

For general information on editing device configurations, see Editing Device Configurations.

Individual lights do not have unique configurable properties. However, you can configure a light using the tabs for SERIAL PROPERTIES, MIRROR PORT, and POWER.

If the ROV system contains a lighting control unit (LCU), it must be configured.

Designating the LCU Electrical Circuit

To configure an LCU, first display the list of installed devices.

See how
1. Display any CONFIGURATION screen. (If necessary, tap CONFIGURATION on the Main Menu.)
2. Tap the DEVICES tab near the screen top.
3. Tap INSTALLED DEVICES.

Then, configure the LCU:

1. Tap LIGHTING.
2. When the list of lighting devices displays, select the LCU.
3. Tap EDIT CONFIGURATION.
4. Tap the LCU PROPERTIES tab.

The correct setting used to designate the LCU electrical circuit varies depending on which LCU model is installed on the ROV system.

For an HD or UHD-III:
● If the ROV system uses LCU 101-6016, the LCU performs line isolation monitoring for the electrical circuit to which it is attached. You must tell the control system which electrical circuit is being monitored by the LCU. From the drop-down box, select the electrical circuit (AC1 or AC2) to which the LCU is attached. If the LCU is not currently attached to an AC circuit, select NONE.
● If the ROV system uses LCU 101-9518, the LCU cannot perform line isolation monitoring. In this case, you should not identify the electrical circuit to which the LCU is attached. From the drop-down box, select NONE (even if the LCU is attached to AC1 or AC2).
For a UHD-II:
● If the ROV system uses LCU 101-6016, the LCU can perform line isolation monitoring for the 120-VAC electrical circuit. From the drop-down box, select AC1.
● If the ROV system uses LCU 101-9518, the LCU cannot perform line isolation monitoring. From the drop-down box, select NONE.

LCU With an AC J-box Connection

If an AC J-box is attached to the LCU, you can designate node ports for the control signals for up to four AC J-box circuits. You must also install a check mark for each LCU port that is connected to a light.
