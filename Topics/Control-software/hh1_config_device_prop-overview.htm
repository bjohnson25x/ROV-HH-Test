PROPERTIES Configuration Tab

See Also for links to configuration options for other device types.

Many devices have a PROPERTIES tab, which contains configuration options that are specific to the device type. For example, if you are configuring a depth sensor, the DEPTH SENSOR PROPERTIES tab lets you configure the readings to be based on either water pressure or depth, and also lets you enter a depth sensor offset.
