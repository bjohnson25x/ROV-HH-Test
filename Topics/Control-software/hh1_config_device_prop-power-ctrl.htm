Power Control Configuration Properties

For general information on editing device configurations, see Editing Device Configurations.

Some power control devices do not have configurable properties. However, you must configure the IPDS and AC J-box.

To configure a power device, first, display the list of installed devices.

See how
1. Display any CONFIGURATION screen. (If necessary, tap CONFIGURATION on the Main Menu.)
2. Tap the DEVICES tab near the screen top.
3. Tap INSTALLED DEVICES.

Then, configure the power device:

1. Tap POWER.
2. When the list of power devices displays, select the desired device.
3. Tap EDIT CONFIGURATION.
4. Tap the PROPERTIES tab.

AC J-Box Configuration Properties

For each port on an AC J-box port is attached to a device, use the drop-down list for that port to indicate which subsea node port allows power to be routed to that device.

To operate a device, turn on power to the associated node port.
