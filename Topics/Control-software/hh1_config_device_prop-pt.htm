Pan and Tilt Configuration Properties

See Also for links to information about pan and tilt control and configuration.

For general information on editing device configurations, see Editing Device Configurations.

The pan actuator and the tilt actuator are considered to be separate devices and are configured separately.

To configure a pan or tilt, first, display the list of installed devices.

See how
1. Display any CONFIGURATION screen. (If necessary, tap CONFIGURATION on the Main Menu.)
2. Tap the DEVICES tab near the screen top.
3. Tap INSTALLED DEVICES.

Then, configure the pan or tilt:

1. Tap PAN/TILT UNITS.
2. When the list of pan and tilt devices displays, select the desired device.
3. Tap EDIT CONFIGURATION.
4. Tap the PAN & TILT PROPERTIES tab.

If necessary, indicate the location of the pan and tilt:

● If the pan and tilt is mounted in a factory-installed location (see Pan and Tilt Configuration Properties), select that location.
● If you have moved the pan and tilt away from a factory-installed location, select CUSTOM for the location.

Defining a Custom Pan and Tilt Location

If you have moved a pan and tilt away from a factory-installed location, you must enter information about how far and in what directions you moved the pan and tilt.

If you select CUSTOM for the pan and tilt location (see figure below), you must enter the distance (in the x, y, and z directions) of the new location from the factory-installed location.

Tapping HELP displays a screen showing the lower location for a factory-installed pan and tilt. Using the shown point, measure the distances (in the x, y, and z directions) of the CUSTOM pan and tilt from that point. Then tap X, Y, and Z to enter the distances. For example, if the pan and tilt has been moved down 0.01 meters and forward .64 meters, you would enter an X value of 0.64, a Y value of zero, and a Z value of -0.01.
