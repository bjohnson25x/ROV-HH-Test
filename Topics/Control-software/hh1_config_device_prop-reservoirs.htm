Reservoir Configuration Properties (Except 8-Liter)

For general information on editing device configurations, see Editing Device Configurations.

To view fluid levels for a compensator or reservoir or to set high-volume and low-volume alarm levels, use the FLIGHT screen COMPS panel (see About Compensation/Reservoir Monitoring).

Note:  This section on reservoir configuration properties does not apply to 8-liter reservoirs, which are attached to the ROV control system through a position sensor array (PSA). The 8-liter reservoirs are configured using the screen for the ROV's PSA.

To configure a reservoir, first display the list of installed devices.

See how
1. Display any CONFIGURATION screen. (If necessary, tap CONFIGURATION on the Main Menu.)
2. Tap the DEVICES tab near the screen top.
3. Tap INSTALLED DEVICES.

Then, configure the reservoir:

1. Tap HYDRAULICS.
2. When the list of hydraulic devices displays, select the desired reservoir.
3. Tap EDIT CONFIGURATION.
4. Tap the RESERVOIR PROPERTIES tab.

5. Enter the capacity (maximum volume) of the reservoir.
6. Install a check mark if you want the reservoir to appear on the dashboard display of reservoir volume over time.
7. Tap SAVE CHANGES.
