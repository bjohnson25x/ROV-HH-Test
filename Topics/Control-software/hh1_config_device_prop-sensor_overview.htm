Sensor Configuration Properties, General

Information in the sensor PROPERTIES screen is specific to the type of sensor.

Sensor configuration properties vary depending on sensor types.

To configure a sensor, first, display the list of installed devices.

See how
1. Display any CONFIGURATION screen. (If necessary, tap CONFIGURATION on the Main Menu.)
2. Tap the DEVICES tab near the screen top.
3. Tap INSTALLED DEVICES.

Then, configure the sensor:

1. Tap SENSORS.
2. When the list of sensor devices displays, select the desired device.
3. Tap EDIT CONFIGURATION.
4. Tap the PROPERTIES tab.
