Configuring TMS Position Sensor Array (PSA) Properties (Electric TMS Only)

See Also for links to more information about configuring sensor device properties.

For an electric TMS, the ROV control system connects to all TMS sensors (latch position sensors, motor speed sensors, etc.) through a position sensor array (PSA). Each PSA can connect to eight sensors.

Because TMS latches and motors are core devices (see Core Preconfigured Devices), you cannot add or remove a TMS latch or motor on the PSA. You can, however, move a latch or motor to another position on the same PSA.

To configure the PSA for TMS sensors, first display the list of installed devices.

See how
1. Display any CONFIGURATION screen. (If necessary, tap CONFIGURATION on the Main Menu.)
2. Tap the DEVICES tab near the screen top.
3. Tap INSTALLED DEVICES.

Then, configure the PSA:

1. Ensure that the user profile for your control station includes supervisor permissions.
2. Tap SENSORS.
3. When the list of sensors displays, select the PSA that is installed on the TMS (not the ROV). The screen then displays information for the PSA, including the DEVICE-SPECIFIC DETAILS section listing the sensors that are attached to PSA slots.

4. To configure the PSA properties, tap EDIT CONFIGURATION.
5. Tap the PSA PROPERTIES tab.
6. Except for core devices, you can add, remove, or move devices on the PSA. You can move a device only to a currently empty slot; to do this, tap the drop-down list for the empty slot and select the device that you want to move.

7. Tap SAVE CHANGES.
