To configure a camera, first, display the list of installed devices.

See how
1. Display any CONFIGURATION screen. (If necessary, tap CONFIGURATION on the Main Menu.)
2. Tap the DEVICES tab near the screen top.
3. Tap INSTALLED DEVICES.

Then, configure the camera:

1. Tap VIDEO.
2. When the list of video devices displays, select the desired camera.
3. Tap EDIT CONFIGURATION.
4. Tap the CAMERA PROPERTIES tab.

Camera Configuration Screens in TAVS

In TAVS, cameras attach to VIMs, and each camera requires connection to a subsea node VIM and a rack node VIM. (The rack node VIM port is called the client VIM.) When you configure a camera, you can:

● Select a pan and tilt for the camera using the CAMERA PROPERTIES tab.

● Add or change the camera's rack node port (client VIM) using the VIDEO TUNNEL tab.
● Set the video format (NTSC or PAL) using the VIDEO TUNNEL tab.

If you want to select a new client VIM for a camera, display the VIDEO TUNNEL tab. Tap REMOVE and then select a new client VIM from the drop-down list.

For more information about configuring a subsea camera in TAVS, see TAVS Subsea Camera Installation and Configuration.

Camera Configuration Screens in HDEVS

In HDEVS, you can configure HD cameras and SD cameras.

An HD camera (like the Schilling HD Ethernet camera) outputs Ethernet signals and attaches to a subsea NIM. When you configure an HD camera, you can use the CAMERA PROPERTIES tab to select a pan and tilt and to perform functions like auto focusing the camera.

On the main configuration screen for the Schilling HD Ethernet camera, you can also select a camera profile.

An SD cameras (like a manipulator wrist camera) attaches to an SDVIM. For an SD camera, the CAMERA PROPERTIES tab is limited to selecting a pan and tilt.

In HDEVS, you do not configure the video format (NTSC or PAL) for an individual camera. Instead, you configure video format by configuring the video decoder attached to the display monitor for the camera (see Video Decoder Configuration Properties).
