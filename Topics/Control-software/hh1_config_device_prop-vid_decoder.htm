Video Decoder Configuration Properties

See Also for links to information about configuring video device properties for HDEVS.

Note:  Video decoders are used only with HDEVS (high-definition Ethernet video system).

With HDEVS, in addition to configuring cameras, you must also configure the video decoder that processes each camera's output for display. When you configure the video decoder, you can:

● Set video format (NTSC or PAL)
● Select a monitor to display output from the decoder

To configure a video decoder, first, display the list of installed devices.

See how
1. Display any CONFIGURATION screen. (If necessary, tap CONFIGURATION on the Main Menu.)
2. Tap the DEVICES tab near the screen top.
3. Tap INSTALLED DEVICES.

Then, configure the decoder:

1. Tap VIDEO.
2. When the list of video devices displays, select the desired decoder.
3. Tap EDIT CONFIGURATION.
4. Tap the DECODER PROPERTIES tab.
