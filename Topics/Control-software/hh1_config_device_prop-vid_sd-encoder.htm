SD Video Topside Encoder Configuration Properties

See Also for links to information about configuring video device properties for HDEVS.

SD encoders are used only with HDEVS.

The SD video topside encoder accepts up to four video streams from SD deck cameras. To configure the encoder, you can label the video streams.

To configure an SD encoder, first, display the list of installed devices.

See how
1. Display any CONFIGURATION screen. (If necessary, tap CONFIGURATION on the Main Menu.)
2. Tap the DEVICES tab near the screen top.
3. Tap INSTALLED DEVICES.

Then, configure the SD encoder:

1. Tap VIDEO.
2. When the list of video devices displays, select the SD encoder.
3. Tap EDIT CONFIGURATION.
4. Tap the TOPSIDE CAMERA PROPERTIES tab.
