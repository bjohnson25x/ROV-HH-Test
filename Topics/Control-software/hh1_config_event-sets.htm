Configuring Video Event Sets and Event Types (HDEVS Only)

See Also for links to more information about using event place markers in recorded video. This information applies only to ROV systems with HDEVS.

Purpose: This function lets you:

● Create and delete event sets.

● Create and delete event types inside.
● Change the description for an event type.

Access: On any CONFIGURATION screen, tap the EVENT SETS tab.

This information applies only to ROV systems that have HDEVS rather than TAVS.

Before you can put event place markers in recorded video, you must organize events by defining individual event types and grouping those types into event sets.

For example, you could create an event set called Land and Lock Tree and BOP. Within that event set, you could create individual event types that included:

● Install VX Ring in Wellhead
● Orient and Land Tree
● Remove Tree Running Tool

Creating an Event Set

1. Display any CONFIGURATION screen.
2. Tap the EVENT SETS tab near the top of the screen.
3. Tap NEW EVENT SET.
4. Enter a name for the event set, and then enter a description explaining the general nature of the event set.

After you create an event set, you should create one or more event types that are related to the event set.

Creating an Event Type

To create an event type for an event set:

1. Display any CONFIGURATION screen.
2. Tap the EVENT SETS tab near the top of the screen.
3. On the screen left side, tap the name of the event set that will contain the event type. The event set then displays on the screen, along with any event types already created for that event set.
4. Tap NEW EVENT TYPE.
5. Enter a name for the event type.
6. If you wish to use a description for the event type, tap its description area and enter the description.

When you insert an event place marker into recorded video, the event type and its description are inserted into the video.

Changing Event Sets and Event Types

To change information for event sets and event types, first display any CONFIGURATION screen and then tap the EVENT SETS tab.

● To change a name or description for an event set or event type, tap the existing text for that item on the screen.
● To delete an event set, select the event set name in the left-hand list and tap DELETE EVENT SET. All event types in that event set are also deleted.
● To delete an event type, first select the event set that contains the event type. Then tap DELETE for the event type.
