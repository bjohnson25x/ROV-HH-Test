Setting Heading Options

See Also for links to information about setting flight control parameters.

Heading Offset

The HEADING OFFSET panel lets you add an offset to the standard magnetic compass heading. You can use this feature to compensate for a known magnetic irregularity or to coordinate or match headings with a compass on another vehicle.

Changes in the heading offset are automatically applied to:

● All compass graphic and digital displays of heading
● Controls that use the compass heading (like the HEADING auto function)

Ignore Magnetic Heading

Note:  This control works only if the ROV contains a compass.

In standard AUTO HEADING mode, the control system uses input from the magnetic compass and the motion reference unit (MRU) yaw rate sensor. When you put a check mark in the IGNORE MAGNETIC HEADING box, the control system ignores input from the magnetic compass and bases the heading only on MRU input. This state can continue for only 5 minutes. After 5 minutes, the control system returns to using magnetic compass information and turns off AUTO HEADING.

You can use the IGNORE MAGNETIC HEADING feature when large metal structures or local magnetic irregularities make input from the magnetic compass unreliable.

Note:  Headings based only on an MRU may drift over time. For more information, see the manufacturer's specifications for the MRU used with this system.
