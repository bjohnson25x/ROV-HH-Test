Setting Maximum Thruster Power and Displacement Speed

See Also for links to information about setting flight control parameters.

Setting Maximum Power

The MAX POWER control sets the maximum power that can be applied to the system from flight joystick inputs.

Ensure that the user profile for your control station includes supervisor permissions.

1. On any CONFIGURATION screen, tap the GENERAL tab.
2. On the list on the left side of the screen, tap FLIGHT CONTROL.
3. In the MAX POWER panel, tap buttons to increase or decrease the percentage of maximum power that can be applied to each thruster.

Setting Maximum Displacement Speed

The DISPLACE auto function moves the ROV a specific distance and/or direction from positions set in active STATIONKEEP, AUTO DEPTH, and AUTO ALTITUDE modes. Once displacement begins, thruster rpm is limited to a maximum displacement speed until the ROV reaches the displacement destination.

To set the maximum displacement speed:

1. Ensure that the user profile for your control station includes supervisor permissions.
2. On any CONFIGURATION screen, tap the GENERAL tab.
3. On the list on the left side of the screen, tap FLIGHT CONTROL.
4. In the MAX DISPLACEMENT SPEED panel, tap buttons to increase or decrease the percentage of maximum speed that can be used for DISPLACE operations.

Typically, you should use lower speeds for small displacement distances and greater speeds for larger displacements.
