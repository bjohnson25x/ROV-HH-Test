About Setting Flight Control Parameters

See Also for links to information about setting flight control parameters.

Purpose: The flight control configuration screen lets you:

● Specify sensors used by the control system to calculate positions and directions.
● Set adjustments to the ROV's center of mass
● Set the maximum available power for thrusters
● Set the maximum speed for the DISPLACE function
● Set a heading offset
● Turn off compass input to the control system
Access: On any CONFIGURATION screen, tap the GENERAL tab and then tap FLIGHT CONTROL.

To set any flight control parameter:

1. Ensure that the user profile for your control station includes supervisor permissions.
2. On any CONFIGURATION screen, tap the GENERAL tab.
3. On the list on the left side of the screen, tap FLIGHT CONTROL.
