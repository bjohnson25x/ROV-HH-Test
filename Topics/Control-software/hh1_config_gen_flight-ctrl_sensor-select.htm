Selecting Sensors to Use for Flight Control

See Also for links to information about setting flight control parameters.

Note:  Sensor quantity and type are different for each ROV system. Your system may not use all sensors discussed in this section.

If the ROV system is equipped with more than one flight sensor of a particular type (for example, two DVLs or a DVL and an altimeter), you must use the SENSOR SELECTION panel to select the sensors that the control system will use for various aspects of flight control.

All sensors used in flight control are integrated with the control software at the factory. They cannot be added to the ROV system later.

When the ROV is equipped for Mid-Water StationKeep:

● If the inertial navigation instrument is functioning, select it as the flight sensor for heading, altitude, and forward/aft/lat.
● If the inertial navigation instrument is not functioning, use the DVL for altitude and lateral.

The table below shows some dependencies between flight sensor panel selections. Also see AUTO ALTITUDE Auto Function.

Flight Sensor Dependencies

FLIGHT Parameter

 Dependency
LATERAL The DVL or inertial navigation instrument selected in this panel is also automatically selected in the ALTITUDE panel unless an altimeter is already selected in the ALTITUDE panel.
HEADING

Selecting “COMPASS” automatically includes input from the MRU (gyro).

Some MRUs provide heading data as well as pitch/roll data. In this case, selecting “MRU” ensures that all heading and pitch/roll data are input from the same device.

ALTITUDE

The Doppler velocity log (DVL) or inertial navigation instrument selected in this panel is also automatically selected in the FORWARD/AFT/LAT panel. If you select an altimeter for ALTITUDE, you can select any DVL or inertial navigation instrument in the FORWARD/AFT/LAT panel.
DEPTH None

To select a sensor:

1. Ensure that the user profile for your control station includes supervisor permissions.
2. On any CONFIGURATION screen, tap the GENERAL tab.
3. On the list on the left side of the screen, tap FLIGHT CONTROL.
4. Tap the desired button in the SENSOR SELECTION panel.
5. Select a sensor from the drop-down list.
