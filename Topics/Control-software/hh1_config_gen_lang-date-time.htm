Setting Language and Date/Time

Note:  The LANGUAGE function (which allows you to select an alternate language) is available only if a license for this feature has been purchased.

Note:  If your system has the extended language set license, the LANGUAGE setting is part of each user profile.

The LANGUAGE panel and DATE/TIME panel let you:

● Set the language for screen display. This selection applies only to the current control station (left or right) and to the current user profile.
● Set the desired date and time.
1. On any CONFIGURATION screen, tap the GENERAL tab.
2. On the list on the left side of the screen, tap DATE/TIME. (If your ROV system has the optional extended language set license, the button label is LANGUAGE/DATE/TIME.)
3. Select items from the drop-down lists.
