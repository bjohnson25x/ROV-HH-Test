Installing and Viewing Software Feature Licenses

Licenses are available to add optional software features (like Mid-Water StationKeep, the extended language set, or the 8-cylinder ISOL-8 pump) to the ROV control system.

Some licenses are installed by typing a license key on the touchscreen, while others require that you insert a license dongle into one of the client computers.

Viewing a List of Installed Licenses

To view a list of currently installed licenses:

1. Ensure that the user profile for your control station includes supervisor permissions.
2. On any CONFIGURATION screen, tap the GENERAL tab.
3. On the list on the left side of the screen, tap LICENSING to display the INSTALLED LICENSES panel. This panel displays any already-installed licenses for optional software features.
4. Tap the license name to see license details.

Installing  a Feature License

Installing a License by Typing a Key

1. Ensure that the user profile for your control station includes supervisor permissions.
2. On any CONFIGURATION screen, tap the GENERAL tab.
3. On the list on the left side of the screen, tap LICENSING to display the INSTALLED LICENSES panel. This panel displays any already-installed licenses for optional software features.
4. Tap INSTALL LICENSE.
5. When the screen keyboard displays, type the key for the feature license. The new feature then becomes available and is listed in the INSTALLED LICENSES panel.

The new software takes effect immediately, and you do not need to reboot the control system computers.

If you accidentally uninstall a software license and you cannot find the license key to reinstall it, contact FMC Schilling Technical Support.

Installing a License Using a Dongle

1. Ensure that the user profile for your control station includes supervisor permissions.
2. Insert the license dongle into one of the client computers. (Both control stations can use the license even though the dongle is installed in only one client computer.)
3. After a brief pause, the new feature becomes available and is listed in the INSTALLED LICENSES panel.

Typically, you should leave the license dongle in the client computer instead of removing and replacing the dongle with each operating session. You can remove the dongle if:

● The feature license is being transferred to another ROV system.
● The client computer has failed and is being replaced with another client computer.

If the client computer containing the license dongle stops communicating with the ROV control system, the license will continue to work for the current operating session (until you shut down or reboot the client computer). If the client computer continues to be unavailable after reboot, you can remove the license dongle and insert it into the second client computer.

Uninstalling a Feature License

1. Ensure that the user profile for your control station includes supervisor permissions.
2. On any CONFIGURATION screen, tap the GENERAL tab.
3. On the list on the left side of the screen, tap LICENSING to display the INSTALLED LICENSES panel. This panel displays any already-installed licenses for optional software features.
4. Tap the name of the license that you want to uninstall.
5. Tap REMOVE LICENSE.

If you remove a feature license, you remove the existing configuration for devices and control system screens that are made accessible only by that license. (For example, if you remove the license for the 8-cylinder ISOL-8 pump, you can no longer access the screen for BOP intervention. Any configuration information for that screen is then lost.)

Before you remove a feature license, you may want to save a configuration backup. If you reinstall the license, you can restore the configuration backup that contains configuration information for that license.
