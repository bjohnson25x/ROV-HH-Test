Adding, Replacing, or Removing a Logo

This function adds, replaces, or removes a logo in the lower left corner of the touch screens. The control system automatically sizes the logo for proper display.

The logo file must be in JPG, PNG, GIF, or BMP format.

Note:  You must add or remove logos separately for the left and right control stations.

Adding or Replacing a Logo

Note:  The instructions below show how to load a logo onto the left control station. To load a logo onto the right control station, substitute "right client computer" and "right control station" in the instructions.

1. Ensure that the user profile for your control station includes supervisor permissions.
2. Put a USB drive or CD containing the logo into the left client computer.
3. On any CONFIGURATION screen for the left control station, tap the GENERAL tab.
4. On the list on the left side of the screen, tap LOAD LOGO.
5. Tap LOAD NEW LOGO.
6. Browse to the desired logo file on the left client computer. The list shows you all JPG, PNG, GIF, and BMP files on the CD drive and/or USB drive that you inserted.
7. Select the desired image file and tap the OPEN button. The screen then displays a preview of the selected file. (The logo is not displayed in color.)
8. Tap APPLY NEW LOGO. The new logo replaces any previously displayed logo. The logo is not displayed in color.
9. If desired, repeat the procedure for the other control station using the other client computer.

Removing a Logo

To remove a logo without replacing it with another logo:

1. Ensure that the user profile for your control station includes supervisor permissions.
2. On any CONFIGURATION screen for the desired control station, tap the GENERAL tab.
3. On the list on the left side of the screen, tap LOAD LOGO.
4. Tap REMOVE CURRENT LOGO.
5. If desired, repeat the procedure with the other client computer.
