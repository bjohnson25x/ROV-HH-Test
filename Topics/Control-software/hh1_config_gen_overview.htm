About Configuring General System Parameters

See Also for links to information about configuring items using the GENERAL tab.

Purpose: Setting general operating parameters. You can:

● Set units of measure for display (like distance, volume, velocity, and pressure)
● Set language (optional feature)
● Set date and time
● Load a new logo for display on the touch screens
● Set the sound type of alert sounds
● Set environmental data (latitude, longitude, and water density)
● Set flight control parameters (like adjusting center of mass, setting maximum power, and selecting sensors for reporting flight data)
● View installed software licenses for optional features (like Mid-Water StationKeep) and install new licenses
● Set TMS parameters (like tether length and minimum tether to be left on drum)
● Set HPU parameters (like maximum pressure)
Access: On any CONFIGURATION screen, tap the GENERAL tab.

Note:  Some items are available only if the user profile for your control station includes supervisor permissions.

To select the desired category of items to configure:

1. If the item that you want to configure requires supervisor permissions, ensure that the user profile for your control station includes supervisor permissions.
2. On any CONFIGURATION screen, tap the GENERAL tab.
3. On the list on the screen left, tap the name of the desired category.

Typically, changed settings take effect immediately, and you do not need to reboot the control system computers.
