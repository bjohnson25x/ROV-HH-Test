Setting Alert Sounds

The control system uses sounds to alert you to potential problems and dangerous conditions.

Caution!  The client computer that you select to issue sound alerts must be attached to speakers. If it is not, you will be unable to hear the sound alerts.

Alerts can apply to the following levels of severity:

● Informational (conditions of interest to operators)
● Warnings (potential problems)
● Alarms (conditions that need immediate attention)

You can set:

● The level of severity below which an alert will not sound. For example, you can specify that an alert will sound for warnings and alarms, but not for information.
● The client computer (left or right) that will sound the alert. Ensure that this client computer is attached to speakers. (Typically, speakers are attached to the right client computer.) If you specify a client computer that is not attached to speakers, you will be unable to hear the alerts.
● The file that provides the sound. This sound applies to all alerts of all levels of severity (for example, the sound for a warning is the same as the sound for an alarm).

Note:  To adjust the loudness of the alert sound, use a screwdriver to adjust speaker volume. Speakers are located on the shelf between the two control stations.

To change a sound setting:

1. On any CONFIGURATION screen, tap the GENERAL tab.
2. On the list on the left side of the screen, tap SOUNDS.
3. Tap the button for the desired sound change and select from the drop-down list.

4. For the MINIMUM ALARM SEVERITY button, select the lowest level for an alert to sound; that level, and all levels above it, will then cause an alert to sound. For example, select "warning" to sound an alert for warnings and alarms but not for informational items.
