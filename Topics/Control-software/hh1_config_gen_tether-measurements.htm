Setting Tether Measurements (Using Length or Wraps)

See Also for links to information about configuring TMS tether.

For the TMS to operate correctly, you must tell the control system how long the tether is and how much you want to leave on the drum.

1. Before you change values for TMS tether settings, use the FLIGHT screen TMS CONTROL panel to disable TMS operation. This action disables power to the drum motor, pinch wheel motor, and latches.
2. Ensure that the user profile for your control station includes supervisor permissions.
3. On any CONFIGURATION screen, tap the GENERAL tab.
4. On the list on the left side of the screen, tap TMS.
5. Use the LENGTH/LAYER button on the panel to specify whether you want to view tether values in LENGTH (meters or feet) or LAYERS (number of layers and wraps). A wrap is a partial layer (see Counting Layers and Wraps). Using the LENGTH/LAYER button, select your preferred method.

6. Enter data for the tether lengths or layers and wraps.

Counting Layers and Wraps

If you are entering values in layers and wraps, be careful to use the correct numbering scheme.

● Layers are always in whole numbers, starting with 1. (There is no layer zero.) A partial layer (where wraps do not go all the way across) is counted as a layer.
● Wraps can be fractions of numbers, starting with zero.

For example:

● If a drum has 12 full layers and a partial layer with 18 and a half wraps, you would enter 13 for the layers (12 for the full layers and 1 for the partial layer) and 18.5 for the wraps.
● If a drum has 12 full layers and no partial layer with wraps, you would enter 12 for the layers and 0 for the wraps.
● If a drum has no full layers and a partial layer of 14 and a third wraps, you would enter 1 for the layers (since you have a partial layer) and 14.3 for the wraps.
