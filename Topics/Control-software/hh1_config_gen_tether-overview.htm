About Configuring TMS Tether Settings

See Also for links to information about configuring TMS tether.

See Also for links to information about operating and controlling the TMS.

For information on configuring TMS sensors on an electric TMS (for example, latch sensors or the drum motor sensor), see Configuring TMS Position Sensor Array (PSA) Properties (Electric TMS Only).

By configuring TMS tether settings, you provide data that the control system uses to ensure safe tether operations.

Set tether data when you:

● Install a new tether.
● Reterminate or shorten the tether.
● Need to change the length of tether that must always remain on the drum.
● Manually spool tether off the drum while the control system is shut down (generates an alarm alerting you to change tether settings)
● Separate the ROV from the TMS while the control system is shut down (generates an alarm alerting you to change tether settings)

The TOTAL TETHER LENGTH and MINIMUM TETHER ON DRUM settings take effect immediately and are retained when you restart the control system.

1. Before you change values for TMS tether settings, use the FLIGHT screen TMS CONTROL panel to disable TMS operation. (This action disables power to the drum motor, pinch wheel motor, and latches.)
2. Ensure that the user profile for your control station includes supervisor permissions.
3. On any CONFIGURATION screen, tap the GENERAL tab.
4. On the list on the left side of the screen, tap TMS.
