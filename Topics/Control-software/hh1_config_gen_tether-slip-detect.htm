Using Tether Slip Detection

See Also for links to information about configuring TMS tether.

Sensors on the TMS can detect when tether movement is slipping so that the control system can automatically stop tether movement. You can turn on tether slip detection using the TMS configuration screen.

If you have enabled pinch wheel slip detection, the control system stops TMS motors when slippage is detected. The TMS is not disabled, and no fault condition is generated.

For an electric TMS only, if you have enabled diamond screw slip detection, the control system disables TMS operation when slippage is detected. A fault condition is generated.

To enable slip detection:

1. Ensure that the user profile for your control station includes supervisor permissions.
2. On any CONFIGURATION screen, tap the GENERAL tab.
3. On the list on the left side of the screen, tap TMS.
4. Put check marks in the desired boxes for slip detection.
