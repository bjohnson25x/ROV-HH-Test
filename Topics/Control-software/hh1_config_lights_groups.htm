By putting some lights into a group, you can control those lights together by:

● Turning the grouped lights on or off
● Changing intensity of all lights in the group

Creating a Light Group

1. On any CONFIGURATION screen, tap the LIGHTS tab.
2. Tap GROUPS.
3. Tap NEW GROUP.
4. When the panel keyboard displays, enter a name for the group.
5. Enter a check mark next to each light that you want to include in the group.

Modifying a Light Group

1. On any CONFIGURATION screen, tap the LIGHTS tab.
2. Tap GROUPS.
3. Tap the button for the group that you want to modify.
4. To change the group name, tap GROUP LABEL and use the pop-up keyboard to change the name.
5. To add a light to the group, tap the check box next to the light to add a check mark. To delete a light from the group, tap a check mark to remove it.
