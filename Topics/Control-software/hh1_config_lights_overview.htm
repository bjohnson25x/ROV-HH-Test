About Configuring Light Groups and Presets

See Also for links to information about configuring lights, including light groups and presets.

Purpose of the lights configuration: For the ROV and TMS, lets you:

● Create light groups so that lights can be controlled together.
● Create light presets (determine which lights are turned on and at which intensities).
Access: On any CONFIGURATION screen, tap the LIGHTS tab.

Locations for Light Control/Configuration Information

For information on configuring a lighting control unit (LCU) or configuring the properties of an individual light, see Lighting Configuration Properties (Including LCUs).

For information on configuring an individual light for auto on or for fuse disabling, see POWER Configuration Tab: Setting Auto-On and Software Fuses.

For information on operating lights, see About Light Control.
