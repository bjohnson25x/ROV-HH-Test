Configuring Light Presets

See Also for links to information about configuring lights, including light groups and presets.

See Also for links to information about configuring and using light presets.

Light presets specify preconfigured on/off and intensity settings for one or more lights.

Note:  Lights that are not included in a preset are not affected when you apply that preset. For example, if you create a preset that includes only forward lights, side-facing lights are not affected when you apply the preset for forward lights. If you want to affect both forward and side-facing lights at the same time, you must include them all in the same preset.

You can create up to five light presets.

Creating a Light Preset from the FLIGHT Screen

1. On the FLIGHT screen, tap LIGHTS.
2. Turn on the lights that you want to include in the preset.
3. For each dimmable light, set the desired intensity.
4. Tap PRESETS.
5. On the drop-down list, tap NEW PRESET.
6. When the pop-up keyboard displays, enter a name for the preset.

The new preset contains the light settings.

Creating a Light Preset from the CONFIGURATION Screen

1. On any CONFIGURATION screen, tap the LIGHTS tab.
2. Tap PRESETS.
3. Tap NEW PRESET.
4. When the pop-up keyboard displays, enter a name for the preset.
5. Lights with check marks belong to the light preset. Tap a check box to add or delete a check mark.
6. For each checked light, set the desired on/off state by tapping the button label. (If a dimmable light is off, intensity is automatically zero.)
7. For dimmable lights that are on, set light intensity by tapping the down arrow next to the light and moving the intensity slider.

Modifying a Light Preset

1. On any CONFIGURATION screen, tap LIGHTS.
2. Tap PRESETS.
3. Tap the button for the preset that you want to modify.
4. To change the preset name, tap PRESET LABEL and use the pop-up keyboard to change the name.
5. To add a light to the preset, tap the check box next to the light to add a check mark. To delete a light from the preset, tap a check mark to remove it.
6. To change the on/off state of a checked light, tap the button label.
7. To change the intensity of a dimmable light, tap the down arrow next to the light and move the intensity slider.
