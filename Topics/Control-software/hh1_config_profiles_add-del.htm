Creating and Deleting User Profiles

See Also for links to information about setting up and using a user profile.

Creating a User Profile

Note:  Only users with supervisor access can create a new user profile.

When you create a new user profile, the profile settings in force at that time become the settings for the new user profile. Profile settings are:

● Access level (supervisor or operator).
● Language preference (if the control system is licensed for the optional extended language set).
● Fight joystick settings for deadband and low-speed gains.
● Tether settings for (1) selecting either the foot pedal or joystick trigger, and for (2) setting trigger gain.
● Flight joystick camera assignments for selecting which cameras are assigned to joystick switch positions.
● Flight joystick vertical thrust control for selecting ASCEND or DESCEND.
● Flight joystick assignment of auto function on/off for STATIONKEEP, AUTO ALTITUDE, AUTO DEPTH, or AUTO HEADING.
● Flight joystick assignment for triggering event markers for unexpected events in recorded video.

If you change any of the above settings while you are logged on as the current user, those new settings are saved in your profile when you log off.

To create a user profile:

Note:  Both supervisors and operators can create new user profiles, but only someone with supervisor permissions can create a new user profile that has supervisor permissions.

1. On any CONFIGURATION screen, tap the PROFILE tab.
2. Tap NEW PROFILE.
3. Enter the username for the new profile.

4. Select an access mode:
— The SUPERVISOR mode lets the user create, delete, and manage his own and all other user profiles.
— The OPERATOR mode lets a user add, change, or delete a password for his profile.

5. Tap SAVE NEW PROFILE. The user profile that you created now displays in the list of profiles.

Deleting a User Profile

Profile deletion restrictions are:

● Only a user with supervisor permissions can delete a user profile.
● The default user profile cannot be deleted.
● The currently active profile cannot be deleted.

To delete a profile:

1. Ensure that the user profile for your control station includes supervisor permissions.
2. Ensure that the active user profile is not the profile that you wish to delete
3. In the list of users on the left side of the screen, select the user profile that you wish to delete.
4. Tap DELETE PROFILE.
