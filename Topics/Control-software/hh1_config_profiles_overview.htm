About Configuring User Profiles

See Also for links to information about setting up and using a user profile.

Purpose: Lets you create and manage user profiles for supervisors and operators.
Access: On any CONFIGURATION screen, tap the PROFILE tab.

Each ROV operator can create an individual user profile that contains settings for various aspects of ROV operation (like joystick button assignments or tether movement device assignments). An operator can use configuration options to :

● Create or delete a user profile
● Manage password protection for a user profile
● Adjust settings for tactile controls and access level (supervisor or operator access)
