Managing User Profile Password Protection

See Also for links to information about setting up and using a user profile.

If you wish, you can set up password protection for a user profile.

Profile password restrictions are:

● A user with supervisor permissions can change anyone’s password. A user with operator permissions can change only his own password.
● The default user profile cannot be protected with a password.

Adding or Changing a Password

To add password protection or change a saved password:

1. On any CONFIGURATION screen, tap the PROFILE tab.
2. Select the desired user profile in the left-hand list.
3. Tap CHANGE PASSWORD.
4. Drag the arrow to the unlock icon.
5. Enter the desired password twice.
6. Tap SAVE.

Deleting a Password

1. On any CONFIGURATION screen, tap the PROFILE tab.
2. Select the desired user profile in the left-hand list.
3. Tap REMOVE PASSWORD
4. Drag the arrow to the unlock icon.
