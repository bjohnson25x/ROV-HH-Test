Logging Off or Switching to Another User Profile

See Also for links to information about setting up and using a user profile.

Switching To Another User Profile

1. Tap the user profile button on the Status Bar.
2. Tap SWITCH USER.
3. Select the desired user profile.

Logging Off Without Switching to Another User Profile

1. Tap the user profile button on the Status Bar.
2. Tap LOG OFF USER. The default user profile is automatically logged in.
