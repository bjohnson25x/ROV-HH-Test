Configuring the Deadband Level on the Flight Joystick

See Also for links to information about configuring flight joystick controls.

See Also for links to general information about configuring tactile controls.

You can adjust the flight joystick deadband level (the mechanical range within which the control system ignores joystick input) when wear or contamination cause false movement commands while the joystick is not being moved (that is, when the joystick is at its mechanical neutral position).

When to Change the Deadband Level Setting

Before you adjust the deadband level, you can:

● Inspect the joystick for foreign objects, contamination, damage, or wear that could prevent the device from returning to or holding at the neutral position. Correct any problems that you find.
● To remedy intermittent or transient joystick offsets, you can set a new neutral (zero) position for the device (see Setting Control Device Neutral (Zero) Position).

If zeroing the joystick does not eliminate the problem, you can compensate for minor, consistent, persistent offsets by increasing the joystick deadband. This procedure increases the distance that you must move the joystick before the control system recognizes that movement as input from the joystick.

You can set separate deadbands for individual joystick movement directions (fore/aft, port/starboard, yaw, and vertical). Vertical movement is controlled by the joystick vertical thrust switch.

Setting the Deadband Level

Note:  This setting is retained in the user profile for the selected control station.

1. Display any CONFIGURATION screen.
2. Tap the TACTILE CONTROLS tab, and then tap the JOYSTICK CONTROL tab.
3. Use the top-right button to select the desired control station (LEFT CLIENT COMPUTER or RIGHT CLIENT COMPUTER).
4. In the Joystick Adjustments panel, tap the DEADBAND button for the desired joystick movement direction (fore/aft, port/starboard, yaw, or vertical).
5. Drag the slider to set the desired percentage increase or decrease for the deadband. The selected percentage displays on the button.
