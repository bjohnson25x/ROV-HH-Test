Assigning Video Event Marking to the Flight Joystick

See Also for links to information about configuring flight joystick controls.

See Also for links to more information about using event place markers in recorded video. This information applies only to ROV systems with HDEVS.

This information applies only to ROV systems that have HDEVS rather than TAVS.

On the flight joystick, you can assign a button to add event place markers for unplanned events in recorded video. (The same button can also be configured to turn selected auto functions on and off.) The button works only for video that is being currently recorded.

Note:  This setting is retained in the user profile for the selected control station.

To configure the button to trigger event markers for unplanned events:

1. Display any CONFIGURATION screen.
2. Tap the TACTILE CONTROLS tab, and then tap the JOYSTICK CONTROL tab.
3. Use the top-right button to select the desired control station (LEFT CLIENT COMPUTER or RIGHT CLIENT COMPUTER).
4. Use the BUTTON FUNCTION drop-down list to select the function that will trigger event markers for unplanned events.
