Scaling Low-Speed Thrust on the Flight Joystick

See Also for links to information about configuring flight joystick controls.

See Also for links to general information about configuring tactile controls.

For each joystick, you can use low-speed scaling to specify the percentage of full thrust that is available. You can specify different scaling for each joystick axis (forward/aft, port/starboard, yaw, and vertical), and that scaling is applied when you push the joystick to its maximum in that axis.

Note:   Low-speed scaling applies only when speed is set to LOW (that is, when the HIGH SPEED button, described in HIGH SPEED Auto Function, is not active).

Note:  The low-speed scaling setting is retained in the user profile for the selected control station.

To apply low-speed scaling:

1. Display any CONFIGURATION screen.
2. Tap the TACTILE CONTROLS tab, and then tap the JOYSTICK CONTROL tab.
3. Use the top-right button to select the desired control station (LEFT CLIENT COMPUTER or RIGHT CLIENT COMPUTER).
4. In the Joystick Adjustments panel, tap the LOW SPEED SCALING button for the desired joystick axis (fore/aft, port/starboard, yaw, or vertical).
5. Drag the slider to set the desired percentage of full thrust. For example, if you set FORE/AFT to 50%, pushing the joystick all the way forward will apply 50% of maximum thrust.
