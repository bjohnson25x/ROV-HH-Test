Purpose: Configures and assigns flight joystick functions. Use to:

● Assign pan and tilt controls to a flight joystick
● Set the direction of ROV flight for the vertical flight control switch
● Scale ROV low-speed joystick controls
● Adjust the joystick deadband
● Zero the joystick (set the neutral position—the position at which the joystick is at rest)
● Assign on/off control for a selected auto function to a flight joystick button
Access: On any CONFIGURATION screen, tap the TACTILE CONTROLS tab and then tap the JOYSTICK CONTROL tab.

Caution!  Before you use any controls on this screen, inspect each joystick for foreign objects, contamination, damage, or wear that could prevent the joystick from returning to or holding at its mechanical neutral position. Strong vibration can also induce an offset to the neutral position. Correct any problems before you use the joystick configuration controls.
