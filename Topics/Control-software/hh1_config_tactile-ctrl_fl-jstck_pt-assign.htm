Assigning Pan and Tilt Controls on the Flight Joystick

The flight joystick contains a mini joystick button that lets you move a pan and tilt (see Flight Joystick Controls and Operations). A three-position selector thumb switch on the joystick lets you indicate which pan and tilt will be moved by the mini joystick.

Before you can use the three-position thumb switch, you must associate up to three pan and tilts with the flight joystick thumb switch. You can then use thumb switch positions (forward, middle, and back) to designate which pan and tilt will move when you operate the flight joystick mini joystick button.

Note:  This setting is retained in the user profile for the selected control station.

To configure which pan and tilts are selected by the three-position thumb switch:

1. Display any CONFIGURATION screen.
2. Tap the TACTILE CONTROLS tab, and then tap the JOYSTICK CONTROL tab.
3. Use the top-right button to select the desired control station (LEFT CLIENT COMPUTER or RIGHT CLIENT COMPUTER).
4. Use the CAMERA SWITCH drop-down lists to select a pan and tilt for each thumb switch position.

Note:  If a pan and tilt is being actively controlled by one flight joystick, any inputs for that pan and tilt from the second joystick are ignored (and discarded) until the first joystick controls are released. As soon as the first joystick stops issuing commands, the other joystick can command the pan and tilt.
