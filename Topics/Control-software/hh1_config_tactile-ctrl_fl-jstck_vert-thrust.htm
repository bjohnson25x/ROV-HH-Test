Configuring Vertical Thrust Control on the Flight Joystick

See Also for links to information about configuring flight joystick controls.

See Also for links to general information about configuring tactile controls.

The flight joystick vertical thrust control applies thrust up or down, but the control has different effects depending on how you have configured it.

Note:  This setting is retained in the user profile for the selected control station.

To configure the vertical thrust control:

1. Display any CONFIGURATION screen.
2. Tap the TACTILE CONTROLS tab, and then tap the JOYSTICK CONTROL tab.
3. Use the top-right button to select the desired control station (LEFT CLIENT COMPUTER or RIGHT CLIENT COMPUTER).
4. Using the drop-down list labeled PRESS UP TO MAKE ROV, select ASCEND or DESCEND.
