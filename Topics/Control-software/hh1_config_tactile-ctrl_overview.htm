About Configuring Tactile Controls

See Also for links to general information about configuring tactile controls.

See Also for links to information about configuring flight joystick controls.

See Also for links to configuration information about tactile tether controls.

Purpose: Configures and assigns controls on the flight joystick and tether control devices.
Access: On any CONFIGURATION screen, tap the TACTILE CONTROLS tab. Then tap either the JOYSTICK CONTROL tab or TETHER CONTROL tab.

Note:  Some tactile control settings are part of each user profile.

On the screen top right, tap RIGHT CLIENT COMPUTER to configure the tactile controls for the right control station, or LEFT CLIENT COMPUTER for the left control station.
