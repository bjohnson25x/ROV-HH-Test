Assigning Tether Control Devices

See Also for links to configuration information about tactile tether controls.

See Also for links to general information about configuring tactile controls.

You can designate whether tether movement is controlled by the flight joystick trigger or by the foot pedal.

Note:  This setting is retained in the user profile for the selected control station.

To assign a tether control device:

1. Display any CONFIGURATION screen.
2. Tap the TACTILE CONTROLS tab, and then tap the TETHER CONTROL tab.
3. Use the top-right button to select the desired control station (LEFT CLIENT COMPUTER or RIGHT CLIENT COMPUTER).
4. Put a check mark in the desired box.
