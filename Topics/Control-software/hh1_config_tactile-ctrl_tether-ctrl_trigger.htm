Adjusting the Joystick Trigger for Tether Control

See Also for links to configuration information about tactile tether controls.

See Also for links to general information about configuring tactile controls.

You can scale the speed of tether movement commanded by the joystick trigger by designating a speed percentage. This scaling applies to all speeds for tether movement. (Speeds are SLOW and FAST for the HD/UHD-III and SLOW, MEDIUM, and FAST for the UHD-II. These speeds are set in the FLIGHT screen TMS CONTROL panel as described in Joystick and Foot Pedal Control of Tether Movement.) The trigger speed adjustment is called a scaling adjustment.

The scaling adjustment applies to all TMS speeds. For example, if you set flight joystick trigger scaling to 55%:

● When the tether is set for SLOW speed, pressing the joystick trigger will move tether at 55% of the SLOW speed.
● When the tether is set for FAST speed, pressing the joystick trigger will move tether at 55% of the FAST speed.

Note:  The scaling adjustment is retained in the user profile for the selected control station.

To adjust the joystick trigger:

1. Display any CONFIGURATION screen.
2. Tap the TACTILE CONTROLS tab, and then tap the TETHER CONTROL tab.
3. Use the top-right button to select the desired control station (LEFT CLIENT COMPUTER or RIGHT CLIENT COMPUTER).
4. Tap SCALING.
5. Drag the slider to select the desired percentage.
