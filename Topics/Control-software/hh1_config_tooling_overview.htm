Configuring Tooling Packages (Including Rate-Controlled Manipulators)

Purpose of the tooling package screen: Lets you:

● View information about installed tooling packages (including manipulators)
● Add and remove tooling
● Configure tooling valve assignments.
Access: On any CONFIGURATION screen, tap the TOOLING tab.

For information on configuring a manipulator for AutoFollow, see Configuring a Manipulator for AutoFollow.

For information on installing or deleting a tooling package, see Adding and Deleting a Hydraulic Tooling Package.

Viewing/Editing Tooling Package Information

To view or edit tooling package information:

1. On any CONFIGURATION screen, tap the TOOLING tab.
2. On the list on the left side of the screen, tap the name of the desired tooling package.

You can then perform these functions:

● View displayed information about the package.
● Modify the label for the selected package.
● Assign valve pack valves for individual tool functions (except for position-controlled manipulators)
● Perform other functions specific to the tool

Assigning Valves to Tooling Package Functions

If the package has functions that are powered by a valve pack, you can use the screen display for the tooling package to indicate which valve is powering each function. The screen below shows valve assignments for a rate-controlled manipulator arm.

To assign a valve to a tool function:

1. On any CONFIGURATION screen, tap the TOOLING tab.
2. On the list of tools on the left side of the screen, tap the button for the desired tool. The screen then displays a list of functions for that tool.
3. Tap the drop-down arrow for the desired function, and the screen displays a list of available valve pack values. (Valves that are already assigned to other uses do not appear on the list.)

Note:  An S next to a valve function identifies a solenoid valve, and a P identifies a proportional valve.

4. If you want to change the gain (percentage of full flow) for a proportional valve, tap the text that shows the current gain.
5. If you want to reverse valve function, tap the REVERSE check box to add a check mark.

Reversing a function makes a valve's port A respond to a command that otherwise would actuate valve port B, and makes valve port B respond to a command that otherwise would actuate valve port A. You can use this function to compensate for incorrectly plumbed hydraulic lines. For example, if incorrect plumbing causes a manipulator upper arm to go down when the rate hand controller issues a command to go up, you can correct the problem by reversing the valve function.
