Configuring the Switcher for TAVS Video

See Also for links to information about video functions and devices for TAVS.

Purpose: Informs the control system about hardware connections to input and output ports on the video switcher, allowing accurate routing for video inputs and outputs.

Access:  From any CONFIGURATION screen, tap the VIDEO SWITCH tab.

This information applies only to ROV systems that have TAVS rather than HDEVS.

Note:  In descriptions of TAVS signal routing, video signals are described as inputs or outputs from the viewpoint of the video switcher. For example, the output from a camera is described as an input because the signals are input to the video switcher.

The video switcher contains input ports and output ports. These ports are connected with cables to video inputs (from devices like topside cameras) and outputs (to devices like monitors, annotators, and DVD recorders). Once cables have been connected to the video switcher, you must inform the control system about the hardware connections so that the control system can accurately route video. You do this by configuring the video switcher. By configuring the VIDEO SWITCH screen, you are providing information to the control system about the devices are physically connected to each video switcher input port and output port. For each port, you can access a drop-down list of items that are available to associate with that port.

Cameras can be only video inputs to the switcher, and monitors can be only video outputs. Other devices, such as annotators and recorders, can simultaneously be both video inputs and outputs. See the ROV system interconnect drawing (in Drawings and Part Lists) to determine how video devices are connected for this ROV system.

You can also use the EDIT TOPSIDE VIDEO DEVICES button to add and remove information about video devices.

Note:  This function only labels ports on the video switcher; it does not route video inputs or outputs. To route video inputs or outputs, use the FLIGHT screen VIDEO panel (see Flight Screen Video Panel). If you do not accurately identify the hardware connections for the video switcher, video will not be correctly routed.

If you have a 16-by-16 video switcher, you can see all 16 input ports and 16 output ports on the screen. If you have a 32-by-32 video switcher, you must scroll to see all input and output ports.

Specifying a Video Switcher Port Connection

For each port on the video switcher, indicate which input or output (if any) is connected to that port.

See how
1. Display any CONFIGURATION page.
2. Tap the VIDEO SWITCH tab.
3. Tap the port’s drop-down and select the item connected to that port.

Adding or Deleting a Video Switcher Port Connection

If you add or delete the hardware connection for a video switcher port, you must inform the control system about the change.

See how

Adding a Video Switcher Port Connection

1. Display any CONFIGURATION screen.
2. Tap the VIDEO SWITCH tab.
3. Tap EDIT TOPSIDE VIDEO DEVICES.
4. Tap ADD VIDEO DEVICE.
5. Tap the button for device types to display the drop-down list.
6. Tap the desired device type.
7. Enter a label for the device.
8. Tap SAVE.

Deleting a Video Switcher Port Connection

1. Display the CONFIGURATION screen.
2. Tap the VIDEO SWITCH tab.
3. Tap EDIT TOPSIDE VIDEO DEVICES.
4. Tap the tab for the desired device category.
5. From the drop-down list, tap the button for the desired device.
6. Tap REMOVE VIDEO DEVICE.
