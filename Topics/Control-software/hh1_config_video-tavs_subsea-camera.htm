TAVS Subsea Camera Installation and Configuration

See Also for links to information about video functions and devices for TAVS.

This information applies only to ROV systems that have TAVS rather than HDEVS.

General steps for installing and configuring a subsea camera are:

1. Connect the camera’s cable to a subsea node port.
2. You will be tunneling the camera’s signals from the subsea node port to a rack node VIM port. Decide which rack node VIM port to use. If that rack node port is not already connected to a video switcher input port, connect a cable between the rack node port and the desired video switcher input port.
3. Using the control system software, install the camera in the control system by specifying the camera’s subsea node port and the rack node port (described in step 2) that will be used to tunnel the video signals.
4. Configure the device in the control system (for example, specify serial communication parameters for zoom and focus signals).
5. On the VIDEO SWITCH screen, confirm that the video input port is configured as the desired node port.

Detailed information about these steps is provided in sections that follow.

Note:  In descriptions of TAVS signal routing, video signals are described as inputs or outputs from the viewpoint of the video switcher. For example, the output from a camera is described as an input because the signals are input to the video switcher.

Connecting a Camera to a Subsea Node

This installation procedure applies to any composite video camera (NTSC or PAL, fixed or with zoom and focus features).

See how

Before you begin the installation, note the camera’s specifications, which are available from either the manufacturer’s manual or the camera’s serialization plate. You will need to know:

● Camera manufacturer and model
● Video format (NTSC or PAL)
● Features (like zoom and focus)

To connect a camera to a subsea node:

1. Attach the camera connector to a video interface module (VIM) on a subsea node.
2. Record the subsea node number and port number for the connected camera. You will need this information when you add the camera to the control system.

Connecting Subsea Camera Signals to the Video Switcher

You will later be tunneling the camera’s signals from the subsea node port to a rack node VIM port. Decide which rack node VIM port to use.

See how

Typically, a rack node has several extra VIM ports waiting to be attached to added subsea cameras. Each extra rack node VIM port is typically already connected with a cable to a video switcher input port.

If the rack node VIM port that you want to use is already connected to a video switcher input port, you do not need to attach additional cables. If not, you must connect the rack node VIM port to a video switcher input port.

Adding/Configuring a Subsea Camera in the Control System

You must add the camera to the control system and then configure the camera’s properties.

See how
1. Display any CONFIGURATION screen.
2. Tap the DEVICE tab at the top of the screen.
3. Tap INSTALL DEVICE.
4. When the INSTALL DEVICE panel displays, tap the VIDEO tab.

5. In the list of preconfigured cameras, tap the button for the camera that you are adding. (If the camera is not on the list of preconfigured devices, see the manual section on installing a custom device.)
6. Tap NEXT STEP.
7. Select the subsea node port to which you attached the camera. The interface module for the port must be a VIM.

8. Tap NEXT STEP, and the screen changes to let you select a client VIM (a video tunnel from the subsea node to a rack node) and a video format.

9. Tap the drop-down list for ADD VIDEO TUNNELS and select the desired rack node VIM from the list of available ports. Ports that have already been assigned to other functions are not displayed in this list.

10. Tap the drop-down box for SELECT VIDEO FORMAT and select a format.
11. Tap NEXT STEP.
12. When the screen asks you to confirm the update to the control system configuration (the addition of the camera), tap INSTALL.
13. Follow instructions in the technical manual to configure the camera (camera properties, serial properties, and power properties).

Configuring the Video Switcher for a Subsea Camera

Once the camera is installed and configured in the control system, you must configure the video switcher.

See how

You can access the configuration for the video switcher from any CONFIGURATION screen by tapping the VIDEO SWITCH tab near the screen top.

If a cable was already connected between the camera’s rack node VIM port and a video switcher input port, the video switcher has probably already been configured to associate the rack node port and the video switcher input port. Check the VIDEO SWITCH screen to confirm that the added subsea camera shows on the screen for the correct switcher input port.

If you had to connect a cable between the camera’s rack node port and a video switcher input port, you will probably need to configure the video switcher to associate the rack node port and the video switcher input port. On the VIDEO SWITCH screen, tap the drop-down list for the video switcher input port that is connected to the rack node port, and select the correct subsea camera.
