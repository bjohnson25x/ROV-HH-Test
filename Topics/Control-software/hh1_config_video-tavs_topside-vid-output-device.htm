TAVS Topside Video Output Device Installation and Configuration

See Also for links to information about video functions and devices for TAVS.

This information applies only to ROV systems that have TAVS rather than HDEVS.

Note:  In descriptions of TAVS signal routing, video signals are described as inputs or outputs from the viewpoint of the video switcher. For example, the output from a camera is described as an input because the signals are input to the video switcher.

Topside video output devices can include:

● Monitors (on the monitor wall or elsewhere, such as on the bridge)
● Annotators (if the annotator is receiving signals from another device)
● Recorders (if the recorder is receiving signals from another device)

To install and configure a topside video output device:

1. Connect the output device’s cable to the desired video switcher output port.
2. From any CONFIGURATION screen, tap the VIDEO SWITCH tab.
3. Tap EDIT TOPSIDE VIDEO DEVICES in the screen upper left portion.
4. When the VIDEO DEVICE CONFIGURATION panel displays, tap ADD VIDEO DEVICE.

5. When the ADD VIDEO DEVICE panel displays, select the desired option:
— If you are adding a monitor, select DISPLAY.
— If you are adding an annotator, select ANNOTATOR.
— If you are adding a recorder, select VIDEO RECORDER.
6. Enter a label for the device.
7. Tap SAVE.
8. In the VIDEO DEVICE CONFIGURATION panel, tap CLOSE.
9. On the VIDEO SWITCH screen, tap the drop-down list for the video switcher output port that is connected to the device, and select the device.
