Dashboard Display

See Also for links to information about control console monitor displays, including the dashboard.

The ROV system dashboard typically displays on the bottom center screen of the monitor wall.

Items on the dashboard vary depending on ROV system installed equipment.

This illustration shows a screen for an HD or UHD-III; the screen for a UHD-II would have slight differences.

If a device supplying input to the dashboard is turned on but is not providing data (for example, if the device supplying heading information is turned on but is malfunctioning), the dashboard items that use that data are displayed in orange. If a device that should supply input to the dashboard is not turned on, the dashboard items that use that data are grayed out.

Dashboard Hydraulic Data

The dashboard hydraulic data section shows information for hydraulic circuits, valve packs, and compensators. Information varies depending on whether the ROV system uses the UHD-III hydraulic configuration or the HD/UHD-II hydraulic configuration. The figure below shows an HD hydraulic configuration.

 

A compensator's volume displays on the dashboard only if the compensator has been configured for dash board display (see Compensator Configuration Properties).
