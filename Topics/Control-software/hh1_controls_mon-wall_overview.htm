About the Control Console Monitors

See Also for links to information about control console monitor displays, including the dashboard.

The arrangement of information on control console monitors depends on whether the ROV system uses TAVS or HDEVS to process and display video.

The control console monitor wall consists of either 6, 9, or 12 LCD screens that can display:

● A dashboard showing critical ROV and TMS information.
● Video from ROV system cameras.
● Navigation (sonar) information.
● Data from a KVM-switched source.
— For ROV systems using TAVS, KVM-switched sources include the RTC, remote diagnostics, and graphical user interfaces for the annotator and some DVD recorders.
— For ROV systems using HDEVS, KVM-switched sources include the RTC, remote diagnostics, and the graphical user interface for the Video Computer App.

To specify routing for video streams from their sources to screens on the monitor wall, use the FLIGHT screen VIDEO panel.

The dashboard display is typically wired directly to the bottom center monitor; in this case, the dashboard cannot be moved to another monitor using control system software. The sonar display is typically wired directly to the monitor above the dashboard; if this is the case, the sonar display cannot be moved using control system software. For other monitors, you use the FLIGHT screen VIDEO panel to assign video streams.

Some monitors on the monitor wall are hard-wired for specific inputs. In a nine-monitor wall:

● Quad-tiled monitors on the top left and top right, (typically named 1-TILER and 3-TILER).
● Sonar appears in the middle center (typically named 5-SONAR).
● System dashboard data appears on the in the bottom center (typically named 8-DASHBOARD).

Ensure that your naming convention for monitors allows these hard-wired monitors to remain in the appropriate places in the FLIGHT screen VIDEO panel.

For information about how monitors are displayed in the VIDEO panel, see VIDEO Panel Display for TAVS.

Monitors for ROV System Using HDEVS

Monitor walls for ROV systems with HDEVS can have nine or twelve monitors.

In the illustrations below, “D” indicates that a decoder (see Hardware and Data Routing for HDEVS Video) is attached to the monitor. Any monitor attached to a decoder can show a single video stream or can be quad tiled to show four separate video streams. The displayed video can be real-time or recorded, and real-time video can be viewed with or without an overlay.
