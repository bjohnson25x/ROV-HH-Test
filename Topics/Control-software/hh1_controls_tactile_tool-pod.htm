Tool Control Pod

See Also for links to information about using specific tactile controls.

The tool control pod lets you operate a rate-controlled slave arm (or other rate-controlled equipment), pan and tilts, lights, cameras, and valve packs.

For tool pod controls, typically:

● Knobs are used to control proportional valves.
● Single-function switches are used to control a single solenoid (on/off) valve.
● Two-function switches are used to control two solenoid valves (one valve is controlled with up/down switch positions and the other valve with left/right positions).

For information on assigning valves to controls on the tool control pod, see Assigning Valve Operation to the Tool Pod.

The Status Bar shows the current pan and tilt assignments (see Control Assignments—Status Bar). You can select and operate a pan and tilt even if it is also assigned to other control devices.
