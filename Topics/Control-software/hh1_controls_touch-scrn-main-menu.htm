Touch Screen Main Menu

See Also for links to information about touch screen controls.

Purpose: Lets you access touch screens for major ROV system functions.

Access:  Left side of the Status Bar (which displays at the top of all screens).

The Main Menu lets you access screens for major ROV system functions.

LAUNCH AND RECOVERY Button

Guides you through launching and recovering the ROV system.

FLIGHT Button

Accesses the FLIGHT screen for:

● ROV and TMS operation
● Survey control
● Light control
● Pan and tilt control
● Camera control
● HPU control
● Compensator/reservoir monitoring
● Power control for HPU and the instrument power equipment (IPDS or IPA)
● Video control
● Device power control

TOOLING Button

● Operation of tooling packages (including manipulators)
● Operation of valve packs

DIAGNOSTICS Button

Provides access to:

● Diagnostics information for subsystems (ROV, TMS, topside, and telemetry)
● Diagnostics information for individual devices
● Calibration controls for individual devices
● Diagnostics information for nodes and node ports
● Access to logs that list issues and events

CONFIGURATION Button

Provides access to functions for:

● Adding, deleting, and configuring devices
● Adding, deleting, and configuring tools (including manipulators)
● Creating and restoring configuration backups
● Configuring tactile controls
● Outputting ROV system data and inputting/outputting survey data
● Changing system parameters
● Creating and editing user profiles
