Check box Tap box to add or remove check mark.

“Trying” or loading If an action takes more than a fraction of a second to complete, the button shows the “trying” state.

Safety-Protected Buttons

Some buttons are safety-protected so that you cannot accidentally tap them. Such buttons typically show a lock icon. To perform a safety-protected function:

1. Tap the function.
2. Drag the arrow to the unlock icon.

The button then changes to the ON state.
