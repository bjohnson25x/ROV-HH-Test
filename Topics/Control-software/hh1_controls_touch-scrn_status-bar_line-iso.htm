Line Isolation Displays for the HD or UHD-III

ROV Line Isolation

This illustration shows an example of the line isolation display for the ROV platform. In the display, color indicates line isolation status:

● Green = good level of resistance
● Orange = questionable level of resistance
● Red = possible fault

If the ROV is not equipped for line isolation monitoring on both AC circuits, the display will show line isolation for only AC1 or AC2.

TMS Line Isolation

This illustration shows an example of the line isolation display for the TMS platform.

The TMS has a single 240-VAC circuit instead of two 120-VAC circuits.

If the TMS is not equipped for line isolation monitoring, the display will not show information for the 240-VAC circuit.

If the TMS IPDS contains an HV Pak that converts 240-VAC power to 340-VDC power for TMS DC drive motors, a value will not display on the 240-VAC line isolation bar. Color coding on the bar for line isolation status still applies.

Line Isolation Measurements

See details

For the 26-VDC (node) circuit

● HIGH SIDE value is leakage resistance from the +26-VDC bus to the chassis (vehicle frame).
● LOW SIDE value is leakage resistance from the 26-VDC return bus to the chassis (vehicle frame).

For more information about line isolation for the nodes, see Node Diagnostics.

For the ROV 48-VDC circuit

● HIGH SIDE value is leakage resistance from the +48-VDC bus to the chassis (vehicle frame).
● LOW SIDE value is leakage resistance from the 48-VDC return bus to the chassis (vehicle frame).

For the ROV 120-VAC and TMS 240-VAC circuits

The value shows any IPDS winding leakage to the chassis (vehicle frame).

Locations for Line Isolation Monitoring

The location for line isolation monitoring depends on the circuit type and the type of installed equipment.

See details

DC Circuit Monitoring

Line isolation monitoring for the 26-VDC circuit takes place in the first node in the node chain for each platform (for example, the first node in the ROV and the first node in the TMS).

Line isolation monitoring for the 48-VDC circuit takes place in the IPDS 48-volt Multi Pak.

AC Circuits if LCU 101-9518 Is Installed or No LCU Is Installed

The IPDS performs line isolation monitoring for AC1 and AC2.

AC Circuits if LCU 101-6016 Is Installed

If LCU 101-6016 is installed and is attached to an AC circuit, then it performs line isolation monitoring for the AC circuit that it is attached to. The IPDS performs line isolation monitoring for the other AC circuit.

● Example 1: LCU 101-6016 is attached to the AC1 circuit. The LCU performs line isolation monitoring for AC1. The IPDS performs line isolation monitoring for AC2.
● Example 2: LCU 101-6016 is attached to the AC2 circuit. The LCU performs line isolation monitoring for AC2. The IPDS performs line isolation monitoring for AC1.

If LCU 101-6016 is not attached to any AC circuit, then the IPDS performs line isolation monitoring for both AC circuits.

For information about configuring LCU 101-6016, see Lighting Configuration Properties (Including LCUs).

Line Isolation Measurements

See details

For the ROV and TMS 26-VDC (node) circuits

● HIGH SIDE value is leakage resistance from the +26-VDC bus to the chassis (vehicle frame).
● LOW SIDE value is leakage resistance from the 26-VDC return bus to the chassis (vehicle frame).

For more information about line isolation for the nodes, see Node Diagnostics.

For the ROV and TMS 48-VDC circuits

The value is leakage resistance from the +48-VDC bus to the chassis (vehicle frame).

For the ROV and TMS 120-VAC circuits

The value shows IPA winding leakage to the chassis (vehicle frame).

Locations for Line Isolation Monitoring

The location for line isolation monitoring depends on the circuit type and the type of installed equipment.

See details

DC Circuit Monitoring

Line isolation monitoring for the 26-VDC circuit takes place in the first node in the node chain for each platform (for example, the first node in the ROV and the first node in the TMS).

Line isolation monitoring for the 48-VDC circuit takes place in the IPA.

AC Circuit if LCU 101-9518 Is Installed or No LCU Is Installed

No line isolation monitoring is performed for the 120-VAC circuit.

AC Circuit if LCU 101-6016 Is Installed

If LCU 101-6016 is installed, it performs line isolation monitoring for the 120-VAC circuit (but only if you configure the LCU properties properly in the ROV control system).

For information about configuring LCU 101-6016, see Lighting Configuration Properties (Including LCUs).
