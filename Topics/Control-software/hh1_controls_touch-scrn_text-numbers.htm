Entering Text and Numbers on the Touch Screen

See Also for links to information about touch screen controls.

You may need to enter text and/or numbers on the screen to perform functions like labeling a button on the screen. For example:

● Labeling a light button with the light location or function
● Labeling a valve for a tooling operation
● Labeling an added device like a camera or navigation instrument

When you perform a function that requires a text entry, a keyboard panel displays on the screen.

Also, some functions require that you enter numbers on a screen numeric keypad panel.
