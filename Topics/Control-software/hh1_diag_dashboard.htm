Diagnostics Dashboard—Overview of Platform and Subsystem Status

See Also for links to general diagnostics information.

Diagnostics dashboard purpose: Provides an overview of ROV system status
Access: From the Main Menu, tap DIAGNOSTICS and then tap SYSTEM. If the diagnostics dashboard does not display, tap the SEE SYSTEM SCHEMATIC button just below the Main Menu button.

Overview of Platform Status

The diagnostics dashboard is divided into sections for the system platforms: ROV, TMS, topside equipment, and telemetry system. For each platform, a status panel with indicator lights shows current issues for each subsystem.

The indicator lights are color coded:

● Green - No current issues
● Yellow - Information (a relevant occurrence has taken place)
● Orange - Warning (a situation likely needs attention)
● Red - Alarm (a situation needs urgent attention)

The indicator light displayed next to a platform or subsystem shows the color of the most serious issue for that platform or subsystem. For example, if the ROV hydraulics subsystem has a red issue, an orange issue, and a yellow issue, the subsystem indicator light will be red.

Note:  You cannot acknowledge an issue from the diagnostics dashboard; instead, you must use the Status Bar issues list (see Viewing System Issues—Status Bar).

Overview of Subsystem Status

Each platform contains subsystems; for example, the ROV platform contains subsystems for compensation, hydraulics, power, etc. The panel for each platform contains buttons that let you access status information for individual platform subsystems.

Note:  The number and type of subsystems varies between ROV systems.
