Component Details Diagnostics Panel

See Also for links to general diagnostics information.

The COMPONENT DETAILS panel of a device’s diagnostics screen shows status information that is specific to that device type. For example, COMPONENT DETAILS information for a pan actuator is different from information for a compensator.

For some devices, the COMPONENT DETAILS panels let you perform functions. For example, for compensators that can be calibrated, the COMPONENT DETAILS panel contains a CALIBRATE button.
