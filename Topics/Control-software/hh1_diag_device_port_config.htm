The PORT CONFIGURATION panel displays on the diagnostics screen for any device that attaches to a node port. The panel contains different information and/or buttons depending on whether the port's interface module is a SIM, VIM, SDVIM, or NIM (see Device Signal Types).

Resending Interface Module Port Setup

If an interface module stops responding properly, tap RESEND IM SETUP to resend to that module the configuration information that was present at system startup. This will not solve hardware problems but could replace corrupt interface module information.

Video Test Patterns

This function is available for a camera attached to a VIM. Tap TEST PATTERN to change the output of the selected VIM from video to color bars. This can help you to determine whether poor video is caused by the camera or by the subsea VIM.

This function is not needed (and is not available for) a camera attached to an SDVIM.
