Viewing and Exporting Device Traffic Diagnostics

See Also for links to more diagnostics information about node ports.

See Also for links to general diagnostics information.

On the diagnostics screens for many devices, the VIEW DEVICE TRAFFIC button displays real-time communications going to and from the device through node ports.

 

You can also save selected device traffic information in a file on an external memory device to use in diagnosing a device communication problem. The saved file will contain up to 100 communications (a mix of REQUEST TO DEVICE and RESPONSE FROM DEVICE communications) along with selected device and system information.

To save the data:

1. Insert an external memory device (like a USB flash drive) into the client computer for your control station.
2. Display the diagnostics page for the desired device (see Diagnostics Information for a Single Device).
3. Tap VIEW DEVICE TRAFFIC.
4. When the RAW DIAGNOSTIC DATA panel displays, tap PAUSE.
5. Tap SAVE TO EXTERNAL DEVICE.
6. When the SELECT FILE SAVE LOCATION panel displays, tap the desired location (folder) on the external memory device.

7. If you want to change the file name for the saved file, tap the file name and enter a different name.
8. Tap SAVE.
