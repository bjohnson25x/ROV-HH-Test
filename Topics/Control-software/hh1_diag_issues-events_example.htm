Example of Issue/Event Sequence

See Also for links to information about issues and events.

If the HPU motor temperature rises above the normal range during vigorous operation, the following sequence of actions might occur:

Example Sequence for Issue Resolution
HPU motor temperature rises above normal.

An issue is generated; it displays in the Active Issues log and as a new issue in the Status Bar issues list.

An event is posted in the event log indicating that an HPU motor temperature issue has been generated.
Operator sees the new issue and acknowledges the issue. When you acknowledge the issue, it is no longer a new issue. An event is posted to the Event log indicating that the issue has been acknowledged. The issue remains an active issue as long as the motor temperature is above normal.
Thruster use drops, and HPU motor temperature drops back into the normal range. An event is posted in the Event log indicating that HPU temperature is within normal range. Since the issue was previously acknowledged, the issue is automatically cleared from the Active Issues log and from the issues lists on the Status bar and the DIAGNOSTICS dashboard.
