Issue Clearing (Returning to Normal Conditions)

See Also for links to information about issues and events.

When an issue is cleared (that is, the condition that caused the issue to be raised later goes back to the normal operating condition or range):

● An event is added to the Event Log to indicate that the equipment returned to the normal operating condition or range.
● The issue disappears from the Active Issues log.
● On the Status Bar issues list and the DIAGNOSTICS dashboard issues lists:
— If the issue has been acknowledged, it disappears from the list.
— If the issue has not been acknowledged, it remains on the list with the message “This issue has cleared.”
