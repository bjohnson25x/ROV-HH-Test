Telling the Control System to Ignore Issues

See Also for links to information about issues and events.

Tapping an issue turns the issue dark gray and displays the DIAGNOSTICS and IGNORE ALARM buttons.

Tapping IGNORE ALARM removes the issue and all issues in the same category from the list and prevents this issue from being reported by the control system.

Caution!  Ignoring an alarm for a single issue causes the control system to ignore all alarms in the same category. For example, if you choose IGNORE ALARM for a low compensator alert, the control system will ignore alarms for all compensators that are low on fluid.

Caution!  Because IGNORE ALARM prevents future notifications about this issue, use this feature carefully.

If you select IGNORE ALARM for an issue, the RESTORE IGNORED ALARMS button displays below the issue list with the count of ignored issues.
