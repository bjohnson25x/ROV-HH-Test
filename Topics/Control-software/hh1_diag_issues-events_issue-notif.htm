Issue Notification Methods

See Also for links to information about issues and events.

The control system informs you that an issue has been raised by:

● Displaying the issue in a list available from the Status Bar (see Viewing System Issues—Status Bar).
● Displaying the issue in a list of issues for each platform, available from the DIAGNOSTICS dashboard (see Diagnostics Dashboard—Overview of Platform and Subsystem Status).
● Sounding an alert. This happens for only some issues, and you can configure level of issue severity that generates an audible alert (see Setting Alert Sounds).
● Displaying the issue in the Active Issues log (see Viewing Logs of Active Issues).
