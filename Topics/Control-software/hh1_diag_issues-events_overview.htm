During operation, the control system notes significant happenings and lets you view lists of those happenings. The happenings are classified as issues or events.

Significant System Happenings
Type Generated By Examples Where Viewed
Issue A piece of equipment is outside of its normal operational condition or range.
● The main HPU circuit oil temperature has risen above normal.
● A node has stopped communicating with the control system
● Status bar (for entire ROV system)
● DIAGNOSTICS dashboard (individually for the ROV, TMS, topside, or telemetry system
● Active Issues log
● Events log
Event

Something of interest has happened in ROV system operation.

Note:  All issues are also considered to be events, but not all events are issues.
● The AUTO HEADING setting has changed.
● The ROV has docked to the TMS.
● An issue has been raised for HPU motor over temperature.
● HPU motor temperature has returned to normal.
● Events log
