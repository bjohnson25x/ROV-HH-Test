Status Bar System Issues List

See Also for links to information about issues and events.

The ISSUES tab on the Status Bar displays the number of current issues for the entire ROV system, and the number of new (that is, unacknowledged) issues.

Note:  The DIAGNOSTICS dashboard displays similar issue information, but for individual system platforms (the ROV, TMS, topside equipment, and the telemetry system).

A color-coded indicator on the tab shows the severity of the most severe issue in the list:

● Green - No current issues
● Yellow - Information (a relevant occurrence has taken place)
● Orange - Warning (a situation likely needs attention)
● Red - Alarm (a situation needs urgent attention)

Tap the ISSUES tab to display the current issue list. New (unacknowledged) issues appear at the top of the list, followed by acknowledged issues.
