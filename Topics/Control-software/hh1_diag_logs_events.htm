Viewing Logs of Event History

See Also for links to information about logs for events and active issues.

The EVENT HISTORY list shows events that have been noted by the ROV control system. Since all issues are events, issues appear in this list along with events that are not issues.

Events are color-coded based on their importance to ROV system operation:

● Yellow - Information (a relevant occurrence has taken place)
● Orange - Warning (a situation likely needs attention)
● Red - Alarm (a situation needs urgent attention)

When an issue arises (because a system component or situation is outside of normal operating conditions), that is recorded as an event. When the issue is cleared (because the component or situation returns to normal parameters), that is also recorded as an event.

When the event log has recorded approximately 1000 events, that log is archived and a new log is started. You can select the desired log to view by using the NOW button.

When an event log is more than 14 days old, it is automatically deleted.
