Displaying the Diagnostics Main Menu

See Also for links to general diagnostics information.

The control system typically informs you about problems or potential problems by notifying you about an issue (see About Issues and Events). Once you become aware of an issue, you can use diagnostics information to find the problem source.

From the Main Menu, you can tap DIAGNOSTICS to access the following types of diagnostics information:

● System: Status information about parts of the ROV system.
● User devices: Port information for devices with user tunnels and mirrored interfaces.
● Logs: Historical data about ROV system functions, including lists of events and active issues.
