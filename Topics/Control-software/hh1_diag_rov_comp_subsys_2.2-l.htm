2.2-Liter Compensator Diagnostics

See Also for links to diagnostics information for specific compensators and reservoirs.

Note:  To view fluid levels for all compensators and reservoirs or to view or set high-volume and low-volume alarm levels, use the FLIGHT screen COMPS panel.

You can view additional status information on the device's diagnostics screen. To view the diagnostics screen, first display the list of installed compensation devices and then select the desired compensator.

See how

From the Main Menu:

1. Tap DIAGNOSTICS.
2. Tap SYSTEM.
3. When the diagnostics dashboard displays, tap the button for the desired subsystem (like HYDRAULICS, COMPENSATION, or LIGHTS) on the desired platform (like ROV or TMS).

Or from any DIAGNOSTICS screen:

1. Tap the platform drop-down button on the upper left, and then tap the desired platform (like ROV or TMS).
2. Tap a button on the left to select the desired subsystem (like HYDRAULICS, COMPENSATION, or LIGHTS).

Tap the button for an individual 2.2-liter compensator to see status information and controls for that device.

Compensation Calibration

Note:  The ROV control system uses “compensation counts” to describe how full a compensator is. These counts are not in any particular unit (like milliliters); they are internally defined units used by the control system to calibrate compensators.

Calibrating the 2.2-liter compensator ensures that the control system's compensator counts match the actual fluid volume. You must calibrate each compensator separately.

Long press any compensator icon or graph to access the calibration controls. You can also access calibration controls from the reservoir’s diagnostics page.

To calibrate a 2.2-liter compensator:

1. Empty all fluid from the selected compensator.
2. On the FLIGHT screen, tap COMP.
3. Expand the COMPENSATOR AND RESERVOIR LEVELS panel.
4. Long press the desired compensator icon.
5. Tap CALIBRATE, and the COMPENSATOR/RESERVOIR CALIBRATION panel appears.
6. Tap SET LEVEL and then select EMPTY. Note the number of counts on the screen; the number should be 2800 or more.
7. Fill the compensator.
8. Tap SET LEVEL and then select FULL. Note the number of counts on the screen; the number should be 4100 or less.
9. Tap DONE.

As noted above, while you perform the EMPTY and FULL calibration, confirm that the numbers in the calibration counts display are within the acceptable range (2800 or more at empty and 4100 or less at full). If either the EMPTY or FULL number of counts is outside the acceptable range, check and adjust the compensator sensor hardware (described in the Compensator section of Power, Telemetry, & Compensation Systems).

Note:  The calibration process can be done either from EMPTY to FULL or from FULL to EMPTY. The EMPTY and FULL count values cannot be the same.
