This manual section describes diagnostic displays for ROV compensators. Similar displays are available for TMS compensators.

Note:  To view fluid levels for all compensators and reservoirs or to view or set high-volume and low-volume alarm levels, use the FLIGHT screen COMPS panel.

To view status and diagnostics information for the compensation subsystem, navigate to the subsystem screen display.

See how

From the Main Menu:

1. Tap DIAGNOSTICS.
2. Tap SYSTEM.
3. When the diagnostics dashboard displays, tap the button for the desired subsystem (like HYDRAULICS, COMPENSATION, or LIGHTS) on the desired platform (like ROV or TMS).

Or from any DIAGNOSTICS screen:

1. Tap the platform drop-down button on the upper left, and then tap the desired platform (like ROV or TMS).
2. Tap a button on the left to select the desired subsystem (like HYDRAULICS, COMPENSATION, or LIGHTS).

Tap COMPENSATION to display summary information for all compensators on the selected platform (ROV or TMS).

Note:  For ROV systems that use 8-liter compensators, summary information does not appear for 8-liter compensators and reservoirs because they communicate with the control system through the ROV's position sensor array (PSA). To view information for the 8-liter compensators and reservoirs, display the diagnostics screen for the ROV's PSA (see 8-Liter Compensator/Reservoir Diagnostics).

For the compensator status indicator:

● Green = normal
● Red = loss of communications
● Gray = compensator not powered on
● Orange = level has dropped below low-level alarm set point
● Yellow = level is above high-level set point
