Thruster Control Unit Details

See Also for links to diagnostics information for specific devices in the ROV flight subsystem.

Tap the button for a TCU to display details about the thrusters that it is controlling.

Watchdog Timeout

The watchdog timeout status indicator for a thruster shows the status of communication between the ROV control system and that thruster's control module on the printed circuit assembly of the thruster control unit (TCU). If the watchdog timeout status indicators for one or more thrusters are red, this indicates that the thruster(s) have been disabled because communication was interrupted for more than a half second.

If one or more watchdog timeout status indicators are red, long press the ROV icon (see below) on the flight device screen. In the menu that displays, select ENABLE ALL THRUSTERS. If the watchdog timeout status indicator(s) are still red, cycle power to the TCU.

CPU Reset

The CPU reset status indicator for a thruster shows the status of the CPU's power supply. If the CPU reset status indicators for one or more thrusters are red, this indicates that the voltage to the CPU has dropped below a threshold and the CPU has reset itself.

If one or more CPU reset status indicators are red, long press the ROV image (see above) on the flight device screen. In the menu that displays, select ENABLE ALL THRUSTERS. If the CPU reset status indicators are still red, check the power connections.
