Direct (Manual) Thruster Control

See Also for links to diagnostics information for specific devices in the ROV flight subsystem.

From the ROV flight subsystem diagnostics display, you can select individual thrusters for manual control.

Caution!  Use direct control mode only on deck and only for diagnostics purposes. Do not try to fly the ROV using direct control mode.

Note:  If the speed set point does not approximately match the thruster speed reported by the speed sensor, disable the thruster until you can determine the cause.

Note:  If a thruster’s rotation is reversed from the expected direction, its installation or connections may be incorrect.

Use direct control mode only on deck and only for diagnostics purposes. Do not try to fly the ROV using direct control mode. To operate thrusters under direct control:

1. Activate DIRECT CONTROL MODE. This stops all thrusters and automatically cancels all auto functions.
2. Tap the down arrow next to a thruster that you want to operate directly. Drag the slider to set the maximum speed and direction.
3. Set speeds for any additional thrusters that you want to operate directly.
4. To turn thrusters on, either:
a. Tap ALL ON. This simultaneously turns on all thrusters, but the thrusters that do not have a commanded speed will not run.
b. Individually tap each thruster that you want to run.
5. To stop thruster operation, you can:
a. Stop all thrusters by tapping ALL OFF. All thrusters stop, but the commanded set points are unchanged.
b. Stop a single thruster by tapping its button. The thruster’s commanded set point is unchanged.
c. Set the speed for all thrusters to zero by tapping ZERO ALL. Thrusters that were on remain on, but with a commanded set point of zero.

To return thrusters to automatic control, tap DIRECT CONTROL MODE. (This stops all thrusters.) You must then enable ROV flight with the ROV FLIGHT button.
