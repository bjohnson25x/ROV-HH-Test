Using Open-Loop Control

See Also for links to diagnostics information for specific devices in the ROV flight subsystem.

In the default (closed-loop) control mode, the speed sensed by thruster sensors is compared to commanded speed, and the control system adjusts valves to make the sensed thruster speed match the commanded speed.

If a thruster speed sensor fails while in the normal closed-loop control mode, the thrusterâ€™s speed automatically increases to maximum. If this happens, switch to open-loop control. This lets you control thruster operation without using thruster sensors, but results in slightly degraded performance.

From the ROV flight subsystem diagnostics display, you can select individual thrusters for open-loop control.

1. In the THRUSTER CONTROL panel, long press the thruster that you want to run in open-loop control.
2. On the panel, tap CONTROL.
3. Tap the desired control mode (OPEN LOOP or CLOSED LOOP).
