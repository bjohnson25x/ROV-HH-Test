Schilling 3-ST Valve Pack Diagnostics

See Also for links to diagnostics information for the ROV hydraulic subsystem.

From the diagnostics screen for the Schilling 3-ST valve pack, you can:

● View circuit information.
● Calibrate all circuits.

Viewing Circuit Pressure and Flow

Note:  The control system uses “feedback counts” (shown on the screen as FB COUNTS) to describe pressure and flow levels. These counts are not in any particular unit (like psi or milliliters); they are internally defined units used by the control system to calibrate hydraulic equipment.

In the list of circuits, "USER PRESSURE" refers to pressure measured manually at the AUX PRESSURE SNSR port with a user-supplied line and gauge.

Calibrating the 3-ST Valve Pack

Pressure and flow settings for 3-station valve pack hydraulic circuits are calibrated at the factory, but changes to circuit components or component wear can make the factory settings inaccurate. If pressure or flow response seems to be inaccurate, calibrate the circuit.

Before beginning the calibration, perform the steps described in the service instruction for 3-ST calibration setup.

Caution!  Heat generated by extended on-deck operation of ROV or TMS equipment can damage the equipment and related components. During on-deck operations, do not override temperature safety limits set in the ROV control system software. Overriding safety limits can result in component damage or failure. When equipment is running in air, do not touch the equipment to test temperature.

Note:  The control system uses “feedback counts” (shown on the screen as FP COUNTS) to describe pressure and flow levels. These counts are not in any particular unit (such as psi or milliliters); they are internally defined units used by the control system to calibrate hydraulic equipment.

You must calibrate each circuit in the Schilling 3-ST valve pack separately.

1. On the DIAGNOSTICS screen for the Schilling 3-ST valve pack, tap CALIBRATE.
2. When the VALVE PACK CALIBRATION panel appears, select the desired circuit and function from the drop-down list. You must separately calibrate pressure and flow for each circuit.
3. Tap START CALIBRATION to begin the process.
a. If you are calibrating circuit pressure, the calibration process will cycle automatically through each pressure set point.
b. If you are calibrating flow, you must tap the screen to select each flow point.
4. When you are prompted at each set point, tap the READING field, and using the pop-up keypad, enter the current value from the analog gauge installed on the valve pack.
5. After you have entered readings for each set point, tap the SAVE CALIBRATION button. You cannot save the calibration unless you have entered all pressure or flow readings.
