Calibrating an HPU (HD and UHD-II Systems Only)

See Also for links to diagnostics information for the ROV hydraulic subsystem.

Not all HPUs require calibration:

● If the ROV has a UHD-III-style hydraulic configuration, no calibration is needed.
● If the ROV HPU has an HD/UHD-II-style hydraulic configuration, you should calibrate the HPU as needed.
● If the TMS has an HPU, you should calibrate the HPU as needed.

If one or more HPUs for this ROV system require calibration, the pressure settings for all system hydraulic circuits are initially calibrated at the factory. However, changes to circuit components or component wear can make the factory settings inaccurate. If you note a significant difference between commanded HPU pressure and the pressure displayed on a circuit pressure gauge, calibrate the circuit.

The HPU pump and motor heat up quickly when run on deck in air.

Caution!  Heat generated by extended on-deck operation of ROV or TMS equipment can damage the equipment and related components. During on-deck operations, do not override temperature safety limits set in the ROV control system software. Overriding safety limits can result in component damage or failure. When equipment is running in air, do not touch the equipment to test temperature.

The ROV control system uses “feedback counts” (shown on the screen as FB COUNTS) to describe pressure levels. These counts are not in any particular unit (like psi); they are internally defined units used by the control system to calibrate HPUs.

You must separately calibrate each HPU hydraulic circuit. To perform a calibration:

1. Display the CALIBRATE button for the appropriate hydraulic circuit. You can either:
— On the FLIGHT screen, tap the HPU button to display the HYDRAULIC PRESSURE CONTROL panel. Tap the button for the desired hydraulic circuit, and then long press anywhere inside the panel.

or

— Display the appropriate diagnostics screen. For an ROV HPU, display the diagnostic screen for the ROV hydraulic subsystem (see About Diagnostics for a Subsystem or Device). For a TMS HPU, display the diagnostic screen for the TMS hydraulic subsystem.
2. Tap CALIBRATE to display the HPU CALIBRATION panel.

3. Tap START CALIBRATION to begin the process. The system will begin to cycle through each pressure set point. The number of set points (and their values) varies depending on which HPU circuit you are calibrating.
4. When you are prompted at each set point, tap the READING field. Using the panel keypad, enter the current value from the analog hydraulic pressure gauge installed on the hydraulic circuit. Be sure to use the correct gauge for the circuit (ROV main or aux, or for a hydraulic TMS, TMS main) that you are calibrating.
5. After you have entered readings for each set point, tap SAVE CALIBRATION. You cannot save the calibration unless you have entered all pressure readings.
