Manipulator Diagnostics

See Also for links to diagnostics information for the ROV hydraulic subsystem.

To see diagnostics information for a rate-controlled manipulator (like the RigMaster), go to the diagnostics display for the valve pack that supplies hydraulic power to the manipulator’s functions.

To see diagnostics information for a position-controlled manipulator (like the Titan 4 or Orion 7P), go to the diagnostics information for the ROV hydraulic subsystem. The display shows feedback information for manipulator function and error conditions, and also shows the current position of the manipulator arm. This information is useful if you cannot adequately view manipulator position with ROV cameras.
