About Diagnostics for a Subsystem or Device

See Also for links to general diagnostics information for platforms, subsystems, and devices.

To access information for a subsystem or a device, you can start from either the diagnostics dashboard or the left-hand navigation panel on any diagnostics screen.

Once you have selected the desired subsystem, you can view:

● Summary information for that subsystem (for example, the hydraulics subsystem, the lights subsystem, or the compensation subsystem). See Summary Diagnostics Information for a Subsystem.
● Information about each device in that subsystem (for example, an individual HPU reservoir, an individual light, or an individual compensator). See Diagnostics Information for a Single Device.

Note:  The number of subsystem-level and device-level screens varies depending on the type and number of installed devices.

Steps for Displaying the Diagnostics Screen for a Subsystem or Device

1. From the Main Menu, tap DIAGNOSTICS.
2. Tap SYSTEM. The screen then displays the most recent diagnostics screen that you have viewed. (This could be the diagnostics dashboard or the diagnostics screen for a specific subsystem or device.)
3. If the screen displays the diagnostics dashboard:

a. Tap the button for the desired subsystem (like COMPENSATION or LIGHTS) on the desired platform (like ROV or TMS). The diagnostics screen for that subsystem then displays.
b. If you then want to display diagnostics information for a particular device, tap the button for that device on the left-hand navigation panel.
4. If the screen displays the most recent diagnostics screen that you have viewed:
a. In the left-hand navigation panel, look at the platform button (directly below the VIEW SYSTEM SCHEMATIC button). If the platform button doesn’t show the name of the desired platform, use the drop-down list to select the platform.
b. In the left-hand navigation panel, tap the button for the desired subsystem.
c. If you then want to display diagnostics information for a particular device, tap the button for that device on the left-hand navigation panel.
