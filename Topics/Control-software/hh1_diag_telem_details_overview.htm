About Telemetry Diagnostics Details

See Also for links to information about telemetry diagnostics.

Purpose: The telemetry diagnostics screens let you view status data for nodes and their communication paths.

Access: On the DIAGNOSTICS dashboard section for the telemetry system, tap the button for the ROV, TMS, or topside platform.

Note:  The number and type of component-level screens vary depending on the number and type of installed devices.

To view status and diagnostics information for the telemetry subsystems, navigate to the desired subsystem screen display.

See how

From the Main Menu:

1. Tap DIAGNOSTICS.
2. Tap SYSTEM.
3. When the diagnostics dashboard displays, tap the button for the desired subsystem (like HYDRAULICS, COMPENSATION, or LIGHTS) on the desired platform (like ROV or TMS).

Or from any DIAGNOSTICS screen:

1. Tap the platform drop-down button on the upper left, and then tap the desired platform (like ROV or TMS).
2. Tap a button on the left to select the desired subsystem (like HYDRAULICS, COMPENSATION, or LIGHTS).

Tap a button to display summary information for the desired telemetry subsystem (ROV, TMS, or TOPSIDE).

Note:  Because telemetry screens for the ROV, TMS, and topside platforms have very similar layout and functions, this manual section shows only the ROV TELEMETRY screen.

To understand information about telemetry diagnostics, you must understand basic functioning of ports and their interface modules as described in Power, Telemetry, & Compensation Systems.
