Node Diagnostics

See Also for links to information about telemetry diagnostics.

To view diagnostics for a node:

1. From the TELEMETRY section of the DIAGNOSTICS screen, tap the platform (ROV, TMS, or topside) containing the desired node.
2. From the list on the left side of the screen, tap the button for the desired node.

Node 26-VDC Line Isolation

● HIGH SIDE value is leakage resistance from the +26-VDC bus to the chassis (vehicle frame).
● LOW SIDE value is leakage resistance from the 26-VDC return bus to the chassis (vehicle frame).

Status indicator colors mean:

● Green = good level of resistance
● Orange = questionable level of resistance
● Red = possible fault

Because of the way that line isolation is measured for +26-VDC circuits, high-side line isolation faults can appear to be low-side faults. However, low-side line isolation faults cannot appear as high-side faults.

Resetting a Node

Resetting a node turns the node off and on again, and reloads the node's configuration information. You can reset a node:

● From the DIAGNOSTICS screen for that node.
● From the DIAGNOSTICS TELEMETRY screen for the platform that contains a node (using a long press, as shown below).
