Client Computer Diagnostics

See Also for links to diagnostics information for other topside equipment.

From a client computer diagnostics screen (in the topside console subsystem), you can:

● View status and usage information
● Restart the computer
● Calibrate the touch screen

To view status and diagnostics information for an ROV system computer, display the diagnostics screen for that device (see Console Subsystem Diagnostics Display).

Calibrating a Touch Screen

Touch screens may require occasional calibration. If touching a screen does not produce the desired effect (for example, if tapping a button does not cause the button to function), you may need to calibrate the touch screen.

To calibrate a touch screen:

1. Go to DIAGNOSTICS > TOPSIDE > CONSOLE.
2. Tap the button for the desired client computer.
3. Tap CALIBRATE TOUCH SCREEN.
