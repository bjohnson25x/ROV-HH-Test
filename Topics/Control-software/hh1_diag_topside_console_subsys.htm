Console Subsystem Diagnostics Display

See Also for links to diagnostics information for other topside equipment.

To view status and diagnostics information for the console subsystem, navigate to the subsystem screen display.

See how

From the Main Menu:

1. Tap DIAGNOSTICS.
2. Tap SYSTEM.
3. When the diagnostics dashboard displays, tap the button for the desired subsystem (like HYDRAULICS, COMPENSATION, or LIGHTS) on the desired platform (like ROV or TMS).

Or from any DIAGNOSTICS screen:

1. Tap the platform drop-down button on the upper left, and then tap the desired platform (like ROV or TMS).
2. Tap a button on the left to select the desired subsystem (like HYDRAULICS, COMPENSATION, or LIGHTS).

Tap CONSOLE to display summary information for console devices.

Creating a Diagnostics Snapshot

The SNAPSHOT feature lets you create a file containing application data like device configurations, alarm notifications, and other system diagnostics information that can be used by factory technical support personnel to diagnose problems. A support technician will tell you when this file is needed.

Note:  Creating a snapshot is not the same as creating a configuration backup (described in About Backing Up and Restoring System Configurations). The configuration backup contains information about all ROV devices with a system interface (including tactile control devices), while a snapshot contains selected information that can be used by support technicians to remotely diagnose problems.

To use the snapshot function:

1. Decide which client screen (left or right) you will use to take the snapshot.
2. Insert a flash drive into a USB port in the client computer (left or right) connected to that screen.
3. Tap CREATE SNAPSHOT.
4. When the SELECT FILE SAVE LOCATION box displays, ensure that the flash drive is selected. If you wish to change the default file name (Snapshot.srs), tap the name and type a new one. The file name must have the extension .srs.

5. Tap SAVE.
6. Safely eject the flash drive and e-mail the .srs file to a technical support representative.
