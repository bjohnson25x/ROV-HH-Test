Diagnosing a Tactile Control Problem

See Also for links to general information about configuring tactile controls.

See Also for links to information about using specific tactile controls.

To troubleshoot a console tactile control, display the diagnostics screen for that device in the topside console subsystem (see Console Subsystem Diagnostics Display).

To diagnose a tactile control problem, activate a control (button, switch, encoder, or axis movement) on the device. If the device is operating properly, the corresponding area on the screen graphic lights up in white. If that area does not light up, device control is not working properly.

For devices that have a range of motion, a bar next to the device image shows feedback from the sensor. (For example, if you move the joystick forward, a bar beneath the joystick fore/aft image shows white fill in the bar. The farther that you move the joystick, the larger the white fill. If the feedback on the screen does not match the movement of the device, the device may not be operating properly.

Use ZERO POD CONTROLS to set a device's mechanical neutral position (the position where it will not actuate a function). For more information, see Setting Control Device Neutral (Zero) Position.
