Topside Power Subsystem Diagnostics Display

See Also for links to diagnostics information for other topside equipment.

To view status and diagnostics information for the topside power subsystem, navigate to the subsystem screen display.

See how

From the Main Menu:

1. Tap DIAGNOSTICS.
2. Tap SYSTEM.
3. When the diagnostics dashboard displays, tap the button for the desired subsystem (like HYDRAULICS, COMPENSATION, or LIGHTS) on the desired platform (like ROV or TMS).

Or from any DIAGNOSTICS screen:

1. Tap the platform drop-down button on the upper left, and then tap the desired platform (like ROV or TMS).
2. Tap a button on the left to select the desired subsystem (like HYDRAULICS, COMPENSATION, or LIGHTS).

Tap POWER to display summary information for topside power devices.

Status Displays for Topside Power

The PDU’s major electrical circuits are:

● ROV instrument power circuit for instrument power (IPDS or IPA)
● ROV HPU circuit
● TMS power circuit (for an HD/UHD-III, for instrument power and motors; for a UHD-II, for instrument power)
● TMS HPU circuit (UHD-II only)

Clean power, which requires its own generator, may not be used on this ROV system. If clean power is not used on this ROV system, no data for clean power is displayed.

See Power, Telemetry, & Compensation Systems for more information about the PDU and its operator interface (called the human-machine interface or HMI).

Some power problems generate alarms and others cause a “soft shutdown.” In a soft shutdown, the control system turns off power to the affected electrical circuit. Power can be restored only by fixing the problem and then pressing RESET FAULT on the PDU HMI.

To access additional information about a power-related alarm or soft shutdown:

1. View the ISSUES log.
2. Select the desired issue and view the details, including the state of the PDU at the time of the alarm.

Lost Communications with PDU

If the control system loses communications with the PDU:

● All PDU communications indicators turn red.
● All PDU circuit power buttons are disabled.
● The control system automatically limits ROV HPU system pressure (see Automatic HPU Motor Shutdown, Power Management, and Pressure Limiting).
