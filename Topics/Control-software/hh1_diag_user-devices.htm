User Device Diagnostics Details

Purpose: For the ROV and TMS, lets you view details about user tunnels and mirror ports.

Access:  In the Main Menu, tap DIAGNOSTICS and then tap USER DEVICES.

Note:  A device displays in the list of user devices only if you have established a user tunnel between the device's subsea node port and a rack node port.

User Tunnels and Mirror Ports

The user devices screen displays information about devices currently configured to use a user tunnel, a mirror port, or both. The arrows displayed with each device indicate the information flow. An arrow pointing away from the subsea port indicate that information is being sent to the topside port. An arrow pointing from a topside port to the subsea port indicates that the topside port is controlling the subsea port.

Tap DETAILS to view information related to all components involved with the user tunnel or mirror port. You can cycle power for individual components from this screen.
