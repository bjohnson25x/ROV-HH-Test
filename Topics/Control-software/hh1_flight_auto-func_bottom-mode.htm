BOTTOM MODE Features
Purpose To minimize stirring up sediment, vertical thrusters are not used to compensate for the effects of pitch or roll during flight.
Access On the FLIGHT screen, tap OTHER and then tap BOTTOM MODE.
Turning on/off

Tap BOTTOM MODE.
Function interactions

If AUTO DEPTH or AUTO ALTITUDE is being used, vertical thrusters run as necessary to maintain those set points.

When the ROV is flying with a pitch or roll attitude, the control system typically uses vertical thrusters during flight to keep the ROV from dipping up and down. If the ROV is operating near the seafloor, this use of vertical thrusters is undesirable because it stirs up sediment. BOTTOM MODE minimizes this use of vertical thrusters.

To turn BOTTOM MODE on or off, tap BOTTOM MODE.

BOTTOM MODE ON

When BOTTOM MODE is ON, the control system does not use the vertical thrusters to compensate for ROV motion due to attitude. Any lateral motion directed by the joystick follows the body coordinates (the current trim) of the vehicle. For example, if the vehicle front is pitched down 10 degrees, forward joystick movement causes the vehicle to move forward and down.

Even if BOTTOM MODE is on, vertical thrusters are still used if the operator commands vertical movement. Also, if AUTO DEPTH or AUTO ALTITUDE is being used, vertical thrusters run as necessary to maintain those set points. BOTTOM MODE prevents vertical thrusters from being used only to compensate for ROV motion due to attitude.

BOTTOM MODE OFF

When BOTTOM MODE is OFF, the control system runs the thrusters as necessary to produce true horizontal and vertical vehicle movement (following inertial coordinates), as commanded by the joystick and regardless of vehicle trim.

For example, if the vehicle is pitched down 10 degrees at the front, forward joystick movement will require use of vertical thrusters to produce true horizontal movement. Likewise, vertical joystick movement may require lateral and vertical thruster operation to achieve true vertical movement.
