Caution!  To return from CRUISE to flight joystick control, first move the joystick to approximate the current direction and speed of ROV flight. Then tap the joystick CRUISE on/off button. Do not move the joystick in the opposite direction from the current CRUISE settings because the thrusters will rapidly reverse direction.

CRUISE Features
Purpose

Allows trims to be set up in all three axes (X, Y, and Z). Can be set from the CRUISE panel or the joystick.
Access On the FLIGHT screen, tap CRUISE.
Data input Either a compass plus motion reference unit (MRU) or an MRU with heading capability.
Turning on/off
● Tap the appropriate panel button (FORE or AFT, STBD or PORT, UP or DOWN).
● Flight joystick button.
Initial set point The heading and speed at the time that you turn on CRUISE.
Changing the set point

Screen values are percentages of full thrust applied to produce motion in fore/aft, starboard/port, and up/down directions.

● Tap FORE/AFT, STBD/PORT, or UP/DOWN.
● Use the joystick to select a new direction and speed, and then release the flight joystick to its neutral position. The direction and speed at the time when you release the joystick becomes the new set point.
Function interactions
● If STATIONKEEP is on, CRUISE FORE/AFT and STBD/PORT buttons are not available.
● If AUTO ALTITUDE or AUTO DEPTH is on:
— CRUISE UP/DOWN buttons are not available.
— Any joystick up/down input is applied to the active AUTO ALTITUDE or AUTO DEPTH mode.
● Joystick yaw control is enabled in CRUISE mode unless the AUTO HEADING mode is on and locked.
● CRUISE mode works in both the normal and reverse flight modes, but you cannot switch between normal and reverse while CRUISE mode is active.
