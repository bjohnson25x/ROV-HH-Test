Auto Depth Features
Purpose Keeps the ROV at a designated depth from the surface.
Access On the FLIGHT screen, tap AUTO DEPTH.
Data input Depth sensor and gyro (TOGS MRU plus a compass or an MRU with heading).
Turning on/off
● Screen AUTO DEPTH button.
● Flight control pod button.
Potential issues

DEPTH mode works only at depths of 2.5 meters or greater.
Initial set point The depth at the time when you turn on AUTO DEPTH.
Changing the set point
● Tap UP or DOWN.
● If the LOCK CONTROLS button is not active, you can use the joystick to fly the ROV to a new depth. Then release the flight joystick to its neutral position. The ROV’s altitude at the time when you release the joystick becomes the new set point.
Function interactions

The DEPTH and ALTITUDE functions cannot be active at the same time.
