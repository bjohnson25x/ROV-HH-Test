DISPLACE Features
Purpose

In X-Y-Z mode, the ROV is moved for a specified distance fore/aft, starboard/port, and up/down.

In DISTANCE/ANGLE mode, the ROV is moved for a specified distance on a bearing relative to ROV forward.

Note:  In both modes, the applied inputs result in a single linear motion (a vector).
Access On the FLIGHT screen, tap DISPLACE.
Data input
● For fore/aft and starboard/port displacement (also, the StationKeep auto function must be operating):
— Compass plus motion reference unit (MRU) or MRU with heading capability.
— DVL.
● For up/down displacement:
— Compass MRU or MRU with heading capability.
— Depth sensor or altimeter.
Turning on/off In the DISPLACE panel, tap FORE/AFT, STBD/PORT, or UP/DOWN.
Entering displacement values X-Y-Z
1. Tap the appropriate panel buttons (FORE/AFT, STBD/PORT, and/or UP/DOWN).
2. Tap the number for the current value to allow keypad entry.

Button availability depends on which auto functions are on.
Distance/Angle
1. Tap the appropriate panel buttons for ANGLE (+/-), DISTANCE (+/-), and/or UP/DOWN.
2. Tap the number for the current value to allow keypad entry.

Button availability depends on which auto functions are on.
Commanding displacement

Tap START to begin displacement.

Tap STOP to interrupt displacement before it finishes.
Function interactions
● STATIONKEEP must be on for FORE/AFT and/or STBD/PORT buttons to be available.
● AUTO DEPTH or AUTO ALTITUDE must be on for UP/DOWN buttons to be available.
