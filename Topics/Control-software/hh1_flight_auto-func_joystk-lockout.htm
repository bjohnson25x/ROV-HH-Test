Locking Out Joystick Input from Auto Functions

See Also for links to general information about auto functions.

See Also for links to information about individual auto functions.

Some auto functions have a LOCK CONTROLS button that lets you lock out joystick input for that function. This prevents accidental joystick input from interfering with the auto function.

Examples:

● If the AUTO HEADING function is locked, you cannot use the joystick to change ROV heading, but you can change pitch and yaw.
● If StationKeep is locked, you cannot use the joystick to change the fore/aft and starboard/port position, but you can move the ROV up and down.

When an auto function is locked, you can still use screen controls in the auto function panel to change the function.

Examples:

● If the AUTO HEADING function is locked, you can change the ROV’s direction of movement with the STARBOARD and PORT keys.
● If StationKeep is locked, you can use the FORE, AFT, STARBOARD, and PORT keys to change ROV position.

To lock an auto function, open the auto function panel and tap LOCK CONTROLS.
