Item Description Manual Location
AUTO ALTITUDE Keeps the ROV at a designated distance from the seafloor.

AUTO ALTITUDE Auto Function

AUTO DEPTH Keeps the ROV at a designated depth from the surface. AUTO DEPTH Auto Function
AUTO HEADING Keeps the ROV pointed at a fixed compass heading. AUTO HEADING Auto Function
AUTO PITCH/ROLL Keeps ROV orientation at a designated pitch and/or roll. AUTO PITCH/ROLL Auto Function
BOTTOM MODE To minimize stirring up sediment, vertical thrusters are not used to compensate for the effects of pitch or roll during flight. BOTTOM MODE Auto Function
CRUISE Trim function that locks in the currently applied joystick commands, allowing the flight to continue with the joystick released. CRUISE Auto Function
DISPLACE Moves the ROV a specified distance and/or direction from the ROVâ€™s current position. DISPLACE Auto Function
HIGH SPEED Lets the ROV operate at the maximum allowed speed in response to joystick movement. HIGH SPEED Auto Function
MANUAL VELOCITY Allows dynamic braking, which minimizes overshoot due to ROV momentum. MANUAL VELOCITY Auto Function
MEASURE Measures the linear distance that the ROV travels as it moves over the seafloor or changes depth. MEASURE Auto Function
PARK An AUTO HEADING mode that uses only the aft horizontal thrusters. Intended for use when the ROV has been parked by using a manipulator to grab onto a structure. PARK Auto Function
REVERSE Reverses thruster response to flight joystick movement and CRUISE set points for fore/aft and starboard/port movement. Used when flying the ROV backward. REVERSE Auto Function
STATIONKEEP

Holds the ROV stationary above a designated point on the seafloor.

Mid-Water StationKeep (optional feature) performs this function when the ROV is out of DVL range of the seafloor.

 STATIONKEEP Auto Function
TRIMS Applies constant thrust in one or more designated directions (fore/aft, starboard/port, vertical, pitch, and roll). TRIMS Auto Function
