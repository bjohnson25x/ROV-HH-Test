MEASURE Features
Purpose Measures the straight-line distance from a specified ROV location to a new location. Does not calculate total distance traveled (including zigzag and up-and-down movement, as shown in figure below).
Access On the FLIGHT screen, tap MEASURE.
Data input DVL for horizontal displacement; depth sensor or DVL/altimeter for vertical displacement.
Turning on/off Tap MEASURE button.
Initial set point The current ROV location when you reset MEASURE.
Measuring procedure
1. Move the ROV to the starting location for the measurement.
2. Tap MEASURE.
3. Tap HORIZONTAL RESET and VERTICAL RESET. The values above the buttons should both be zero.
4. Move the ROV to the desired new location. The HORIZONTAL and VERTICAL values show the point-to-point distances traveled horizontally and vertically. The TOTAL value shows the total point-to-point distance (a combination of the vertical and horizontal distance).
Function interactions MEASURE distance measurements are not stopped, started, or influenced by other ROV controls. Unless you tap the RESET buttons, the MEASURE function accumulates data until (1) the altitude sensor (typically a DVL) is out of range, or (2) you turn off the MEASURE function.
