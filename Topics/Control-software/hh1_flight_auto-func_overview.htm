Purpose: Auto functions:

● Perform certain ROV flight operations automatically.
● Assist operators in flying the ROV.

Access:  On the FLIGHT screen, tap the button for the desired auto function.

Sensors on the ROV (like a depth sensor or a DVL) provide information that is used by the ROV control system to make the auto functions work.

Auto Function Compatibility

Many auto control functions are compatible with each other. (For example, the HEADING, ALTITUDE, and STATIONKEEP functions can be on at the same time.) Incompatibilities are usually obvious (for example, if STATIONKEEP is on, you cannot use the CRUISE function's FORE/AFT OR STBD/PORT buttons).

Functions that are incompatible with a currently active auto function are shown on the screen as unavailable or inactive.

Auto Function Long Press Submenu

When you long press an auto function button, a submenu displays to let you select options for the auto function and for the sensor that provides input to the auto function.

See how

The SELECT SOURCE button displays only if the auto function is off.

If an auto function is unavailable, long pressing the function's button explains why.

Turning an Auto Function Sensor On and Off

1. On the FLIGHT screen, long press the auto function’s button.
2. When the submenu displays, tap the button for the function’s sensor (for example, compass, MRU, depth sensor, or DVL).
3. Tap POWER ON or POWER OFF.

Assigning a Sensor to an Auto Function

If the ROV has more than one sensor that could provide input for an auto function (for example, two DVLs or both a gyro and a compass), you must specify which sensor should be used.

See how

You can select this sensor:

● From any CONFIGURATION screen.
● By long pressing the auto function button on the FLIGHT screen.

To change the auto function sensor from the FLIGHT screen:

1. If the auto function is on, tap the function’s button to turn it off.
2. Long press the function button.
3. Tap SELECT SOURCE.
4. Tap the button for the desired sensor.
