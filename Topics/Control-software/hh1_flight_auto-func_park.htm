PARK Features
Purpose An AUTO HEADING mode that uses only the aft horizontal thrusters. Intended for use when the ROV has been parked by using a manipulator to grab onto a structure.
Access On the FLIGHT screen, tap OTHER and then tap PARK.
Turning on/off Tap PARK.

In PARK mode, forward lateral thrusters are disabled to reduce turbulence near the work site.
