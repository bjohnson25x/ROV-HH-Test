Auto Function Sensor Dependencies

See Also for links to general information about auto functions.

See Also for links to information about individual auto functions.

Note:  Sensor quantities and types vary with each ROV system. Your system may not use all sensors listed below.

Most auto functions require one or more sensors to perform their flight-control functions. For information on how to assign a sensor for flight control, see Selecting Sensors to Use for Flight Control.

Auto Function Sensor Requirements
Mode Required Sensors

StationKeep
● Gyro: TOGS MRU plus a compass or an MRU with heading*
● DVL: WH1200

Cruise

None

Displacement XY
● Gyro: TOGS MRU plus a compass or an MRU with heading*
● DVL: WH1200

Displacement Z
● Gyro: TOGS MRU plus a compass or an MRU with heading*
● Depth sensor or altimeter

Trim

None

Auto Heading

Gyro: TOGS MRU plus a compass or an MRU with heading*

Auto Depth
● Gyro: TOGS MRU plus a compass or an MRU with heading*
● Depth sensor

Auto Altitude
● Gyro: TOGS MRU plus a compass or an MRU with heading*
● Altimeter or DVL (WH1200)

Mid-Water StationKeep
● Gyro: MiniPOS/NAV3 or Ixsea PHINS (must include a DVL)
● USBL

*MRUs with heading include CDL MiniPOS2, CDL MiniPOS/NAV3, Ixsea Octans, Ixsea PHINS, CDL INSense, and CDL TOGS/NAV
