Auto Function Set Points

See Also for links to information about individual auto functions.

The set point is a value commanded by the operator (for example, for a particular heading or depth). The ROV control system then works to reach and/or hold that commanded value.

Note:  Commanded set points for HEADING, ALTITUDE, DEPTH, CRUISE, and TRIMS controls may not represent the vehicle’s actual position, attitude, or speed if ocean currents or other forces are too strong. Also, if the ROV was moving when a set point was established, momentum may carry the ROV beyond the set point. It may then take several seconds for thrusters to return the ROV to the set point position.

Setting an Auto Function Set Point

For many auto functions, you can use a keypad to enter a value for the set point. (Tap the current set point to display the keypad.) However, some auto functions (including auto depth and auto altitude) require that you enter a set point by tapping buttons to increment or decrement the value.

Controlling Set Point Resolution

“Resolution” refers to the size of the increments in which you can change the set point value. For example, instead of adjusting altitude in meters, you can change the resolution to adjust altitude in tenths of meters.

To change the increments in which you can adjust an auto function set point:

1. On the FLIGHT screen, long press the auto function button.
2. When the submenu displays, tap EDIT SET POINT RESOLUTIONS.
3. The panel displays current increments for screen controls and tactile controls. Tap Screen or Tactile, and then tap the desired increment.
4. Tap SAVE.
