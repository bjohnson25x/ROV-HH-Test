TRIMS Features
Purpose

Applies constant thrust (as a percentage of total thrust) in one or more designated directions (fore/aft, starboard/port, vertical, pitch, and roll). Uses include:

● Compensating for currents.
● Moving the ROV along a continuous route at constant speed.
● Maintaining a constant ROV attitude.
Access On the FLIGHT screen, tap TRIMS.

Data input

 Compass plus motion reference unit (MRU) or MRU with heading capability.
Initial set point Must be entered by the operator.
Turning on/off Screen button in TRIMS panel: FORE/AFT, STBD/PORT, VERTICAL, ROLL, or PITCH.
Changing the set point FORE/AFT STARBOARD/PORT VERTICAL
● Tap the appropriate panel button (FORE or AFT, STBD or PORT, UP or DOWN).
● Tap the number for the current set point to allow keypad heading entry.
● Turn the appropriate flight control pod trim scroll wheel.
ROLL PITCH
● Tap the appropriate panel button (STBD or PORT, UP or DOWN).
● Tap the number for the current set point to allow keypad heading entry.
Initial set point Thrust being applied at the time when you turn on the TRIM function.
Changing the set point

Note:   TRIM settings are added to any flight joystick input.

1. Tap STBD or PORT.
2. Tap the number for the current heading to allow keypad heading entry.
3. Turn the flight control pod HEADING scroll wheel.
4. If the LOCK CONTROLS button from TRIMS is not active, you can use the joystick to yaw the ROV to a new heading, and then release the flight joystick to its neutral position. The ROV’s heading at the time when you release the joystick becomes the new set point.
Function interactions

If STATIONKEEP is on, TRIM FORE/AFT and STBD/PORT buttons are not available.

If AUTO ALTITUDE or AUTO DEPTH is on, TRIM UP/DOWN buttons are not available.

If AUTO ROLL is on, the TRIM ROLL button is not available. If AUTO PITCH is on, the TRIM PITCH button is not available.

If CRUISE is on, the TRIM ROLL and TRIM PITCH buttons are not available. If PARK is on, all TRIM functions are unavailable.

Turning thrusters off automatically turns off all TRIM functions.

Using Tool Pod Controls for the TRIMS Auto Functions

Assigning TRIMS to a Tactile Control

For the TRIMS auto function, you can assign an individual function (like fore/aft or roll) to a tactile control on the console tool pod.

1. Long press TRIMS.
2. Tap EDIT TACTILE ASSIGNMENTS.
3. Tap the desired trim function on the submenu.
4. When the TACTILE CONTROL ASSIGNMENT panel displays, tap the desired tactile control for that trim function. Tactile controls that can potentially be used for this function are shown as buttons with rounded edges. (Since trim functions require proportional control, knobs rather than switches should be used for this function.)
5. Tap ASSIGN TO.

Removing TRIMS from a Tactile Control

1. Long press TRIMS.
2. Tap EDIT TACTILE ASSIGNMENTS.
3. When the submenu displays, tap any of the TRIMS buttons on the submenu.
4. When the TACTILE CONTROL ASSIGNMENT panel displays, tap the control from which you want to remove the TRIMS function.
5. Tap CLEAR.
