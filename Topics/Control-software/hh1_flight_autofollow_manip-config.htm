Configuring a Manipulator for AutoFollow

See Also for links to AutoFollow (manipulator tracking) information.

Note:  AutoFollow can be used only on manipulators that are controlled with position feedback, like the TITAN 4 and CONAN 7P.

To set up a manipulator to use AutoFollow, you must add a position-controlled manipulator to the control system.

Note:  Only pan and tilt units installed on the ROV can be used with AutoFollow. Pan and tilts on other platforms (like the TMS or a skid) cannot use AutoFollow.

Note:  You cannot configure an already-installed manipulator to use AutoFollow. If you want to use AutoFollow for a manipulator that is already installed, you must first remove the manipulator from the control system software and then reinstall it.

To install a position-controlled manipulator:

1. Ensure that the user profile for your control station includes supervisor permissions.
2. From any CONFIGURATION page, tap the TOOLING tab.
3. Tap ADD TOOLING PACKAGE.
4. Select the position-controlled manipulator that you wish to use, and tap CONTINUE.
5. Select the desired ports for the manipulator.
