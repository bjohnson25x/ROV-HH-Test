AutoTrack Compatibility with Auto Functions

See Also for links to AutoTrack (survey operations) information.

Functions that you can use with AutoTrack

● AUTO DEPTH
● AUTO ALTITUDE
● AUTO ROLL
● AUTO PITCH
● TRIMS (vertical only)
● DISPLACE (vertical only)
● MEASURE
● LOW SPEED/HIGH SPEED flight joystick control speed selection

Functions that you cannot use with AutoTrack

● AUTO HEADING
● BOTTOM MODE
● PARK
● TRIMS (forward/aft and port/starboard only)
● DISPLACE (forward/aft and port/starboard only)
● Forward/aft control with the flight joystick (because the forward/aft joystick function controls AutoTrack speed)
● Lateral (port/starboard) control
● REVERSE/NORMAL direction for joystick
