AutoTrack Controls (Screen and Joystick)

See Also for links to AutoTrack (survey operations) information.

Touch Screen AUTOTRACK Panel

For access from the FLIGHT screen, tap the down arrow on the AUTOTRACK button.

Current data from the survey system for the heading and cross-track error (lateral offset) are displayed below the ROV icon.

Use the screen panel's MODE button to select:

● AutoTrack STATIONKEEP mode (preferred). In this mode, DVL data is used to control ROV position and forward speed.
● AutoTrack CRUISE mode. This mode keeps thrust at a constant rpm and does not influence ROV position. Use CRUISE mode only when the DVL cannot achieve bottom lock.

You must select either AutoTrack STATIONKEEP or CRUISE mode before you enable AutoTrack. To switch modes, you must first disable AutoTrack by tapping the AUTOTRACK button.

Flight Joystick Controls for AutoTrack

When the ROV is performing survey operations, the joystick button that normally turns the CRUISE auto function on and off is instead used to set ROV speed to zero (stop).

Large lateral or yaw joystick movement (more than 80% of full deflection) cancels AutoTrack and returns the ROV to standard control.
