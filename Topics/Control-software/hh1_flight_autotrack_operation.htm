AutoTrack Operation

See Also for links to AutoTrack (survey operations) information.

Typical Steps for Using AutoTrack

1. Ensure that all ROV system data inputs and outputs needed for the survey operation are properly configured (see ROV Motion and Position Output (Survey String) and Configuring Data Strings and Ports).
2. On the FLIGHT page, tap the AUTOTRACK down arrow to display the AUTOTRACK panel.
3. When the panel displays, tap the MODE drop-down list to select STATIONKEEP mode or CRUISE mode (see AutoTrack Controls (Screen and Joystick)).
4. Tap the AUTOTRACK button to enable AutoTrack operation. All ROV control system modes that are incompatible with AutoTrack become disabled, and the ROV automatically aligns itself to the heading of the survey line.
5. Start ROV movement forward along the survey line by either moving the flight joystick forward or entering a speed greater than zero in the AUTOTRACK panel.
6. If you want the ROV to deviate from the heading and cross-track error commands being provided by the survey system (for example, to correct for a current), use the HEADING OFFSET and TRACK OFFSET controls on the AUTOTRACK panel.
7. To change ROV speed forward along the survey line, use either the flight joystick or the SPEED controls on the AUTOTRACK panel.
8. To stop ROV movement forward along the survey line, tap STOP. AutoTrack remains enabled (so thrusters still respond to heading and cross-track error commands), but thrusters stop moving the ROV forward. If AutoTrack is operating in CRUISE mode, momentum may carry the ROV forward. If this is undesirable, you can counter the forward movement by moving the flight joystick back or by setting a negative speed in the AUTOTRACK panel SPEED controls.
9. To disable AutoTrack, tap the AUTOTRACK button.

While AutoTrack is operating, the survey system continuously updates the cross-track error (the deviation of the ROV from the survey line) and the heading command. The ROV automatically adjusts its position according to the latest information for the cross-track error and heading.

Controlling ROV Speed During AutoTrack

You can adjust the ROV speed setting using the flight joystick or the AUTOTRACK panel SPEED controls.

The survey system continuously updates the commanded ROV heading. The ROV may slow down along the survey line as it changes heading, but the ROV then accelerates to the desired speed.

If the AUTOTRACK panel speed is set to a negative value, the control system commands ROV thrusters to apply aft force. Because AutoTrack is not designed to move the ROV aft, use negative speed settings only to counter ROV momentum or a forward current.

Tapping STOP on the screen or on the joystick returns the speed setting to zero. Thruster operation remains enabled for all other compatible flight control modes, and AutoTrack remains enabled.

Note:  Although pressing STOP zeroes the joystick/screen speed setting, water current and/or momentum may carry the ROV forward. If this motion is undesirable, you can command an appropriate negative speed.

Disabling AutoTrack

When AutoTrack is disabled:

● Speed is set to zero (although thrusters remain enabled for use by any active flight control modes that are compatible with AutoTrack). Since momentum or currents can cause the ROV to continue moving, you can counter the movement with the flight joystick or enable STATIONKEEP to hold ROV position.
● Control modes that are incompatible with AutoTrack become available.

To manually disable AutoTrack, you can:

● Tap the touch screen AutoTrack button.
● Make a large lateral or yaw joystick movement (more than 80% of full deflection).

The ROV control system automatically disables AutoTrack and returns maneuvering control to the operator when:

● The control system has not received survey data for more than 3 seconds.
● DVL bottom lock is lost (applies only if AutoTrack is operating in STATIONKEEP mode).
● Heading sensor data is not received.
● The cross-track error is more than 5 meters (16 ft).
● One or more thrusters have malfunctioned, making maneuvering control inadequate.
