Accessing Survey String Output

See Also for links to AutoTrack (survey operations) information.

You can typically access survey string output through a serial expander port that has an RJ-11 connector. See ROV Motion and Position Output (Survey String) for information about survey string characteristics.

For information on selecting a survey string format and output port for the survey string, see Configuring Data Strings and Ports.

Pins on an RJ-11 port are:

● Pin 1, Receive
● Pin 2, Signal ground
● Pin 4, Transmit
