Changing Compensator/Reservoir Alarm Levels

See Also for links to information about compensator/reservoir monitoring.

Alarms have the following meanings:

● Low-level alarm = fluid volume has dropped below the low-level set point.
● High-level alarm = fluid volume has increased above the high-level set point.

To change the low-level and/or high-level alarm for a compensator or reservoir:

1. Display the expanded COMPENSATOR & RESERVOIR LEVELS panel.
2. Tap the arrow next to the percentage.
3. When the pop-up box displays, drag the sliders to adjust the levels.

Tap INDEX ALL to set all compensator low-level alarms to 20% below the current readings and upper-level alarms to 5% above the current volume readings. (INDEX ALL does not affect high-level alarms that are set to 100%.)
