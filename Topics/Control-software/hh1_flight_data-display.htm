ROV System FLIGHT Screen Flight Data Displays

See Also for links to general information about ROV flight.

The upper part of the FLIGHT screen displays:

● ROV position and flight data
● TMS position and tether data

The control system receives data for the display (for example, for ROV heading, depth, and altitude) from sensors that you designate to control flight (see Selecting Sensors to Use for Flight Control).

Note:   Gyros require time to align before they can begin transmitting data. If flight data from gyros does not immediately begin to display after startup, the gyro may not have completed alignment.

TMS Tether and Umbilical Display

Long press the tether/umbilical display to:

● Turn power to the ROV compass on or off.
● Turn power to the TMS compass on or off.
● Reset umbilical turns to zero.
● Reset tether turns to zero.

ROV Heading and Thrust Display

In the heading and thrust display, each thruster shows a vector (a cone) representing the percent of thruster rpm. Light-colored fill in the cone increases as thrust rpm increases.

The ROV motion vector (arrow) shows actual (not commanded) direction of travel. White fill in the arrow increases as speed increases.

Note:   The ROV motion vector displays only if the ROV has bottom lock.

Viewing Auto Function Set Points

The set point for an auto function displays as a green triangle in the flight data display.

● Solid green triangle indicates a locked set point (joystick input is not allowed to change the set point).
● Green outlined triangle is a set point that is not locked (joystick input is allowed to change the set point).

A set point may be different from an actual value. For example, currents may prevent the actual heading from being the same as the AUTO HEADING set point.
