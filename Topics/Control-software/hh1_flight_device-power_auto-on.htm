Changing Device Auto-On Status from the DEVICE POWER Panel

See Also for links to information about controlling power to individual devices.

Caution!  Lights and other devices that can overheat on deck should typically not be set to Auto-On.

If a node port is set to auto-on, a device attached to that port is automatically powered on when a platform receives power. To change a device's auto-on status, long press the device's label in the DEVICE POWER panel. A check mark next to the AUTO-ON button means that auto-on applies to that device. To remove or install a check mark, tap the button.

You can also change auto-on status for a device by editing the device's configuration (see POWER Configuration Tab: Setting Auto-On and Software Fuses).
