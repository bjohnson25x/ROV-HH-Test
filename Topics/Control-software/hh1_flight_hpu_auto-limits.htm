Automatic HPU Motor Shutdown, Power Management, and Pressure Limiting

See Also for links to information about operating and controlling hydraulic circuits.

Automatic HPU Motor Shutdown

The control system automatically turns off electrical power to the ROV HPU motor (and on the TMS HPU motor for hydraulic TMS units) if any of the following safety limits are exceeded:

● Motor temperature is greater than 80 °C (176 °F).
● Oil temperature is greater than 80 °C (176 °F).
● Oil volume is less than 0.44 liters (0.12 gal) for HD and UHD-III; less than 0.55 liters (0.15 gal) for UHD-II.

These limits are shown in the expanded HYDRAULIC PRESSURE CONTROL panel (see Viewing and Overriding HPU Shutdown Limits) and are not affected by user-set alarm thresholds.

Restart

An HPU that has been automatically turned off does not automatically restart. When each bar graph in the expanded HYDRAULIC PRESSURE CONTROL panel shows a value below its shutdown limit, you can start the HPU from the POWER MANAGEMENT AND CONTROL panel.

Automatic ROV HPU Power Management

Note:  Automatic power management directly affects the ROV main hydraulic circuit, but auxiliary (aux) hydraulic circuits may be indirectly affected.

The control system monitors motor line current for the main circuit ROV HPU electric motor. If line current exceeds the control system's safety threshold, the hydraulic control manifold lowers hydraulic circuit pressure (reducing motor current draw).

High current draw may indicate:

● Temporary need for excessive hydraulic power (for example, when many thrusters are running near their maximum speeds).
● Pump or motor malfunction.

The control system allows hydraulic circuit pressure to gradually increase to normal when motor current draw drops below the safety threshold.

When power management goes into effect, the message POWER MANAGEMENT appears in orange letters near the ROV icon on the FLIGHT screen. The dashboard display (see Dashboard Power Data) also contains a gauge showing power management information.

Automatic ROV HPU Pressure Limiting

Note:  Automatic HPU pressure limiting affects only the ROV main hydraulic circuit. Auxiliary (aux) hydraulic circuits are not affected.

Pressure Limiting Due To PLC Communications

A programmable logic controller (PLC) in the PDU provides motor current information that the control system uses to keep current draw at a safe level. If the PLC is not communicating with the control system (meaning that the control system cannot receive current draw information from the PLC), the control system limits the ROV HPU main circuit to a “safe” pressure:

● For the HD hydraulic configuration, 138 bars (2000 psi).
● For the UHD-II hydraulic configuration, 138 bars (2000 psi),
● For the UHD-III hydraulic configuration, 207 bars (3000 psi).

This prevents the ROV HPU from drawing too much current in response to commands for operation of thrusters and other hydraulic equipment.

If the PLC fails to communicate with the control system during startup, the control system initially sets ROV main circuit pressure at 34 bars (500 psi). You can then raise circuit pressure, but no higher than the safe pressure.

PLC communications can also be lost during ROV operations. If this happens when the ROV HPU main circuit pressure is higher than the safe pressure, the control system ramps down the HPU main circuit to the safe pressure and generates an alarm. Until communications with the PLC are restored, you can set main circuit pressure only to values between zero and the safe pressure.

For information on using the HYDRAULIC PRESSURE CONTROL panel to set circuit pressures, see Setting Hydraulic Circuit Pressures.

Pressure Limiting Due to Pump Load (UHD-III Only)

In the UHD-III main hydraulic circuit, high pump flow may prevent the ROV HPU from achieving the commanded pump pressure. In this situation, the ROV control system limits the commanded HPU main circuit pressure to no more than 21 bars (300 psi) above the actual pressure, and the message “Command Limit Active” appears on the diagnostics screen for the ROV hydraulic subsystem.
