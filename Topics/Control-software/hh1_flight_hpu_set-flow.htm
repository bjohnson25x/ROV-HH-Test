Setting Hydraulic Circuit Flow (UHD-III Only)

See Also for links to information about operating and controlling hydraulic circuits.

For ROV systems with the UHD-III hydraulic configuration, you can set hydraulic flow for an ISOL-8 pump aux hydraulic circuit only if that circuit is configured for flow control.

Maximum flow for an eight-cylinder ISOL-8 pump is:

● Aux circuit 1, 197 lpm (52 gpm).
● Aux circuit 2, 99 lpm (26 gpm).

Maximum flow for a four-cylinder ISOL-8 pump (aux circuit 1) is 132 lpm (35 gpm).

To change the hydraulic circuit flow:

1. Ensure that the ISOL-8 pump circuit is configured to allow flow control (see ISOL-8 Pump Device Configuration Properties).
2. From the FLIGHT screen, tap HPU.
3. When the HYDRAULIC PRESSURE CONTROL panel displays, tap the tab for the desired aux circuit.
4. Drag the slider in the FLOW gauge to change circuit flow.
