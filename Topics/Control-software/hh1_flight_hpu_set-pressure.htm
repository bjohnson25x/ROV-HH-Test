Setting Hydraulic Circuit Pressures

See Also for links to information about operating and controlling hydraulic circuits.

When the HPU is powered on, hydraulic circuits have default pressure settings. Following HPU startup, the operator can raise pressure for each circuit up to a maximum value.

You can adjust circuit pressure using controls in the HYDRAULIC PRESSURE CONTROL panel.

Default pressures apply at HPU startup even if other pressures were in effect when the HPU was shut down.

*Default maximum value is 211 bars (3000 psi), but you can increase to 345 bars (5000 psi) by overriding the limit.

Caution!  Operating an ISOL-8 pump hydraulic circuit at pressures above 211 bars (3000 psi) may significantly reduce the life of ISOL-8 pump cylinders.

If you are setting pressure for the main hydraulic circuit, the HYDRAULIC PRESSURE CONTROL panel shows only a pressure gauge. If you are setting pressure for an aux circuit that has been configured for flow control, the panel shows both a pressure gauge and a flow gauge.

Selecting Pressure Control and Pressure Sensors, UHD-III Hydraulic Configuration

The ROV control system can use two methods for HPU pressure control:

● Closed-loop control, where feedback from a pressure sensor on HPU pump A or pump B is used to make the actual pressure equal to the commanded pressure.
● Open-loop control, where sensor feedback is not used.

Typically, the most accurate pressure control results when you use closed-loop control and you let the control system chose which sensor (on pump A or pump B) to use. If one pressure sensor fails, you can configure the control system to use only the other sensor. If both pressure sensors fail, you can configure the control system to use open-loop control.

To configure the HPU pressure control:

1. Ensure that the user profile for your control station includes supervisor permissions.
2. Long press anywhere in the HYDRAULIC PRESSURE CONTROL panel to display the submenu below.

3. Tap PRESSURE CONTROL SOURCE.
a. If only one of the pressure sensors is operating correctly, tap the button to choose that sensor to provide feedback to the control system. This action locks out input from the failed sensor.
b. If both sensors have failed, select the OPEN LOOP option.

Changing Hydraulic Circuit Pressure After Startup

To change pressure for a hydraulic circuit:

1. On the FLIGHT screen, tap HPU.
2. Tap the tab for the desired hydraulic circuit to display the circuit’s HYDRAULIC PRESSURE CONTROL panel.
3. To change the pressure set point using the PRESSURE gauge:
a. Drag the slider to the desired pressure, or
b. Tap the up or down arrows, or
c. Tap one of the gauge's gray areas to go to the preset maximum or minimum pressure. You can configure these values using the GENERAL tab on any CONFIGURATION screen (see HPU Settings).

Note:  To enable an ISOL-8 pump circuit, HPU pressure must be at least 17 bars (250 psi).

Overriding Auxiliary Circuit Pressure Limits, UHD-III Only

Caution!  Operating an ISOL-8 pump hydraulic circuit at pressures above 211 bars (3000 psi) may significantly reduce the life of ISOL-8 pump cylinders.

Maximum allowed pressure for normal operation of the ISOL-8 pump auxiliary hydraulic circuits is 211 bars (3000 psi). However, you can override this limit to increase pressure up to 352 bars (5000 psi).

To override a pressure limit:

1. Ensure that the user profile for your control station includes supervisor permissions.
2. On the FLIGHT screen, tap HPU.
3. Tap the tab for the desired hydraulic circuit to display the circuit’s HYDRAULIC PRESSURE CONTROL panel, and expand the panel.
4. Tap MAXIMUM PRESSURE OVERRIDE.
