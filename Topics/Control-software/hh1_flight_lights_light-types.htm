LED Varies—consult OEM manual for light model AC Must connect to an LCU.
DC Must connect to a subsea node port either directly or through a light junction box.

A light junction box lets you connect up to four DC LED lights to a single subsea node port, and an LCU lets you connect up to 12 incandescent lights to a single subsea node port. Lights that are connected to a light junction box or an LCU are not listed individually in the INSTALLED DEVICES display (see Accessing Device Configuration Information). However, the FLIGHT screen LIGHTS panel shows all installed lights individually.

DC Light Connections

DC lights can be connected to:

● A standard subsea node SIM, NIM, or VIM port. Connects a single light to the port.
● A dual SeaNet SIM port. Lets you plug two DC lights into the port. You cannot control each light individually; for example, one of the lights cannot be on while the other is off.
● A light junction box. Lets you connect up to four lights, each with individual intensity control. The light junction box connects to a subsea node port.
