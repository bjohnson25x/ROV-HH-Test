About Light Control

See Also for links to information about ROV and TMS lights.

Purpose: The touch screen LIGHTS panel contains controls for operating lights on the ROV and TMS. Panel appearance varies according to the number and types of lights and how they are connected to the ROV and TMS. You can also control light intensity for selected lights with the tool control pod.

Access: On the FLIGHT screen, tap LIGHTS.

Caution!  While the ROV system is on deck, limit operation of any light to a very quick function check. Longer operation can damage lamps or trigger an automatic over-temperature shutdown.

LIGHTS Panel Functions

The LIGHTS panel lets you:

● Turn lights on and off
● For dimmable lights, change light intensity
● Group lights so that they can be controlled together
● Apply light presets (to determine which lights are turned on and at which intensities)

Light Power

You can turn on an individual light only if the node and port for that light are turned on. For lights attached to an LCU, you can turn on the light only if the 120-volt circuit for that LCU is turned on.

To configure lights for auto on or for fuse disabling, see POWER Configuration Tab: Setting Auto-On and Software Fuses.
