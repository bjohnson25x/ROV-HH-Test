Controlling Lights with Presets

See Also for links to information about configuring and using light presets.

See Also for links to information about ROV and TMS lights.

Light presets specify preconfigured on/off and intensity settings for one or more lights.

You can create and save different light presets to use in different situations. For example, for general ROV flight, you could use a light preset that turns on forward lights at full intensity and dims or turns off all other lights. But for operations with poor visibility, you could switch to a preset that dims forward lights and turns on side-facing lights at full intensity.

Lights that are not included in a preset are not affected when you apply that preset. For example, if you create a preset that includes only forward lights, side-facing lights are not affected when you apply the preset for forward lights. If you want to affect both forward and side-facing lights at the same time, you must include them all in the same preset.

Applying a Preset

To apply the desired light preset:

1. In the LIGHTS panel, tap PRESETS to display available presets.
2. Tap the button for the desired preset. When the preset is applied:
a. Each light that is defined as “on” in the preset turns on, and with the intensity defined in the preset for that light.
b. Each light that is defined as “off” in the preset turns off.
c. Lights that are not in the preset keep their current status.
