Changing a Light's Intensity

You can control light intensity for dimmable lights.

For a light button’s intensity bar, the amount of light green indicates intensity. For example, if half of the bar shows light green, the light is at 50% intensity.

Tapping the down arrow on a light button brings up a panel for adjusting intensity. Drag the slider to adjust intensity.

For dimmable lights:

● If a light is off and you set the intensity to a number greater than zero, the light turns on at the designated intensity.
● If a light is on and you set the intensity to zero, the light turns off.
● If a light is off with its intensity at zero, tapping the button’s left side both turns on the light and sets the intensity to 100%.
● If a light is off with its intensity greater than zero, tapping the button’s left side turns on the light at the set intensity.
