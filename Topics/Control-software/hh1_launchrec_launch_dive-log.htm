Confirm User Profiles

If desired, change the user profile and tactile assignments for the station that you are using.

If you need to change the settings and assignments for a user profile, you must log in to that user profile.

If you need to change the settings and assignments for a control station other than the one that you are using, you must go to that control station and use its touch screen.
