Annotator String Output for Video in TAVS

See Also for links to information about configuring data inputs and outputs.

Note:  This information applies only to ROV systems that have TAVS rather than HDEVS.

The annotator string output from the serial expander supplies ROV navigation data to the annotator serial port. This information is then used to annotate video.

All information in the annotator string is in meters and degrees. The information is output on port 9 of the serial expander at 2 Hz and 9600 baud using the comma-delimited string:

$<Heading>,<ROV Depth>,<ROV Altitude>, <ROV Pitch>, <ROV Roll>, <Date>, <Time>, <CR><LF>

Example string: $226.6, 1.3, 0.0, 00.0, 00.1, 11/10/2010, 19:3:23<CR><LF>
