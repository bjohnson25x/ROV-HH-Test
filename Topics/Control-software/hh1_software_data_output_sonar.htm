Sonar Data Output

See Also for links to information about survey data and data import/export.

Note:  A sonar computer may not be used with this ROV system.

Video output from the rack-mounted sonar computer is directed to a screen on the monitor wall (see About the Control Console Monitors). Sonar data is available at the port designated by the sonar computer configuration.

For information on using sonar software, see the OEM specifications for that software.

To interact with the sonar computer, use the ROV control console KVM switch to assign the system keyboard and mouse to position 3 (sonar).
