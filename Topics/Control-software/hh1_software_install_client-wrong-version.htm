Client Computers Not Running the Same Software Versions

See Also for links to information about installing ROV system software.

If the right and left client computers are not running the same software version, the computer network detects the incompatibility.

This condition can happen:

● If you install a new software version on one client computer, but not the other
● If you revert to a previous software version on one client computer, but do not revert to the same version on the other client computer

To fix the incompatibility:

1. Restart both client computers. The computer network detects that the software versions on the client computers are not the same.
2. On the screens for both client computers, tap GO TO SYSTEM CONFIGURATION.

3. Determine which client computer is not running the desired software version.
4. On the screen for that client computer, tap the button in the SOFTWARE VERSION section.

5. In the drop-down menu for SELECT THE SOFTWARE VERSION TO USE, tap the desired version.
6. On both client computers, tap RETURN TO LAUNCH.

Both computers then start up with the selected software version.
