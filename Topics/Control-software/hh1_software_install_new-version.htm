Installing a New Control System Software Version

See Also for links to information about installing ROV system software.

New control system software versions are periodically delivered and should be installed as soon as you receive them. New versions are generally delivered on an external memory device (DVD, CD, or flash drive).

If the ROV system has a video computer, you must use the same external memory device to install a new version of the Video Computer App on the video computer (see Installing/Configuring the Video Computer App). The software version number of the Video Computer App must match the software version number of the control system computers.

Note:  The procedure below describes how to install a new software version if the external memory device containing the software is inserted into the right client computer. You could alternatively insert it into the left client computer, altering the instructions as appropriate.

To install a new software version:

1. Insert the external memory device containing the new software version into the right client computer.
2. Restart both client computers.
3. As soon as the left client computer displays the CONFIGURE button, tap it.
4. After a short delay, the right client computer displays software that is available for loading. The software version on the external memory device is listed in the screen section labeled FROM LOCAL DRIVE  (USB/DVD).

5. Tap INSTALL for the software version on the external memory device, and the new software is installed on the right client computer. (The message NOT ALL COMPUTERS COULD BE LOCATED may appear on the right client computer.)
6. On the screen for the left client computer, locate the section INSTALL OR UPGRADE SOFTWARE.

7. Tap the button in the INSTALL OR UPGRADE SOFTWARE section.
8. On the drop-down list in the FROM REMOTE CLIENT section, locate the entry for the software that you just installed on the right client computer. (The version number for that software is listed in the RIGHT CLIENT box at the bottom of the right client computer's screen.) On the left client computer, tap INSTALL for the new software version. The software then installs and the left client computer continues the startup process.
9. Remove the external memory device from the right client computer.

Note:  When you install a new software version, all previous versions (including configuration information) are retained. You can easily select and run a previous software version in the rare case of a hardware/software conflict in the new version. Both left and right client computers must be running the same version.
