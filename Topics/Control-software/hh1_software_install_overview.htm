About Installing Control System Software Versions and Patches

See Also for links to information about installing ROV system software.

Purpose: The software installation procedures let you:

● Install a new software version for the ROV control system.
● Install software patches.
● Select and run another software version.
Access: Restart the left and right client computers. Then tap CONFIGURE when that button displays on each computer's screen.

Software Not Covered Here

Information in this section applies only to ROV control system software.

● If you are installing software for some other application (like the Video Computer App or a test application like Node Changer), see documentation for those applications.
● If you are installing a license for a software feature (like Mid-Water StationKeep, the extended language set, or the 8-cylinder ISOL-8 pump) to add that feature to the existing ROV control system, see Installing and Viewing Software Feature Licenses.

Control System Software Introduction

When you start up the ROV control system, the currently installed version of the control system software automatically starts after a short countdown time. To install new software, you must tap CONFIGURE before the countdown time ends.

If you replace a client computer, you must install software that matches the version on the other client computer.

If the ROV system has a video computer, the software version number of the Video Computer App must match the software version number of the control system computers. If you change the control system software version, you must similarly change the Video Computer App version. If the versions do not match, the Video Computer App will be able to operate only in the offline mode (see Video Computer App Offline Mode).

Caution!  If your control console has two operator stations, you must ensure that the software configuration designates one station as a left control station and the other as a right control station. Setting two left or two right control stations causes the control system to malfunction.

Caution!   The left and right client computers must be running the same software version.
