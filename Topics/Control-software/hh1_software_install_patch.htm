Installing a Control System Software Patch

See Also for links to information about installing ROV system software.

Software patches are delivered as needed from the factory, and you should install them promptly. The patch is delivered as a single executable file with the file name extension .SRsvcPatch. If the file is not already on an external memory device (DVD, CD, or flash drive), copy the file into the root level of an external memory device.

Note:  You must install the patch on both the right client computer and left client computer. If you install the patch on only one client computer, the control system software may not function properly.

Note:  The procedure below describes how to install a software patch first into the right client computer and then into the left client computer. You could alternatively install the patch into the left client computer first, altering the instructions as appropriate.

To install a software patch:

1. Insert the external memory device containing the patch into the right client computer.
2. Restart the right client computer. Leave the left client computer running.
3. The screen displays the SELECT THE PATCH TO APPLY section. Select the desired patch from the drop-down list.

4. Tap APPLY for the desired patch.
5. If the patch requires the computer to restart, a panel prompts you to allow the restart. If the computer restarts, it may do so either once or twice, depending on the type of patch that you are installing.
6. After the right client computer installs the patch, the screen returns to the SELECT THE PATCH TO APPLY panel. Tap RETURN TO LAUNCH, and the computer launches the right client control system.
7. Remove the external memory device from the right client computer and insert it onto the left client computer.
8. Restart the left client computer. Leave the right client computer running.
9. Starting with step 3, repeat the instructions above, but for the left client computer instead of the right client computer.
10. Remove the external memory device from the left client computer.
