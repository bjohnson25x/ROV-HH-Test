Reverting To Another Control System Software Version

See Also for links to information about installing ROV system software.

When you install a new control system software version, all previous versions (including configuration information) are retained. You can easily select and run another version in the rare case of a hardware/software conflict in the new version.

Note:  The procedure below describes how to run another software version, first with the right client computer and then the left client computer. You could alternatively start with the left client computer, altering the instructions as appropriate.

To run another software version:

1. Restart the right client computer. Leave the left client computer running.
2. As soon as the right client computer displays the CONFIGURE button, tap it. The screen below then displays.

3. Tap the button in the SOFTWARE VERSION section. In the drop-down list for SELECT THE SOFTWARE VERSION TO USE, tap the desired version and then tap RETURN TO LAUNCH.
4. Restart the left client computer. Leave the right client computer running.
5. Repeat steps 2 and 3, but for the left client computer instead of the right client computer.

Note:  You must select the same software version for the left client computer and the right client computer.

Both client computers then start up with the selected software version.
