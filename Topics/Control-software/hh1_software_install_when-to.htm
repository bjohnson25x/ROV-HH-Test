The table below contains an overview of software installation scenarios. Detailed instructions are provided in other manual sections.

Software Installation Overview
Installation Issue Software Delivery Method Installation Summary

A new version of the software is available.

Note:  When you install a new software version, all previous versions (including configuration information) are retained. You can easily select and run a previous version in the rare case of a hardware/software conflict in the new version.

 New versions of the software are delivered on an external memory device (DVD, CD, or flash drive).

Insert the external memory device into one of the client computers and restart that computer. The software recognizes that a new version is available and lets you install it.

After you install the software on one client computer, you must then install it on the other client computer. You do not need to move the external memory device (like a flash drive) to the other client computer to do this.
A software patch is available.

Can be:

● Delivered on an external memory device
● Sent via e-mail
● Copied from an FTP site

Once you have installed the patch on one client computer, you must then install it on the other client computer. You must move the external memory device to the other computer to do this.

You want to revert to a previous version of the control system software.

 Previous software versions are retained by the client computers, so you do not need to reinstall a previous software version to run it.

Restart both client computers and tap CONFIGURE for each. Select the desired software version for each client computer.

Note:  The left client computer and the right client computer must be running the same software version.

Caution!  If your control console has two operator stations, you must ensure that the software configuration designates one station as a left control station and the other as a right control station. Setting two left or two right control stations causes the control system to malfunction.
