Control System Startup

See Also for links to information about control system startup and shutdown.

Caution!  Avoid power interruptions to the console during the startup process. Do not turn off console power, and ensure that the universal power supply (UPS) is turned on and that batteries are fully charged.

Note:  The PDU must reach full power before you can use all control system functions.

This section presumes that you have started up the PDU (see Power, Telemetry, & Compensation Systems).

To use the ROV system, you must turn on:

● PDU control power
● Power to the control system computers

Powering Up Control System Computers

You do not need to turn on the computers in any particular order.

1. Turn on house power at the van's breaker panel.
2. At the 19-inch rack, turn on the universal power supply (UPS) by holding in the power button (see figure) for several seconds until the system beeps and the LED reads “STARTING.” The UPS will make a clicking sound when the inverter comes on line.
3. Press the switch on the power strip (see figure) to energize the control computers. The computers are set to start up when power is applied. If a computer does not start immediately, open the panel on computer front and briefly press the computer's power switch.
4. If the ROV system has a video computer, see About the HDEVS Video Computer App for information on starting up the video computer.

After a brief pause, the control system application automatically launches and the screen shown below displays. The "LOADING..." text remains until the control system software is operational and the FLIGHT screen displays on the control station monitor.

Pressing the CONFIGURE button (at the screen bottom right) interrupts the loading process. Press this button only if you need to:

● Install a different software version.
● Load a software patch.

Turning on PDU Control Power

To turn on PDU control power for ROV launch, press the button labeled CONTROL POWER - ON on the PDU door.

If you need to cycle PDU power after the ROV system is already operating, go to the diagnostics screen for the topside power subsystem (see Topside Power Subsystem Diagnostics Display) and tap CONTROL POWER.
