Suction Pile Graphical User Interface

See Also for links to information about tool hydraulics.

The SUCTION PILE operation screen provides control and feedback to assist with using a suction pile pump. The suction pile pump operation screen operates in Differential Pressure Flow Calculation Mode (DP Flow Mode). DP Flow Mode requires two Stellar Technology Inc. (STI) sensors.

Feedback data is gathered and calculated as shown in the table below.

Data Sources for Suction Pile Pump
Feedback DP Flow Mode
Displaced volume Calculated from DP sensor connected to Venturi at pump inlet
Stinger differential DP sensor connected to end of stinger
Flow rate Calculated from DP sensor connected to Venturi at pump inlet
Heading ROV heading
Progress Current depth minus seafloor depth

Installation

You must install the Suction Pile tooling package in order to view the suction pile configuration and operation screens. See Adding and Deleting a Hydraulic Tooling Package

Before using the suction pile pump, you must also ensure that an appropriate flow source sensor and stinger source sensor are installed and included in the list of control system devices. See Installing and Removing Devices for more information.

Configuration Screen

Displaying the Suction Pile Configuration Screen

1. From the Main Menu, tap CONFIGURATION.
2. Tap the TOOLING tab at the top of the page.
3. On the left side of the screen, tap the SUCTION PILE tool button.

Note:  The suction pile configuration screen will be locked out when suction pile PUMP CONTROL is enabled, preventing any edits to the configuration.

Adding Suction Pile Devices

LABEL

You can change the "New Suction Pile" label when configuring the Suction Pile tooling package.

PUMP CONTROL

Before you can operate the suction pile pump for the first time, you must select the ROV valve pack valve that will be used to power the suction pile pump. The control system will allow you to select any valve that is proportionally controlled, but you must ensure that the valve has sufficient flow to power the pump for the suction pile operation.

FLOW SOURCE SENSOR AND STINGER SOURCE SENSOR

You must also select the sensors that will be used for flow feedback and stinger pressure.

Calibrating the Flow Source Sensor

If you do not wish to use the default calibration settings for the FLOW SOURCE SENSOR, you may import a calibration file.

1. Insert the USB flash drive containing the CSV calibration file.
2. Tap the LOAD CALIBRATION button, select the file, and tap OPEN.

Note:  The screen will display a graph of the calibration data (pressure, volume, and progress) when the import is successful.

3. After the file has been imported, you will be able to change the label of the calibration data set.

Note:  You can delete any calibration files that have been imported, but can not delete the default calibration.

Operation Screen

Displaying the Suction Pile Operation Screen

1. On the Main Menu, tap TOOLING.
2. Tap the TOOLS tab on the screen top right.
3. Tap the SUCTION PILE tab near the top of the screen.

Note:  While on the Suction Pile Operation Screen, you can access power options, a diagnostics link, and a configuration link (for the assigned valve pack and both STI sensors) by long-pressing on the Suction Pile tab near the top of the screen. You can also change the label for the suction pile from this screen.

Operating the Suction Pile Pump

SUCTION PILE SETUP

The LABEL text box allows you to supply a label. This label will be included in the header of the log file.

The SEAFLOOR DEPTH UPDATE button lets you enter the seafloor depth. Because the seafloor depth value automatically defaults to its most recent setting, you must use the pop-up number entry pad to manually provide seafloor depth.

The ZERO SENSORS button zeroes both the flow sensor and the pressure sensor, at the same time. Only zero the sensors when no pressure is being applied to the sensor ports.

PUMP CONTROL BUTTON

The suction pile pump is controlled by the Pump Control button on the top right of the Suction Pile screen. Turning on pump control will begin logging and allow you to control the valve. Stopping pump control will end logging and disable control of the valve. Re-starting pump control will create a new log file.

Note:  If you want to "pause" pump control in order to continue logging, you can leave pump control enabled and command the valve to 0%.

PUMP CONTROL SLIDER

The pump control slider commands the percentage of maximum opening for the valve on the valve pack. (For example, a value of 25 indicates that the proportional valve is 25% open.) You can command the percentage of flow using the slider or the up/down arrows. If you want to assign a tactile control to the flow percentage, use the EDIT TACTILE ASSIGNMENTS button, located near the bottom right corner of the screen.

Calculated values (such as flow rate) are not displayed until the operator issues a command using the pump control slider.

PROGRESS

ROV heading, ROV depth, ROV altitude (when available), and pile height are displayed in a graphic in the middle of the screen. This display provides quick feedback of these values over the course of the operation.

Current pressure, volume, and progress are displayed in a chart on the right side of the screen.

The pile turns display in the lower right corner indicates how many times the pile has turned since pump control was enabled, based on how the ROV heading changes after starting the suction pile pump.

EXPORT DATA BUTTON

Each time that you use the suction pile pump, the control system saves the chart data on both the left client computer and right client computer.

If you wish to export a log of the chart data to an external memory device:

1. Connect the external memory device (like a USB flash drive) to the client computer that you are using for the export operation. For example, if you are using the left console touch screen for the export operation, insert the external memory device into the left client computer.
2. Tap the EXPORT DATA button.
3. When the export screen panel displays, use the drop-down list to select the chart to export. Because chart files use the CSV file format, they can be imported into most spreadsheet programs.
4. Tap CONTINUE.
5. When the SELECT FILE SAVE LOCATION panel displays, navigate to the location of the external memory device, select the desired folder, and tap SAVE.

Key Steps for Suction Pile Pump Usage

  1. Once the ROV is deployed to the seafloor near the pile, confirm bottom lock with altitude reporting.
  2. Position the ROV near the pile location and note the ROV altitude and depth. Press the SEAFLOOR DEPTH UPDATE button and enter the sum of the ROV altitude and depth.
  3. Set up the ROV on the pile for operations: dock to the pressure/suction port on the pile.
  4. Tap the PUMP CONTROL button to start pump operations.
  5. Once pile operation is complete, tap the PUMP CONTROL button again.
  6. To view the data, tap the EXPORT DATA button in the lower right corner of the screen.
  7. Repeat steps 1 through 6 for the next pile operation.

Troubleshooting

Troubleshooting
Symptom Possible cause(s)
Pile turns display indicates possible loss of data, indicating that the value may not be accurate Loss of ROV heading feedback
Volume display indicates possible loss of data, indicating that the value may not be accurate Loss of communications to the flow feedback sensor

Pump control can not be enabled

Note:  If any of the conditions to the right occur while the pump is being operated, the valve command will be set to zero and an alarm will be raised. However, you will still be able to continue the operation.
● A valve has not been assigned to the pump (see Adding Suction Pile Devices)
● Seafloor depth has not been set (see Operating the Suction Pile Pump)
● The valve pack for the assigned valve has lost communications
● The valve pack for the assigned valve is powered off
● The assigned valve is disabled
● The valve pack for the assigned valve does not have a properly configured HPU connection (see Configuring Hydraulic Device Properties)
● The stinger sensor has not been assigned (see Adding Suction Pile Devices)
● The stinger sensor has lost communications
● The flow feedback sensor has lost communications
● The stinger sensor is powered off
● The flow feedback sensor is powered off
● The flow feedback sensor has not been assigned (see Adding Suction Pile Devices)
● The pressure is too low on the hydraulic circuit the valve is connected to (could be HPU main, HPU aux, or an ISOL-8 circuit)
● The hydraulic circuit the valve is connected to is powered off (no HPU power)
