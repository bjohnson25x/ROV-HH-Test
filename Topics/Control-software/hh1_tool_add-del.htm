Adding and Deleting a Hydraulic Tooling Package

See Also for links to information about tool hydraulics.

Adding a Tooling Package

1. Ensure that the user profile for your control station includes supervisor permissions.
2. Display any CONFIGURATION screen.
3. Tap the TOOLING tab at the top of the page.
4. Tap ADD TOOLING PACKAGE.
5. When the ADD NEW TOOLING PACKAGE panel displays, tap the desired tooling package.

6. Tap CONTINUE.

A configuration page for the new tooling package displays so that you can configure port assignments, tool hydraulics, and other settings.

Deleting a Tooling Package

1. Ensure that the user profile for your control station includes supervisor permissions.
2. Display any CONFIGURATION screen.
3. Tap the TOOLING tab at the top of the page.
4. At the bottom of the left-hand list of tooling packages, tap REMOVE TOOLING PACKAGE.
