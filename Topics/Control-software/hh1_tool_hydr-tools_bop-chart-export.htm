Exporting Chart Data for a BOP Intervention

See Also for links to information about BOP intervention.

Each time that you perform a BOP intervention, the control system saves the chart data for that intervention on both the left client computer and right client computer.

If you wish to export chart data to an external memory device:

1. Connect the external memory device (like a USB flash drive) to the client computer that you are using for the export operation. For example, if you are using the left console touch screen for the export operation, insert the external memory device into the left client computer.
2. Tap EXPORT CHART DATA.
3. When the export screen panel displays, use the drop-down list to select the chart to export. Because chart files use the CSV file format, they can be imported into most spreadsheet programs.

 

4. Tap CONTINUE.
5. When the SELECT FILE SAVE LOCATION panel displays, navigate to the location of the external memory device, select the desired folder, and tap SAVE.
