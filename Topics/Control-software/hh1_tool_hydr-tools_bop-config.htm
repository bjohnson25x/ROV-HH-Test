Configuring the BOP Intervention Tool

The BOP intervention process assumes that you are using a hot stab tool that is controlled by a valve pack valve. You cannot perform a BOP intervention unless you first configure an ROV valve pack valve to control fluid flow to the hot stab.

1. From the Main Menu, tap CONFIGURATION.
2. Tap the TOOLING tab at the top of the page.
3. On the left side of the screen, tap the BOP tool button.
4. From the ISOL-8 PUMP CIRCUIT drop-down list, select the BOP intervention circuit that will pump fluid to the hot stab.
5. From the BOP HOT STAB drop-down list, select a valve pack valve to control fluid flow to the hot stab.
6. If desired, set the valveâ€™s gain (flow as a percentage of full flow).
7. If necessary, switch the A and B output ports for the valve (see Using a Valve to Control the BOP Hot Stab).

Configuring the ISOL-8 Pump

For the ISOL-8 pump BOP intervention circuit, you must configure the circuit to contain the desired number of cylinders (see Schilling Valve Pack Device Configuration Properties).
