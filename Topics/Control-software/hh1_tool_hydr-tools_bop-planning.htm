Planning a BOP Intervention

See Also for links to information about BOP intervention.

As you plan for a BOP intervention, consider these issues:

● To perform BOP intervention, the ROV must have a license for an 8-cylinder ISOL-8 pump. Ensure that this license has been installed (see Installing and Viewing Software Feature Licenses).
● Ensure that the BOP intervention tool has been installed as a tooling package (see Adding and Deleting a Hydraulic Tooling Package).
● Based on ratings for the BOP model, determine the volume of fluid required to close the BOP. The control system uses this information to estimate the time to close the BOP.
● Based on ratings for the BOP model, determine the allowed maximum pressure in the ISOL-8 pump BOP intervention circuit during the intervention. The control system uses this information to estimate the time to close the BOP, to command pressure for the ISOL-8 pump, and to calculate the target pressure for the main hydraulic circuit.
● Ensure that the ISOL-8 pump and the BOP intervention tool are correctly configured for the intervention operation (see Configuring for BOP Intervention).
— If a valve is needed to control flow to the hot stab, ensure that a valve pack valve has been assigned for hot stab control.
— If the ISOL-8 pump has two reservoirs, determine whether you want reservoir contents to mix during the intervention. If so, you must configure the ISOL-8 pump on deck.
