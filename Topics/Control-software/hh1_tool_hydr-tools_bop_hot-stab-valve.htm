Using a Valve to Control the BOP Hot Stab

See Also for links to information about BOP intervention.

The BOP intervention process assumes that you are using a hot stab tool that is controlled by a valve on an ROV valve pack.

Ensure Hot Stab Valve Control

Once you have enabled BOP control, the ROV control system prevents you from manually controlling the HPU main hydraulic circuit and the ISOL-8 pump circuit that you have designated for pumping fluid during the BOP intervention. You can still manually control the ISOL-8 circuit that will not be pumping fluid during the intervention.

If the valve pack containing the hot stab valve uses either the main hydraulic circuit or the ISOL-8 pump circuit designated for fluid pumping, the control system automatically checks to ensure that the fluid circuit can be commanded. If the valve pack containing the hot stab valve uses the ISOL-8 pump circuit that is not designated for the BOP intervention, you must check to ensure that the circuit is being commanded and has sufficient pressure and flow to operate the hot stab valve.

Error Messages for the Hot Stab Valve

Messages appear on the BOP intervention screen if:

● A hot stab valve has not been assigned.
● The hot stab valve is not associated with a hydraulic circuit.
● The hot stab valve has a fault condition (like overcurrent or undercurrent).
● The valve pack is not communicating with the control system.

Closed Position for a BOP Hot Stab Valve

Once you tap the BOP CONTROL button on the BOP intervention screen, the BOP process takes control of the hot stab valve. If you stop or pause the intervention, the BOP processes commands the valve to close, and you can see the “valve closed” position on the valve’s valve pack screen.

If the hot stab is being controlled by a valve with two output ports (A and B), the “valve closed” position on the screen will be different from the usual position. For example, if output port A is being used to allow fluid flow to the hot stab, the closed position for that valve will be set to output port B rather than to the typical center position. If output port B is being used to allow fluid flow to the hot stab, the closed position for that valve will be output port A.

If the hot stab is being controlled by a valve with a single output port, then the “valve closed” position on the screen will be in the center as usual.

The BOP control software assumes that output port A is the “open” position for the hot stab valve. If the open position is actually B rather than A, you must reverse the valve ports by using the screen for configuring the BOP tool (see Configuring for BOP Intervention).
