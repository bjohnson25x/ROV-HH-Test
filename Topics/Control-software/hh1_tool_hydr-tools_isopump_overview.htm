About Operating Tools with the ISOL-8 Pump

See Also for links to information about using the ISOL-8 pump to operate hydraulic tools.

See Also for links to information about tool hydraulics.

For ROV systems with the UHD-III hydraulic configuration, you can optimize the performance of many hydraulic tools by adjusting settings for ISOL-8 pump operation. These settings include pressure for the HPU main circuit and settings for the ISOL-8 pump circuit pressure and flow.

Modes of tool operation are:

● Pressure-control mode (using ISOL-8 pump circuit pressure to control tool force).
● Flow-control mode (using ISOL-8 pump circuit flow to control tool speed).
● Optimized performance mode.
