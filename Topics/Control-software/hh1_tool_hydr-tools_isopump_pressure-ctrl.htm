ISOL-8 Pump Tool Operation in Pressure-Control Mode

See Also for links to information about using the ISOL-8 pump to operate hydraulic tools.

See Also for links to information about tool hydraulics.

Most subsea tools are configured for operation using pressure control, where tool force is more important than tool speed. Follow the instructions in this section to use the ISOL-8 pump in pressure-control mode to operate a typical hydraulic tool.

Note:  Extended operation of a hydraulic tool or operation at pressures above 207 bars (3000 psi) causes heat buildup that can degrade performance and/or damage equipment.

The instructions in this section assume that:

● The ROV HPU motor is running.
● The ROV HPU main circuit is ready for operation.
● Any valve pack that will be used in the tool’s operation has power and is ready for use.
● If the ISOL-8 pump has two circuits, you are using Aux circuit 2 for tool operation. (If you are using Aux circuit 1, adjust instructions as necessary.)
● You have configured Aux circuit 2 to allow flow control (see ISOL-8 Pump Device Configuration Properties).
● You have primed the ISOL-8 pump.
● The ISOL-8 pump Aux circuit 2 is not currently enabled.

To operate a tool:

1. On the FLIGHT screen, tap HPU.
2. When the HYDRAULIC PRESSURE CONTROL panel displays, tap the ROV MAIN tab.

3. On the PRESSURE gauge, drag the slider to change the ROV HPU pressure setting to 207 bars (3000 psi), which is the standard value for high-pressure operations. This pressure will provide sufficient power to operate most tools.
4. Tap the CIRCUIT 2 tab.

5. Tap ENABLE to enable circuit operation.
6. Look at the feedback values on the PRESSURE and FLOW gauges. If these values are approximately zero, continue with the next numbered step. However, because ISOL-8 pump pistons move to their “ready” positions when you enable pump operation, these values may show some flow and pressure. If this is the case, continue with the next lettered step.
a. If the ISOL-8 pump has a standard hydraulic circuit setup that includes Cardev filters, pressure will bleed off. Flow should also stop after the pistons have moved to their ready position. Wait until both the pressure and flow feedback values are approximately zero, and then continue with the next numbered step.
b. If the ISOL-8 pump has a nonstandard hydraulic configuration that does not include Cardev filters, the pressure feedback value can be high (even higher than 3000 psi). To lower the pressure, move the PRESSURE and FLOW sliders upward to briefly operate the tool until the pressure feedback value is approximately zero. After the pump pistons stop moving, the flow feedback value should also be approximately zero. Return the PRESSURE and FLOW sliders to zero, and then continue with the next numbered step.
7. On the ISOL-8 pump circuit FLOW gauge, slowly drag the slider upward to raise the flow setting. Because the PRESSURE gauge setting of zero prevents the pump from operating, the flow feedback value should remain approximately zero. As you drag the FLOW slider, also watch the pressure feedback value and verify that the pressure does not change (that is, it remains approximately zero).

Note:   A changing pressure feedback value indicates a problem; if the value does change, return the FLOW gauge slider to zero and diagnose the problem.

8. Continue to drag the FLOW slider until it is at its maximum value. (You cannot configure this maximum value). The flow feedback value should remain approximately zero.
9. Use the ISOL-8 pump circuit to operate the tool in the same way that you would use a typical pressure-controlled HPU—by raising the ISOL-8 circuit pressure setting to the desired value. As you raise the PRESSURE slider, the pressure feedback value should gradually rise until it reaches a value near the pressure setting.

If the load on the ISOL-8 pump circuit exceeds the circuit’s capacity, the pressure feedback value will begin to lag behind the pressure setting. See Optimizing the ISOL-8 Pump for Tool Operation for information on how to proceed.

Typically, most of the load on the ISOL-8 pump comes from the tool being operated, but the load can also include flow through other items on the ISOL-8 pump circuit or on the ROV HPU main circuit. Examples of these other items include thrusters, Cardev filters, and valve packs.
