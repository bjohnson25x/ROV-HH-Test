Hydraulics for Position-Controlled Manipulators

See Also for links to information about tool hydraulics.

Position-controlled manipulators (like the Titan 4 or Orion 7P) are operated with the free-standing master controller and cannot be controlled from the touch screen. However, you can view the position of the manipulator arm on the touch screen. This function is useful if you cannot adequately view manipulator position with ROV cameras.

If the ROV system includes a feature for performing automatic tool positioning with a position-controlled manipulator, controls for this function appear on the manipulator screen.
