Hydraulics for Rate-Controlled Manipulators

See Also for links to information about tool hydraulics.

Rate-controlled manipulators (like the ATLAS 7R and RigMaster) are typically operated with the rate hand controller, but you can also use screen buttons. You can use only one type of control (rate hand controller or screen) at a time.

Rate-controlled manipulators are typically operated by a valve pack on the ROV main HPU circuit.

To view and control valves for a rate-controlled manipulator, first access the screen for manipulator.

See how
1. From the Main Menu, tap TOOLING.
2. Tap the TOOLS tab in the upper right.
3. Tap the tab in the upper left for the desired hydraulic tool.

Other Manipulator Settings

Long press a manipulator function button to access features that may include:

● Reversing motion direction (for example, if directing flow to port A previously caused a wrist to rotate clockwise, reversing direction will cause the wrist to rotate counter clockwise).
● Setting the flow rate (limiting flow to a percentage of full flow).
● Assigning a valve to the function. In the list of available valves, S indicates a solenoid valve and P indicates a proportional valve.

Setting the Flow Rate for Manipulator Proportional Valves

If functions of a rate-controlled manipulator are assigned to proportional valves, you can set the maximum flow rate for each valve as a percentage of full flow. You can change the flow rate while the valve is open or closed.

To access the flow rate panel, long press the manipulator function label on the screen (for example, the WRIST ROTATE label) and select SET GAIN.
