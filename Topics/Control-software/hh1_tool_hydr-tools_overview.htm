About Controlling Hydraulic Tools

See Also for links to information about tool hydraulics.

Purpose: Lets you:

● Control valves for installed hydraulic tools like rate-controlled manipulators and hydraulic intervention tools.
● View the position of position-controlled manipulators.

Access: On the Main Menu, tap TOOLS and then tap the TOOLS tab.

Note:  The type of displayed items varies depending on the hydraulic tools installed on the ROV system.

When you select the TOOLS tab, all installed tooling packages (including rate-controlled manipulators) are displayed on tabs at the left top. Tap the desired tab to display controls and/or status information for that tool.

Control functions vary depending on the tool’s type. Typically, control functions let you set hydraulic properties for individual tool valves and functions.

See the manipulator manuals for manipulator operation instructions.

WARNING!  Do not operate manipulators or other hydraulic tools on deck when personnel are nearby.
