Assigning Valve Operation to the Tool Pod

See Also for links to information about valve pack control.

You can assign valve operation to a switch or knob on the tool control pod. (You can also continue to operate the valve with the valve’s touch screen controls.)

See the section on tool pod control for information on functions that are typically assigned to particular valve types. (For example, proportional valves are typically assigned to knobs rather than switches.)

To assign a valve to a control pod switch or knob, first display the valve’s screen.

See how
1. From the Main Menu, tap TOOLING.
2. Tap the VALVES tab in the upper right.
3. Tap the tab in the upper left for the desired valve pack.

Then you can access a valve’s tactile control assignment.

4. Tap the valve’s button.
5. Tap EDIT FLOW TACTILE ASSIGNMENTS.
6. When the TACTILE CONTROL ASSIGNMENT panel displays, select RIGHT COMPUTER or LEFT COMPUTER from the drop-down list.
7. Tap the knob or switch that you want to use. Tactile controls that can potentially be used for this function are shown as buttons with rounded edges. (You can control a proportional valve with a switch, but you cannot control a nonproportional valve with a knob.)
8. Tap ASSIGN TO.
9. Tap CONTINUE. The assignment then displays for that valve on the valve pack screen.

You can clear a tactile control assignment or replace a current assignment with a new one.

To remove an assignment from a control pod switch or knob, tap the control and then tap CLEAR.

Caution!   Use caution when you clear a tactile control assignment because some controls may be assigned to critical functions.

You can assign a new function to a tool pod control that is already being used for a function. The new function replaces the old one.
