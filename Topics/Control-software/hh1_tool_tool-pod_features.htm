Tool Control Pod Features

See Also for links to information about valve pack control.

Controls on the switch panel of the tool control pod are independent of each other and can be assigned to operate individual valves in the control system. Only one switch at a time can be assigned to a specific valve.

The tool pod switch panel contains switches and knobs. The controls typically have the following functions:

● Knobs control proportional valves, and are also used for lights and flight functions.
● Single-function switches control one solenoid (on/off) valve and are also used for lights. If you assign a proportional valve to a switch, the valve will behave as though it is a solenoid valve (either on or off).
● Two-function switches control two solenoid valves (one with up/down switch positions and the other with left/right positions).

A single-function switch is either momentary or maintained:

● Momentary: switch returns to OFF when you release the switch
● Maintained: switch stays in the commanded position after release

All two-function switches are momentary.

Note:  You can assign a single-function switch to operate a proportional valve, but the switch will only fully open or close the valve to the percentage set in flow rate panel (see Setting Flow Rate for Proportional Valves).

Each knob clicks through 32 positions to make a full rotation. Four full rotations are allowed in each direction (with no mechanical stops), corresponding to 128 discrete values forward and 128 values back.

To reset a knob to the zero/neutral position (regardless of the mechanical position of the knob), push in the knob.

Note:  You can assign a proportional knob to a solenoid valve, but it will only fully open or close the valve.
