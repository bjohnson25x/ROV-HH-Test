Schilling 12-Station Valve Pack

See Also for links to information about valve pack models.

A Schilling 12-ST valve pack can contain up to 12 proportional valves. Valve types can include:

● Valves with two output ports and with flow control
● Valves with two output ports, flow control, and pressure control
● High-flow valves with a single output port and flow control

The type of each valve is automatically detected by the ROV control system and displayed on the screen that contains valve pack controls.

The Schilling 12-ST valve pack is typically used with an auxiliary (tooling) hydraulic circuit.

You cannot operate all valves on the valve pack without exceeding the total current that is available at the valve pack’s node port. For this reason, you should monitor the current gauge at the bottom of the screen when valves are operating. This gauge shows the total current draw for all valves on the valve pack, and the yellow bar marks the allowed current limit. If current draw exceeds this limit, the gauge turns orange and the control system generates an alarm.

For valves with pressure control, a status indicator shows the feedback pressure along with a chart of pressure over time. If a valve with pressure control is not currently being commanded to control pressure, the status indicator is grayed out.
