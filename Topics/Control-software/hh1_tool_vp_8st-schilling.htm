Schilling 8-Station Valve Pack

See Also for links to information about valve pack control.

See Also for links to information about valve pack models.

A Schilling 8-ST valve pack can contain up to eight proportional valves. If the ROV system includes a rate-controlled manipulator, this manipulator’s functions are typically assigned to the Schilling 8-ST valve pack.

The type of each valve is automatically detected by the ROV control system and displayed on the screen that contains valve pack controls.

If any valve pack valves are assigned to a rate-controlled manipulator or other tool, those valves show on the screen as unavailable for assignment or control for other functions.
