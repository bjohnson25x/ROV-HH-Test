Purpose: For the ROV and TMS, lets you control individual valves on valve packs. Some valve packs also let you adjust valve pack and HPU flows and pressures.

Access: From the Main Menu, tap TOOLING. Then tap the VALVES tab in the screen upper right.

Note:  Proportional valves can be operated with a proportional control like a tool pod knob that lets you vary the flow by turning the knob. However, proportional valves can also be operated with a nonproportional control like the rate hand controller or a tool pod switch. If the valve is being operated with a nonproportional control, the valve behaves like a solenoid valve, and flow is either off or fully on.

If you have assigned valve pack valves to functions for a rate-controlled manipulator, those valves will show on the valve pack’s screen as being unavailable for other assignment.

You can operate a valve using:

● Screen controls
● A switch or knob on the tool control pod
● The rate hand controller

Before you can view valve status or use screen controls for individual valves, you must display the screen for the valve pack that contains the valve.

See how
1. From the Main Menu, tap TOOLING.
2. Tap the VALVES tab in the upper right.
3. Tap the tab in the upper left for the desired valve pack.

Tapping a valve’s button lets you access a panel with valve controls. These controls let you:

● Enable and disable valve operation (see Enabling and Disabling Operation for a Valve).
● For proportional valves only, set the flow rate (see Setting Flow Rate for Proportional Valves).
● For pressure-reducing valves only, set valve pressure (see Setting Valve Pressure).
● Edit a valve’s tactile control assignment (assigning valve control to a switch or knob on the tool control pod); see Assigning Valve Operation to the Tool Pod.

A valve’s flow controls let you turn valve flow on and off (see Turning Valve Flow On and Off).

Long pressing a valve’s button lets you:

● Edit the valve label.
● Edit the action labels for port A and port B.
● Edit the valve’s tactile control assignment.
