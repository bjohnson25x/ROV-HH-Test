Setting Flow Rate for Proportional Valves

See Also for links to information about valve pack control.

You can vary the flow rate only for a proportional valve. Valve behavior then depends on whether the valve is assigned to a proportional control (like a knob) or a nonproportional control (like a switch).

● For a proportional control like valve screen buttons or a knob on the tool control pod, you can set the maximum flow rate as a percentage of full flow. (Setting the maximum flow percentage is called setting the gain.) You can then operate the valve over a range from zero flow up to the set maximum. If you do not set a maximum flow, the default value is 100 percent of full flow.
● For a nonproportional control like the rate hand controller or a switch on the tool control pod, you can still set the maximum flow rate as a percentage of full flow. During operation, the valve behaves like a solenoid valve, and flow is either off or fully on.

On the screen, flow control is called “flow gain.”

Before you can set the maximum flow rate for a valve, you must access the screen for the valve pack that contains the valve.

See how
1. From the Main Menu, tap TOOLING.
2. Tap the VALVES tab in the upper right.
3. Tap the tab in the upper left for the desired valve pack.

Setting Maximum Flow Rate

These instructions describe how to set the flow rate for valves that are not assigned to a rate-controlled manipulator or other nonproportional control. To assign flow rates for manipulator-assigned valves, see Hydraulics for Rate-Controlled Manipulators.

To set the maximum flow rate, display the flow rate panel for the desired valve. You can change the flow rate while the valve is open or closed.

When the FLOW GAINS panel displays, set the desired maximum flow rate as a percentage of the valve’s full flow. In the figure below, the valve has two output ports (A and B).

Varying Flow During Valve Operation

If you are operating the valve from the screen:

1. In the flow rate (FLOW GAINS) panel, set the desired maximum flow rate.
2. Operate the valve as shown in Turning Valve Flow On and Off.
3. To change flow rate, use the FLOW GAINS panel and set a new maximum flow.

If you are operating the valve from a proportional tactile control (a tool control pod knob):

1. In the flow rate (FLOW GAINS) panel, set the desired maximum flow rate.
2. Initially, the knob is set for 3% of maximum flow (even if a different flow is set on the screen). Turn the knob to increase or decrease the flow rate. Each knob click changes the flow by 3%.

If you are operating the valve from a nonproportional tactile control (like the rate hand controller or a tool pod switch):

1. In the flow rate (FLOW GAINS) panel, set the desired flow rate.
2. Use the tactile control to turn the valve on and off.
3. To change the flow rate, use the FLOW GAINS panel.
