Setting Valve Pressure

See Also for links to information about valve pack control.

Some valves let you control pressure, adjusting it from zero up to system operating pressure.

Before you can set the valve pressure, you must access the screen for the valve pack that contains the valve.

See how
1. From the Main Menu, tap TOOLING.
2. Tap the VALVES tab in the upper right.
3. Tap the tab in the upper left for the desired valve pack.

You cannot control pressure for a valve that is assigned to a rate-controlled manipulator, even if that valve has pressure-setting capability.

Setting Pressure

To set valve pressure, display the pressure/flow rate panel for the desired valve. You can change the pressure while the valve is open or closed.

Note:  Because the pressure feedback sensor measures pressure on the valve output, adjusting the commanded pressure when the valve is closed usually does not affect pressure the pressure feedback. When the valve is opened, the pressure feedback will change.

When the pressure/flow rate panel displays, set the desired pressure as a rough percentage of full system pressure. (The same panel lets you set flow rate for proportional valves.)

Varying Pressure During Valve Operation

If you are operating the valve from the screen:

1. In the pressure/flow rate panel, set the desired pressure.
2. Operate the valve as described in Turning Valve Flow On and Off.
3. To change pressure, display the pressure/flow rate panel and set a new pressure. Continue to adjust the pressure until you achieve the desired tool performance.

If you have assigned valve pressure to a proportional tactile control (like a tool control pod knob):

1. In the pressure/flow rate panel, set the desired pressure.
2. Turn the knob to increase or decrease valve pressure. Each knob click changes the pressure by 3 percent.
