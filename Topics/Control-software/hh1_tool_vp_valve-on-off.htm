Enabling and Disabling Operation for a Valve

See Also for links to information about valve pack control.

You cannot operate a valve from the screen unless that valve is enabled. By default, valves are automatically enabled when you start up the ROV control system. After startup, you can disable and reenable individual valves.

By default, valves are automatically enabled (ready to be operated) when you start up the ROV control system.

Before you can enable or disable operation of a valve pack valve, you must access the screen for the valve pack that contains the valve.

See how
1. From the Main Menu, tap TOOLING.
2. Tap the VALVES tab in the upper right.
3. Tap the tab in the upper left for the desired valve pack.

To enable or disable operation for a valve:

1. Tap the valve button that contains a down arrow.

2. A button displays that is labeled either ENABLED or DISABLED, indicating the valveâ€™s current state. Tap the button to change the state.

When the button label is ENABLED, the valve is ready for operation. When the button label is DISABLED, you cannot operate the valve.
