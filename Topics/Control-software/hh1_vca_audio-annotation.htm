Adding Audio Annotations to Recorded Video

See Also for links to information about the Video Computer App. The Video Computer App is used only with HDEVS.

This information applies only to ROV systems that have HDEVS rather than TAVS.

You can record an audio annotation at a specific location in a video segment by putting an event place marker at that location and recording audio there.

Annotations can be made either while video is being recorded or added later during video editing.

You cannot edit an audio annotation that has already been created, but you can delete it and record a replacement.

Recorded audio has these characteristics:

● An audio annotation must start at the location of an event place marker.
● Maximum length for a single audio annotation is 15 minutes (though most annotations will be much shorter). If you want to record an annotation that is longer than 15 minutes, you can start a second one immediately after the first.
● An audio annotation can continue from one segment to another (as long as the total length is not more than 15 minutes), but the segments must be continuous. If even a small time gap exists between segments, you must start a new audio annotation for the new segment.
● Audio annotations cannot overlap each other. If you are adding an audio annotation after video recording has already taken place, the audio annotation cannot continue past an event place marker that also contains audio. (The annotation automatically stops.) However, if the event place marker doesn’t include audio, the annotation continues past the event place marker. If you see that an audio annotation is going to interfere with another, more important annotation, you can delete the less important annotation from its event place marker.

Adding Audio Annotation During Video Recording

You can add audio annotation while video is being recorded.

See how
1. Ensure that an audio input device is connected to the video computer.
2. Use options on the CONFIGURATION screen of the Video Computer App to set audio recording options.
a. Select the audio input device. All audio input devices detected by Windows appear in the drop-down list of devices. You can use a USB headset or a stand-alone microphone.
b. Speak in a clear voice, and adjust the volume of the audio input device until the input volume fills two thirds of the screen gauge.
3. Display the control that you will use to create an event place marker for the audio annotation. You can insert an event place marker from:
— The Video Computer App using the EVENTS panel on the VIDEO DASHBOARD (see Inserting Event Place Markers from the Video Computer App). Ensure that the desired event set is displayed.
— The ROV control system touch screen using the EVENTS panel on the FLIGHT screen (see Inserting Event Place Markers from the FLIGHT Screen). Ensure that the desired event set is displayed.
— Button 2 on the FLIGHT joystick, if that button is configured for event triggering (see Assigning Video Event Marking to the Flight Joystick). The button can add unplanned events only.
4. At the desired point in recording, add an event place marker. Instead of touching and releasing the control for adding the marker (a screen button or the flight joystick button), hold down the control.
5. After a half second, the screen shows with a red circle that audio recording has begun, and you can speak into the microphone.
6. Continue to hold down the button until you are finished with the audio annotation.

Audio recording continues until you release the control button or 15 minutes have passed, whichever comes first.

Note:  If multiple video computers are online when you add the event place marker, that marker is added to all video being recorded by all video computers. However, the audio annotation is added only to video computers that have configured microphones.

Adding Audio Annotation After Video Is Recorded

When you are editing recorded video, you can add audio annotation to any event place marker. If an event place marker does not exist at the desired location in the video, you must add an event place marker and then add the audio annotation.

See how
1. Ensure that an audio input device is connected to the video computer.
2. Use options on the CONFIGURATION screen of the Video Computer App to set audio recording options.
a. Select the audio input device. All audio input devices detected by Windows appear in the drop-down list of devices. You can use a USB headset or a stand-alone microphone.
b. Speak in a clear voice, and adjust the volume of the audio input device until the input volume fills two thirds of the screen gauge.
3. Display the PLAYBACK screen of the Video Computer App and use the drop-down to select the desired project.
4. Click the Events tab.
5. Click the desired event place marker.
6. Click and hold Record Audio.
7. After a half second, the button displays as a red circle, indicating that audio recording has begun. Start speaking into the microphone.
8. Continue to hold down the button until you are finished with the audio annotation.

During recording, the button shows how much time is left before the recording must end (either because it has reached the 15-minute limit or because it has reached another event place marker that has an audio annotation).

Deleting an Audio Annotation

1. Display the PLAYBACK screen of the Video Computer App and use the drop-down to select the desired project.
2. Click the Events tab.
3. Click the desired event place marker.
4. Click Delete Audio.

The event place marker remains, but without attached audio.
