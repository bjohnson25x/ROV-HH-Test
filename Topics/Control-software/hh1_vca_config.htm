Configuring Recording Details and Viewing Overlays

See Also for links to information about the Video Computer App. The Video Computer App is used only with HDEVS.

See Also for links to information about configuring for recording on the video computer. This information applies only to ROV systems with HDEVS.

This information applies only to ROV systems that have HDEVS rather than TAVS.

Purpose: The VIDEO CONFIGURATION screen of the Video Computer App lets you:

● Use all functions available in the video preview panel.

● Set the number, type, and quality of video channels for recording .
● Specify the audio input device for audio annotations.

● Change project labels (project name, location, and dive number).
● View the current overlay (and access a screen for editing the overlay).
● Export the current overlay or import a saved overlay.
● Create a video diagnostic snapshot
Access: Click the Gear icon on the VIDEO DASHBOARD.

For information on creating a video diagnostic snapshot, see Creating a Video Diagnostic Snapshot.

Configuring Recording Details

Use the Recording Details panel to configure video types and quality.

After you have changed any items in the Recording Details panel, click SAVE. To cancel your changes without saving, click the Home icon to return to the VIDEO DASHBOARD.

See details

Video Type

From the drop-down list, select the combination of video streams for recording:

● HD/HD: records two HD video streams.
● HD/SD/SD: records one HD and two SD video streams.
● SD/SD/SD/SD: records four SD video streams.

File Template

Select the preferred file name template for recorded video segments. Choose the template that will let you most easily sort and identify the video file names.

Video Quality

● Smallest Size: decreases video quality and decreases the space needed to store the video.
● Best Quality: increases video quality and increases the space needed to store the video.

Audio Input

Use the Audio Input drop-down box to select an input device for audio annotations. The list contains all audio input devices that have been detected by the Windows operating system.

Use the audio volume gauge to set the volume for annotations. Speak in a clear voice, and adjust the volume of the audio input device until the input volume fills two-thirds of the screen gauge.

Project Details

In the Project Details panel, click in fields to enter information for:

● Project name
● Location
● Dive number

The information is used to categorize recorded video segments.

If you leave fields blank, that information will be missing in descriptions of recorded video segments.

After you have changed any items in the Project Details panel, click SAVE. To cancel your changes without saving, click the Home icon to return to the VIDEO DASHBOARD.

Viewing the Overlay Panel

The OVERLAY DETAILS panel displays images (like a company logo) and data (like the current date/time, ROV depth, and ROV heading) that can overlaid on a video stream for viewing or recording. After you create an overlay layout, you can save it for future use.

See details

For information on creating and editing overlay layouts, see Creating/Editing Video Overlays. For information on adding an overlay to a video stream, see About Recording Video with the Video Computer.

You can use buttons on the right of the overlay panel to manage overlays:

● Click Import to load a saved overlay layout.
● Click Export to save the current overlay layout for future use.
● Click Edit to go to the OVERLAY LAYOUT EDITOR.
