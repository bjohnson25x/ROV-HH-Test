Using the VIDEO DASHBOARD

See Also for links to information about the Video Computer App. The Video Computer App is used only with HDEVS.

This information applies only to ROV systems that have HDEVS rather than TAVS.

Purpose: The VIDEO DASHBOARD of the Video Computer App lets you:

● Use the video preview panel to select video streams from ROV system cameras for previewing, overlaying, and recording. Selecting a video stream automatically begins continuous backup recording.
● View project labels (project name, location, and dive number).
● Put event place markers in video as it is being recorded (to mark selected locations in the video).
● View a list of external memory devices that are currently enabled for client recording.
● View and select from a list of recently recorded video segments.
● Access other screens of the Video Computer App.
Access: Displays when you start up the Video Computer App or when you click the Home icon in the upper right corner of other Video Computer App screens.

Video Preview Panel

This panel shows video streams that have been selected for recording. For information on selecting video streams to display in the panel, see About Recording Video with the Video Computer.

Projects Details Panel

Information in this panel is used to categorize video segments. You can change the project details on the VIDEO CONFIGURATION screen (see Configuring Recording Details and Viewing Overlays).

EVENTS Panel

Video Computer Information Panel

When the Video Computer application is on offline mode, error messages appear in this panel.

ENABLED STORAGE Panel

The ENABLED STORAGE panel shows which external memory devices are ready to receive client recording video. As soon as you start client recording, all video streams currently being routed to the video computer will be copied onto all external memory devices in the ENABLED STORAGE panel.

RECENT VIDEOS Panel

This panel lists the most recent video segments. HD video segments are labeled with an “HD” icon.

Click a video segment to automatically start playback of that video.
