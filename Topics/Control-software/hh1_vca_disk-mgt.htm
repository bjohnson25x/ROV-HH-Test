Managing External Memory Devices

See Also for links to information about configuring for recording on the video computer. This information applies only to ROV systems with HDEVS.

Purpose: The DISK MANAGEMENT screen of the Video Computer App lets you:

● Manage the recording status of external memory devices attached to the video computer.
● View the status of video computer internal drives.
Access: Click the Disk Drive icon on the VIDEO DASHBOARD.

This information applies only to ROV systems that have HDEVS rather than TAVS.

The DISK MANAGEMENT screen is divided into the following panels:

● The video preview panel shows all video streams that are currently selected for recording. These video streams are currently being recorded on the video computer internal drives in continuous backup recording.
● The INTERNAL panel shows the status of the internal drives that are performing continuous backup recording.
● The AVAILABLE STORAGE panel shows external memory devices that are attached to the video computer but have not been enabled to allow client recording on the devices.
● The ENABLED STORAGE panel shows external memory devices that are attached to the video computer and have been enabled to allow client recording.

In the AVAILABLE STORAGE panel, a drop-down next to each external memory device lets you edit the label for the device or enable the device for client recording (see Configuring for Client Recording).

In the ENABLED STORAGE panel, a drop-down next to each external memory device lets you edit the label for the device or finalize the device to end a recording session (see Finalizing Client Recording for Export).
