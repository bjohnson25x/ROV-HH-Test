You can put an event place marker at a specific location in a video segment so that you can search for and go directly to that marked location. You can also add event type descriptions and audio annotation at event place markers.

Defining and Grouping Events

Events are organized by individual event types that are grouped into event sets.

For example, you could create an event set called Land and Lock Tree and BOP. Within that event set, you could create individual event types that included:

● Install VX Ring in Wellhead
● Orient and Land Tree
● Remove Tree Running Tool

Each event type can have a text description that adds more detail about the event type.

You can create and delete event sets and event types from the ROV control system touch screen (see Configuring Video Event Sets and Event Types (HDEVS Only)). You cannot create and delete event sets and types from the Video Computer App.

Adding Event Place Markers to Recorded Video

You can put an event place marker at a specific location in a video segment so that you can search for and go directly to that marked location. Event place markers can be added:

● While video is being recorded. The event place marker goes on all video currently recording, both with client recording and continuous backup recording.
● After video has been recorded. When you put an event place marker on a recorded video segment, identical event place markers are automatically put on all video segments that were recorded at the same time.

When you add an event, it is added not only to video being recorded on the video computer used to add the event, but to all video being recorded by all video computers that are online (that is, all video computers that are communicating with the ROV control system). When you edit or delete an event, those changes take place for all instances of that event on all video computers that are online.

Marking Unplanned Events

Typically, you will define event sets and event types before you begin a job. However, sometimes during an operation, you will want to mark a video location for some event that you did not anticipate or that you forgot to define before you started the operation.

You can add an event place marker for an unplanned event using the same screen controls that you would use for defined event types. You can also use a flight joystick button to add an unplanned event. To use the flight joystick for this function, however, you must configure that flight joystick button (see Assigning Video Event Marking to the Flight Joystick).
