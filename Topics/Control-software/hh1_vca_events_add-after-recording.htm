Adding Event Place Markers After Video Recording

See Also for links to more information about using event place markers in recorded video. This information applies only to ROV systems with HDEVS.

This information applies only to ROV systems that have HDEVS rather than TAVS.

The PLAYBACK screen of the Video Computer Application lets you add event place markers to video segments that have already been recorded.

When you put an event place marker on a recorded video segment, identical event place markers are automatically put on all video segments that were recorded at the same time. For example, suppose that cameras A, B, and C recorded video segments at the same time. After recording has occurred, you can play back the segment for camera A and add an event place marker. Identical event place markers are then automatically added to the segments from cameras B and C.

To add an event place marker to a recorded video segment:

1. Display the PLAYBACK screen for Video Computer Application.
2. Select the desired project and internal drive.
3. Click the Videos tab.
4. Click the desired video segment to start playback.
5. When playback reaches the event for which you want to add an event place marker, click the camera icon. An event (of the unplanned event type) is added to that location for all event segments that were recorded during that time.

When you add an event from the PLAYBACK screen, controls are automatically displayed to let you edit that event (for example, to change the event type and change the description). For information on editing an added event, see Editing and Deleting Event Place Markers.
