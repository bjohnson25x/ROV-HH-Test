Adding Event Place Markers During Video Recording

See Also for links to more information about using event place markers in recorded video. This information applies only to ROV systems with HDEVS.

This information applies only to ROV systems that have HDEVS rather than TAVS.

The EVENTS panel lets you add event place markers to video that is currently being recorded. You can also change the description of an event place marker that you just added to the recorded video.

You can access the EVENTS panel from both the VIDEO DASHBOARD and from the ROV control system FLIGHT screen. Panels at both locations let you perform the same functions.

You can also use the flight joystick to add event place markers for unplanned events.

Inserting Event Place Markers from the Video Computer App

See how

 

1. Ensure that the desired video stream or streams are being recorded. Be aware that when you insert an event place marker, that marker goes into all video streams that are currently being recorded by all video computers that are online.
2. Display the VIDEO DASHBOARD.
3. If the EVENTS panel does not show the desired event set, display the drop-down list of event sets and click the desired set.
4. At a place in the video stream where you want to insert an event place marker, click the button for the desired event type.
5. Look at the description of that event type (shown in the Description box at the bottom of the EVENTS panel). If you want to change the description of the event type for only this instance, type the new description in the box.
6. Click Close. (Once you click Close, you will not be able to edit the event place marker description from the EVENTS panel. You can, however edit the event place marker from the Events tab of the PLAYBACK screen.)

The event place marker is then inserted into all video streams being recorded.

Modifying One Instance of an Event Type Description

The EVENTS panel displays the name of the most recently inserted event type, along with the description of the event type. This description is configured in the ROV control system.

You may sometimes want to change the description for just one instance an event type. (For example, if the description is “inspect pipe junction,” you might want to change the description for one instance to “inspect pipe junction, saw damage.”) You can make this change by inserting the event place marker and then typing a new description.

If you then insert another instance of the same event type, the description defaults to the text in the event type’s configuration.

If you want to change the description for all instances of that event type, you must change the event type’s configuration as described in Configuring Video Event Sets and Event Types (HDEVS Only).

Marking Unplanned Events with the Flight Joystick

1. Ensure that the desired video stream or streams are being recorded. Be aware that when you insert an event place marker, that marker goes into all video streams that are currently being recorded by all video computers that are online.
2. Ensure that the flight joystick is configured so that button 2 inserts an event place marker for an unplanned event (see Assigning Video Event Marking to the Flight Joystick).
3. Press button 2 to insert a marker for an unplanned event.
