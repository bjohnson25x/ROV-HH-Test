Editing and Deleting Event Place Markers

See Also for links to more information about using event place markers in recorded video. This information applies only to ROV systems with HDEVS.

This information applies only to ROV systems that have HDEVS rather than TAVS.

The PLAYBACK screen of the Video Computer Application lets you edit and delete event place markers in video segments.

To edit an event place marker, you must first select the marker.

Displaying the List of Event Place Markers

See how

To edit the event type or description for an event place marker in a recorded video segment:

1. Display the PLAYBACK screen for Video Computer Application.
2. Select the desired project.
3. Click the Events tab.

Editing an Event Place Marker

For any event place marker, you can change the event type and/or description.

See how
1. Display the Events list (see Displaying the List of Event Place Markers).
2. Click the desired event place marker.
3. Click Edit Event.

4. Select the desired event set.
5. Select the desired event type.
6. If you wish to change the default description for only this instance of the event type, you can click the description and type a different one.
7. Click Save.

Deleting Event Place Markers

You can delete multiple event place markers at once from the Events list.

See how
1. Display the Events list (see Displaying the List of Event Place Markers).
2. Click Delete Events.
3. For each event place marker that you want to delete, put a check mark in the box in the Delete column.

4. Click Delete.
