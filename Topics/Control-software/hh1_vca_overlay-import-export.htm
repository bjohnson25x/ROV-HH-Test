Importing/Exporting Video Overlays

See Also for links to information about configuring for recording on the video computer. This information applies only to ROV systems with HDEVS.

This information applies only to ROV systems that have HDEVS rather than TAVS.

After you create an overlay layout, you can save it on an external memory device (like a USB flash drive) for future use. Once you have saved an overlay layout, you can import the layout at any time to use it or edit it.

Exporting (Saving) an Overlay Layout

Once you have created a layout, you can save it on an external memory device for future use.

See how
1. Connect an external memory device to the video computer.
2. Go the VIDEO CONFIGURATION screen.
3. Click EXPORT (on the upper right of the overlay display).
4. When the SELECT FILE SAVE LOCATION panel displays, navigate to the desired folder on the external memory device. (You can click a folder name in the file path to go to that folder.)

5. Type the desired file name. The name must have the file name extension “.srOverlay.”
6. Click on the desired overlay file and then click SAVE.

Importing a Saved Overlay Layout

As soon as you import a saved overlay layout, that layout is immediately used for all viewed and recorded video streams.

See how
1. On the video computer, connect the external memory device that contains the saved overlay layout.
2. Go the VIDEO CONFIGURATION screen.
3. Click IMPORT (on the upper right of the overlay display).
4. When the SELECT FILE TO OPEN panel displays, navigate to the desired folder on the external memory device. (You can click a folder name in the file path to go to that folder.)

5. Click OPEN.

Importing Incompatible Overlay Layouts

When you import an overlay layout, you may sometimes see in the OVERLAY LAYOUT panel that an item in that layout is enclosed in red box. This can happen when the overlay layout contains an item that is incompatible with the ROV system configuration. For more information, see Overlay Layouts with Incompatible Items.
