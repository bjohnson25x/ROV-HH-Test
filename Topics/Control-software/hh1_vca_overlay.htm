Creating/Editing Video Overlays

See Also for links to information about configuring for recording on the video computer. This information applies only to ROV systems with HDEVS.

This information applies only to ROV systems that have HDEVS rather than TAVS.

For information in creating a video diagnostic snapshot, see (Creating a Video Diagnostic Snapshot).

The OVERLAY LAYOUT EDITOR screen lets you create new layouts and edit existing ones.

To display the OVERLAY LAYOUT EDITOR:

1. Go the VIDEO CONFIGURATION screen (see Configuring Recording Details and Viewing Overlays).
2. Click EDIT (on the lower right of the overlay panel).

Adding/Deleting Overlay Items in a Layout

The left side of the OVERLAY LAYOUT EDITOR screen contains a menu of items that you can add to an overlay layout. To add an item, click the menu item and drag it from the menu onto the desired location on the layout grid.

To delete an item from the grid, click the item and then click the red X at the grid top left.

The menu of overlay items includes operations information, images, and user-defined text.

See details
● ROV flight information (like ROV velocity and orientation)
● TMS information (like TMS depth and, if the TMS has a compass, heading)
● Survey information (like survey position and kilometer post [KP])
● Hydraulics information (like HPU pressure and reservoir volume)
● Tooling hydraulics (like valve pack pressure)
● Miscellaneous information (like a text label, current date and time, and a logo image)

Editing Overlay Items

Note:  Changes to an overlay layout are saved as soon as you make them.

Click an overlay item to select it, and then edit the item using functions on the layout ribbon.

For many selected items, the layout ribbon looks like this:

However, functions on the layout ribbon change depending on which layout item you have selected. For example, if you select an image, options for text-based characteristics (like font type and size) disappear.

If you have stacked overlay items on top of each other, you can use the “order in stack “ buttons to determine the order in which the items are stacked.

To size a layout item, select it and then drag the item’s handles to make the item larger or smaller.

If you drag the Image menu item into the layout grid, the Change Image button displays on the ribbon. Click this button and navigate to the desired image to select it.

Specifying Color for an Overlay Item

Click the Color button to display a drop-down list of color selections. You can:

● Click a color in the color chart.
● Enter color values by dragging the sliders.
● Type color values directly into the value boxes. The top color value box lets you type in a hexadecimal value in order of ARBG (see below)

Color component labels are:

● R = amount of red light in the color
● G = amount of green light in the color
● B = amount of blue light in the color
● A = amount of transparency (with zero producing a completely transparent and therefore invisible layout item)

To close the color drop-down, click anywhere else on the screen.

Undoing Changes to an Overlay Layout

Changes to an overlay layout are automatically saved as soon as you make them. If you want to discard all changes to a layout and revert to the original, click UNDO ALL CHANGES.

Note:  Once you click BACK to return to the VIDEO CONFIGURATION screen, you cannot use the UNDO ALL CHANGES to return an overlay layout to its original condition. If you want to undo changes, be sure to click UNDO ALL CHANGES before you click BACK.

Overlay Layouts with Incompatible Items

Sometimes when you create or import overlay layouts, an item in the layout can be incompatible with the ROV system configuration.

When an incompatible item appears in the overlay layout, it is surrounded by a red rectangle and an error message displays on the screen.

Incompatibilities can exist because:

● The overlay item cannot be used with this model of ROV system. (For example, an overlay item for ISOL-8 pump flow cannot be used on an HD ROV system because the HD cannot use the ISOL-8 pump.) In this case, you will typically want to delete the incompatible overlay item.
● The input for overlay item does not exist for the ROV system. (For example, the overlay contains an item for TMS heading, but the TMS does not have a compass.) In some cases, you may be able to correct the error by deleting the overlay item and then reinserting it.
