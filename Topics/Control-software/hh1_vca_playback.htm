Playing Recorded Video

See Also for links to information about the Video Computer App. The Video Computer App is used only with HDEVS.

This information applies only to ROV systems that have HDEVS rather than TAVS.

Purpose: The PLAYBACK screen of the Video Computer App lets you:

● View a list of files for all recorded video segments.
● Filter the video file list by internal drive (hard drive) and project name.
● Select video segments for playback or deletion.
● Export video segments to an external memory device.
● View a list of all events place markers that are contained in the listed videos.
● Insert and edit event place markers.
● View and export a still photo from the video located at an event place marker.
Access: Click the Playback icon on the top right of the VIDEO DASHBOARD list of recorded videos.

You can play back any video segment stored on the video computer internal drives (hard drives).

By selecting the Videos tab, you can display a list of video segments. By selecting the Events tab, you can view a list of event place markers that are contained in the listed videos.

For information about adding and editing events to video segments that have already been recorded, see Adding Event Place Markers After Video Recording.

Selecting a Video for Playback

Click the Videos tab to display a list of videos. Then use the drop-down list to filter the list by project.

See how

In the list of video segments for the project, click a segment to start playback of that segment.

Typically, it will not matter which internal drive you select to filter the files because each video file is copied onto both drives. However, if one drive fails, you can use the filter to select the drive that is still operating.

A camera indicator (white square) displays next to the selected segment and all segments from the same camera. If you select a video segment from another camera, the camera indicators then display next to all segments from that camera.

Using Video Playback Controls

When you select a video segment, it immediately begins playing. If you do not stop the segment or select a new one, the segment plays until it reaches the end.

The playback controls include a check box for Continuous Playback. If this box is checked, as soon as a video finishes playing, the next video in the list automatically begins to play. If this box is not checked, no new video begins.

Controls beneath the video display let you pause play, drag the progress bar to quickly go forward or backward through the segment, or move to the next segment or previous segment recorded by that camera. You can also insert an event place marker into the recorded video, which lets you capture a still photo from that video location (see Adding Event Place Markers After Video Recording).

See how

Because the video computer can record multiple video streams at the same time, you may have simultaneously recorded multiple views of the same task from different cameras. For example, you may have recorded one view of a task from a manipulator wrist camera while you recorded a second view of the same task from an ROV-mounted camera.

When you play back a video segment, a highlighted box below the video display indicates which camera captured the video. If additional cameras were capturing recorded video at the same time, other boxes display with the names of those cameras. You can click one of the camera names to switch to the video segment for that camera.

When you hover the mouse over an event place marker on the video progress bar, a pop-up panel shows the event type and also shows still photos of all recorded camera views of that event. You can also click the pop-up panel to go to the event in the Events tab (for example, to edit the event).

Using Event Place Markers to Control Playback

By clicking on the Events tab of the PLAYBACK screen, you can display a list of events that are contained in the selected project (that is, in all the videos listed in the Videos tab).

See how

To play back a video segment that contains an event place marker:

1. Display the PLAYBACK screen.
2. Click the Events tab.
3. Click the desired event place marker.
4. If more than one video stream was being recorded at the time of the event, click on the camera for the desired video stream.

You can also view a still photo from the video located at an event place marker. For information on exporting a still photo of an event, see About Exporting Video Files.
