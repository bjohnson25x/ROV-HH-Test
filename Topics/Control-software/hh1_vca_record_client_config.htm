Configuring for Client Recording

See Also for links to more information about client recording on the video computer. This information applies only to ROV systems with HDEVS.

To prepare for client recording:

1. Connect to the video computer all external memory devices that you want to use for client recording. You can use only USB 3 devices that are attached to USB 3 ports. (The USB ports on the front of the video computer are not USB 3 ports and cannot be used to connect external memory devices for client recording.)
2. On the VIDEO DASHBOARD, click the DRIVE MANAGEMENT icon.

The AVAILABLE STORAGE panel lists all connected external memory devices. (Warning messages indicate if any of the devices are not fast enough for recording, or do not have enough space.)

3. For each device that you want to use for client recording, click the drop-down list and then click ENABLE.

Each enabled external memory device then moves into the ENABLED STORAGE panel. Recording does not begin automatically, however. You must start recording as described in Starting and Stopping Client Recording.
