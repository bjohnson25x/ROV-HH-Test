Finalizing Client Recording for Export

See Also for links to more information about client recording on the video computer. This information applies only to ROV systems with HDEVS.

This information applies only to ROV systems that have HDEVS rather than TAVS.

You should not remove an external memory device from a video computer until you have finalized the client recording video segments on that device.

Finalizing an external memory device:

● Attaches audio annotations to their video segments.
● Creates a video log that lists all video files.

Once you have finalized a client recording session for a device, you can no longer create, edit, or delete any event place markers, event type descriptions, or audio annotations for the video segments on that device.

To finalize a client recording session:

1. Stop all client recording video segments (see Starting and Stopping Client Recording). This action is not required, but is a good practice.
2. Make any desired changes to event place markers, event type descriptions, or audio annotations (see Editing and Deleting Event Place Markers and Adding Audio Annotations to Recorded Video).
3. Display the DRIVE MANAGEMENT screen of the Video Computer App.
4. For each external memory device in the ENABLED STORAGE panel that you want to finalize, click the drop-down list and click Finalize. The external memory device than moves into the AVAILABLE STORAGE panel.

Pop-up messages then notify you that video is being copied onto the external memory device and that a video log is being created. When the messages disappear from the screen, you can disconnect the external memory device from the video computer.
