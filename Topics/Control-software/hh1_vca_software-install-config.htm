Installing/Configuring the Video Computer App

See Also for links to information about the Video Computer App. The Video Computer App is used only with HDEVS.

This information applies only to ROV systems that have HDEVS rather than TAVS.

You should install Video Computer App software when:

● A new version is available.
● A software patch is available.
● You replace the video computer.

When you install a new software version, all previous versions are retained. You can easily select and run a previous version at any time.

Note:  The software version number of the Video Computer App must match the software version number of the control system computers. If you change the control system software version, you must similarly change Video Computer App version. If the versions do not match, the Video Computer App will be able to operate only in the offline mode (see Video Computer App Offline Mode).

Installing a New Software Version

New software versions of the Video Computer App are delivered as part of a new control system version (on a USB flash drive). Software patches can be delivered on a flash drive, sent via e-mail, or copied from an FTP site. When you receive a new software version or patch, you should install it promptly.

See how

To install a new software version:

1. If the Video Computer App is already running, shut it down.
2. Insert the USB flash drive into the video computer.
3. Start up the computer. The computer automatically detects the Video Computer App software on the flash drive. The computer also checks the control system client computers and notes versions of the Video Computer App that are stored there.

4. Click INSTALL for the software version that is located on the flash drive.

The software then installs and the Video Computer App starts up.

Installing a Software Patch

Software patches are delivered as needed from the factory, and you should install them promptly. A patch is delivered as a single executable file with the file name extension .SRsvcPatch. If the file is not already on an external memory device (DVD, CD, or flash drive), copy the file into the root level of an external memory device.

See how

To install a software patch:

1. If the Video Computer App is already running, shut it down.
2. Insert the external memory device containing the patch into the video computer.
3. Start up the computer, and it will detect the software patch on the external memory device.
4. When the screen displays SELECT THE PATCH TO APPLY, select the desired patch from the drop-down list.
5. Click APPLY for the desired patch.
6. If the patch requires the computer to restart, a panel prompts you to allow the restart.

Reverting to Another Software Version

When you install a new software version, all previous versions are retained. You can easily select and run another version if necessary.

Note:  The software version number of the Video Computer App must match the software version number of the control system computers. If you change the control system software version, you must similarly change Video Computer App version. If the versions do not match, the Video Computer App will be able to operate only in the offline mode (see Video Computer App Offline Mode).

See how

To run another software version:

1. If the Video Computer App is already running, shut it down.
2. Start up the video computer.
3. As soon as the screen displays the CONFIGURE button, click it. (You have approximately 5 seconds.) Configuration selection panels then display.

4. Click the button in the SOFTWARE VERSION panel. In the drop-down list for SELECT THE SOFTWARE VERSION TO USE, click the desired version and then click RETURN TO LAUNCH.

The Video Computer App then starts up with the selected software version.

Configuring the Video Computer

If the ROV system has a single video computer, it is designated as video computer 1. If the ROV system has two video computers, the second must be designated as video computer 2.

See how

To configure the video computer number:

1. If the Video Computer App is already running, shut it down.
2. Start up the video computer.
3. As soon as the screen displays the CONFIGURE button, click it. (You have approximately 5 seconds.) Configuration selection panels then display.

4. Click the button in the CLIENT CONFIGURATION section. Select video computer 1 or video computer 2, and then click RETURN TO LAUNCH.

The Video Computer App then starts up.
