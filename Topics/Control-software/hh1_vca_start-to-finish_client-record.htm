Performing Client Recording On the Job

See Also for links to more information in the video recording start-to-finish guide. This guide applies only to ROV systems with HDEVS.

This information applies only to ROV systems that have HDEVS rather than TAVS.

To begin client recording for video:

1. Route desired video streams from ROV system cameras to the Video Computer App. The Video Computer App then begins to record all those video streams on video computer internal drives (in continuous backup mode).

Video streams can be routed from the VIDEO panel of the ROV control system FLIGHT screen or from the preview panel of the VIDEO DASHBOARD.

2. Start client recording.
3. As events occur in the operation, insert event place markers into the video. You can insert the event place markers from the ROV control system touch screens, the flight joystick, or the Video Computer App.
— If you insert an event place marker for an unplanned event, you can later change it to a named event type.
— To add an audio annotation to an event place marker, hold down the button for the event place marker. After a half second, audio recording begins and you can start speaking into the microphone. Audio recording continues until you release the button or 15 min have passed.
— You can edit video later to add/delete event place markers, change event types, add event type descriptions, or add/delete audio annotations.
4. Stop client recording. (You can restart and stop video recording as often as needed to capture points of interest.)
5. Add, remove or modify events (see Changing Recorded Video Data).
6. Finalize all external memory devices (in the ENABLED STORAGE panel of the DRIVE MANAGEMENT screen of the Video Computer App). You must finalize each device to incorporate audio annotations and to create video logs.
