Configuring for Video Recording

See Also for links to more information in the video recording start-to-finish guide. This guide applies only to ROV systems with HDEVS.

This information applies only to ROV systems that have HDEVS rather than TAVS.

Setting up the Video Computer App

Configuring Recording Options

Use the VIDEO CONFIGURATION screen of the Video Computer App to set up recording options.

See how
1. In the Recording Details panel, select the required number of channels and the recording resolution (HD or SD).
2. Select the desired video quality.
3. If you plan to make audio annotations:
a. Select the audio input device. All audio input devices detected by Windows appear in the drop-down list of devices. You can use a USB headset or a stand-alone microphone.
b. Speak in a clear voice, and adjust the volume of the audio input device until the input volume fills two-thirds of the screen gauge.
4. Enter the project name, location and dive number.
5. Click Save.
6. Return to the VIDEO CONFIGURATION screen.
7. In the Overlay Details panel, click Edit.
8. Edit the overlay layout according to the job specifications.

Setting up Devices for Client Recording

Prepare for client recording by setting up the external memory devices.

See how
1. Connect to the video computer all external memory devices that you want to use for client recording.
2. Go to the DRIVE MANAGEMENT screen of the Video Computer App. In the AVAILABLE STORAGE panel, for each external memory device that you want to record on, display the drop-down list and click Enable.
3. Once you enable client recording for a video stream, all video from that stream is automatically exported to all enabled external memory devices.

Setting Up the ROV Control System

Use the ROV control system touch screens to create new event sets and event types for known points of interest that will occur during the job, and to configure the joystick to trigger unplanned events during operation.

Event Sets and Event Types

From any CONFIGURATION screen on the touch screen, click the EVENT SETS tab to create job-specific event sets and event types (see Marking Events During Video Recording (HDEVS Only)).

See how

An event set is made up of events that will occur close in time to each other. By grouping related events into an event set, you reduce the number of events displayed on the screen at any time, making it easier to navigate through and choose events.

An event is a point of interest in the recorded video. You can create named event types to insert as event place markers into recorded video. Adding event place markers to recorded video has the following benefits:

● You can easily navigate to an event place marker in video both from the Video Computer App PLAYBACK screen and from the video log for exported video segments.
● You can add event type descriptions to event place markers for inclusion in the video log.
● You can add audio annotations to event place markers.
● Adding an event place marker generates a still photo from every camera that was recording at the time of the event.
● At any time after a video segment has been recorded, you can easily navigate to an event place marker in the recorded video to change the event type and to add, update, or remove an event type description or audio annotation.
● When you export or finalize video segments on an external memory device, a video log is created on the device. In addition to listing files on the external memory device, the video log includes all event place markers included in the video segments, along with the still photos and associated event type descriptions. A provided link opens each video that was recorded during the event time, to play the video with audio annotation.

Before a job begins, you should create event types for all points of interest in the job.

Joystick Event Trigger Button

If you wish to use a joystick button to trigger an unplanned event, configure flight joystick button 2 as an event trigger. This lets you trigger events, take still photos, and record audio annotations without releasing the joystick.

Flight joystick configuration is performed from the ROV system touch screen (see Assigning Video Event Marking to the Flight Joystick).
