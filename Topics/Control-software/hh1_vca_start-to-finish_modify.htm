Changing Recorded Video Data

See Also for links to more information in the video recording start-to-finish guide. This guide applies only to ROV systems with HDEVS.

This information applies only to ROV systems that have HDEVS rather than TAVS.

You can easily add, delete, or edit event place markers from video segments after recording.

Adding Event Place Markers

After video segments have been recorded, you can add event place markers from the PLAYBACK screen of the Video Computer App. Start playback for the desired video segment, navigate to the desired location in the video, and click the camera icon below the playback progress bar.

This action creates an unplanned event at the current playback position of the video segment and also creates a still photo of that video frame. The same event place marker is also added to all video segments that were being recorded at the same time as the segment you are viewing.

Changing or Deleting Event Place Markers

From the PLAYBACK screen of the Video Computer App, you can easily navigate directly to an event place marker in a recorded video segment. Click the Events tab to see a list of all event place markers in the video segments. When you click an event place marker, the video segment containing that event place marker begins to play. (Buttons also appear for all other video segments that were recorded at the same time.)

Once you click on an event, you can:

● Change the event type.
● Add, modify, or delete event type descriptions associated with the event place marker.
● Add or delete an audio annotation.
● Delete the event place marker.
