About Exporting Video Files

See Also for links to more information about exporting video files and then viewing exported files. This information applies only to ROV systems with HDEVS.

See Also for links to information about the Video Computer App. The Video Computer App is used only with HDEVS.

This information applies only to ROV systems that have HDEVS rather than TAVS.

Export of video files from the video computer can happen automatically or manually:

● When you perform client recording, the video segments are recorded onto an external memory device, and are therefore automatically exported. (Before you remove the external memory device containing client recordings, you must finalize the recording session as described in Finalizing Client Recording for Export.)
● When video segments are recorded only on the video computer’s internal drives during continuous backup recording, they must be exported manually onto an external memory device. You can also manually export a still photo from an event place marker in recorded video.

In both manual and automatic exporting, exported files include still photos of events and a video log with links to video files.

You can export videos to (not the same as record to) any external memory device listed in the AVAILABLE section of the DRIVE MANAGEMENT screen. You cannot export to external memory devices listed in the ENABLED section of the screen, even if client recording is not currently taking place.

Manually Exporting Video Segments

See how
1. Connect an external memory device to the video computer.
2. Display the DRIVE MANAGEMENT screen of the Video Computer Application. Ensure that the external memory device is displayed in the AVAILABLE section of the screen. (A message is displayed if the external memory device does not have enough free space.)
3. Display the PLAYBACK screen.
4. Click the Videos tab.
5. Use the drop-down list to select the desired project.
6. Click EXPORT, and controls are displayed for exporting video segments.

7. Click the check box for each video segment that you wish to export. You can also click the Export check box above the list to select all video segments in the list.
8. From the Destination drop-down box, select the desired external memory device. You do not need to navigate to a specific folder on the external memory device; the Video Computer App automatically creates a folder structure containing appropriate folders for exported video segments.

9. Click EXPORT.

After export has begun, you can click CANCEL at any time to stop the process.

Manually Exporting Still Photos from Videos

When you export videos to an external memory device, still photos of events are automatically included with the videos. You can also manually export a still photo of an event to an external memory device.

See how
1. Ensure that an external memory device is connected to the video computer.
2. Display the PLAYBACK screen.
3. Click the Events tab.
4. Click the desired event place marker. Still photos then appear for each video stream that contains the event place marker.
5. Right click the desired still photo.
6. When the export pop-up box appears, click it.
7. When the SELECT FILE SAVE LOCATION panel appears, browse to the desired location.
8. Click SAVE, and the still photo is exported as a JPEG file.
