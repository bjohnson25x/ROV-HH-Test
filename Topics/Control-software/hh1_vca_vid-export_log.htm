Viewing Exported Video Files Using the Video Log

See Also for links to more information about exporting video files and then viewing exported files. This information applies only to ROV systems with HDEVS.

This information applies only to ROV systems that have HDEVS rather than TAVS.

After video files have been exported onto an external memory device, you can remove that device from the video computer for transfer to another computer, transfer to a client, or archiving. A video log located on the external memory device makes it easy to navigate to and view specific video segments and still photos.

Caution!   Do not remove an external memory device while the video computer is writing to that device. If you do, files on the device may be unusable or inaccessible.

Note:  If you are exporting client recording files to an external memory device, you should finalize the device (see Finalizing Client Recording for Export) before you remove it from the video computer. If you do not finalize the device, you will not be able to view video files using the video log.

The video log is published in two forms, both of which contain hyperlinks to video segments and to still photos of events.

● A spreadsheet (in Microsoft Excel format).
● A browser-accessible web page (HTML format).

Accessing the Video Log

The video log is created when you export files to an external memory device. To access the video log after you have removed the external memory device from the video computer:

1. Connect the external memory device to a computer.
2. In a file manager application, browse to the external memory device. Because a high-level folder is created each time that you disconnect the external memory device from the video computer, you may see several folders containing different export sessions.

3. Click the desired folder and then click the desired video log format (spreadsheet or browser-accessible).

Spreadsheet Video Log

You can open the spreadsheet video log (with any application that reads Microsoft Excel spreadsheets) to view video files.

See how

You can sort columns in the spreadsheet to arrange and locate files, and you can search for and filter data in the columns.

The spreadsheet video log has two tabs, one for video files and one for events.

The Videos tab of the spreadsheet video log contains the following items for each video file:

● File name (which is a hyperlink ). Click the file name to start playing the file with the computer’s default video application.
● Video start and end times
● Project name
● Location
● Dive number
● Camera label (source of the video stream)

The Events tab of the spreadsheet video log contains:

● The time of the event.
● Event type and description.
● File name (which is a hyperlink) of the event still photo. Click the file name to display the still photo with the computer’s default picture application.

Browser-Accessible Video Log

You can open the browser-accessible video log with any web browser to view video files.

See how

To view a video:

1. Click the Video tab.
2. Use the project drop-down list to select the desired project.
3. In the list of video files, videos are grouped by project, location, and dive number. Click a video file, and it begins to play.
4. If you want to make a copy of the video file, right click the playback panel.

To view a video containing a particular an event place marker:

1. Click the Events tab.
2. Use the project drop-down list to select the desired project.
3. Click the desired event place marker. The screen displays still photos of all video segments containing the event place marker. Select the desired camera, and that video plays.

4. If you want to make a copy of the video file, right click the playback panel.
5. If you want to make a copy of a still photo for that event place marker, right click the still photo.
